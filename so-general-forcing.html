Section 2.7 Other forcing functions

¶

From 2.6 in the text.

Exponential forcing functions are very important, especially when allowing complex exponents. But they aren't the whole world. In this section we go over two other major techniques for trying to get particular solutions to linear, 2nd-order ODEs. Both assume that you have access to the complete null solution already.

Subsection 2.7.1 Undetermined coefficients

¶

In the method of undetermined coefficients we assume that we already know everything about the particular solution except for some constants to be found using algebra. Best to start with examples.

Example 2.7.1

Let's find a particular solution of \(y'' +4y'+4y=8t^2\text{.}\) We reason as follows. If we plug in some \(Ct^2\) for \(y\) with a constant \(C\text{,}\) then all the derivatives generate multiples of \(t\) and \(1\text{.}\) That is, both sides of the ODE will be quadratic polynomials. So we generalize just a bit to allow

\begin{equation*} y_p = A + Bt + Ct^2\text{.} \end{equation*}

Plugging that into the ODE yields

\begin{align*} 2C + 4(B+2Ct) + 4(A+Bt+Ct^2) \amp = 8t^2\\ 4C t^2 + (8C+4B) + (2C+4B+4A) \amp = 8t^2\text{.} \end{align*}

This has to be an identity for all \(t\text{.}\) Matching like powers, we can read off \(C=2\text{,}\) then \(B=-4\text{,}\) and finally \(A=3\text{.}\) This provides us with \(y_p=2t^2+4t+3\text{.}\) (The complete solution would require us to use the characteristic equation to find the null solution as well.)

In general, if \(f(t)\) is a polynomial, then we let \(y_p\) be a polynomial of the same degree with unknown coefficients. Substitution into the ODE and matching powers gives equations to determine those. As in the example above, you must put all the polynomial terms into \(y_p\text{,}\) even if they are missing from \(f(t)\text{.}\)

The next example has a polynomial times an exponential.

Example 2.7.2

Let's try for \(y'' -2y'-3y=10te^{4t}\text{.}\) For much the same reasons as in the last case, we try

\begin{equation*} y_p = (A + Bt)e^{4t}\text{.} \end{equation*}

There's nothing to do now except grind out some derivatives.

\begin{align*} 8(2A+B+2Bt)e^{4t} - 2(4A+B+4Bt)e^{4t} -3 (A + Bt)e^{4t} \amp =10t e^{4t}\\ 8(2A+B) - 2(4A+B) -3A + t(16B-8B-3B) = 10t\text{.} \end{align*}

From this, by matching powers, we deduce \(B=2\) and then \(5A+12=0\text{.}\) So \(y_p = [2t - (12/5)]e^{4t}\text{.}\)

So if \(f(t)\) is a polynomial times \(e^{st}\text{,}\) let \(y_p\) be the same exponential times a polynomial of the same degree with unknown coefficients. This technically applies too if the polynomial has degree zero, which was our mainstay in Section 2.4. It also applies for imaginary or complex exponents.

The third major type of forcing amenable to UC is sin or cos. We did this type of forcing in Example 2.4.2 by converting the forcing to a complex exponential. Here is how it plays out with UC instead.

Example 2.7.3

Consider \(y''+y'=\sin(2t)\text{.}\) Derivatives of \(sin(2t)\) procuce only itself and \(cos(2t)\text{,}\) and we are justified in choosing

\begin{equation*} y_p = a\cos(2t) + b\sin(2t)\text{.} \end{equation*}

Inserting this into the ODE leads to

\begin{equation*} [-4a\cos(2t) - 4b\sin(2t)] + [-2a\sin(2t) + 2b\cos(2t)] = \sin(2t)\text{.} \end{equation*}

This is true for all time if and only if we match the coefficients of the trig functions,

\begin{equation*} -4a + 2b = 0, \qquad -4b-2a = 1 \text{.} \end{equation*}

The solution of these equations is \(a=-1/10\text{,}\) \(b=-1/5\text{.}\) So \(-[\cos(2t)+2\sin(2t)]/10\) is a particular solution.

The method can be described when combining all of the above: forcing by sine or cosine, multiplied by a polynomial and/or exponential. I believe a term like \(e^{\alpha t}\cos(\beta t)\) is more easily treated as a complex exponential, and the whole process is usually best left to a computer if more than two unknown coefficients are needed.

There is one more important situation to be aware of, another variation on a familiar theme. If your \(y_p\) is part of the null solution of the ODE, you have to "promote" it by powers of \(t\) until it no longer overlaps.

Example 2.7.4

Given \(y'' -2y'-3y=4e^{-t}\text{,}\) we are led to try \(y_p=Ae^{-t}\text{:}\)

\begin{align*} (Ae^{-t}) - 2(-Ae^{-t}) - 3Ae^{-t} \amp = 4e^{-t} \\ 0 \amp 4e^{-t} \text{.} \end{align*}

This is impossible to satisfy. What's happened is that the characteristic polynomial is \((s+1)(s-3)\text{,}\) so \(e^{-t}\) is part of the null solution (as the above calculation verified). Instead we have to use \(y_p=Ate^{-t}\text{,}\) which gives

\begin{align*} Ae^{-t}(t-2) - 2Ae^{-t}(1-t) - 3Ate^{-t} \amp = 4e^{-t} \\ -4Ae^{-t} \amp 4e^{-t} \text{,} \end{align*}

so \(A=1\) and \(y_p\) is valid.

Subsection 2.7.2 Variation of parameters

¶

While UC relies on "educated guesswork" followed by derivatives and algebra, variation of parameters follows a simpler recipe but requires integration. Given \(L[y]=f\text{,}\) suppose we have a general null solution in the form

\begin{equation*} y_n(t) = c_1 y_1(t) + c_2 y_2(t)\text{.} \end{equation*}

For problems with constant coefficients, \(y_1\) and \(y_2\) are exponentials (or sin/cos equivalents), possibly multiplied by \(t\text{.}\) In any case, the VP trick is to replace the integration constants with functions of time:

\begin{equation*} y_p(t) = c_1(t) y_1(t) + c_2(t) y_2(t)\text{.} \end{equation*}

To keep notation simpler, we assume below that the coefficient of \(y''\) in the ODE is 1.

Taking this odd assumption on faith for now, we compute

\begin{equation*} y_p' = c_1'y_1+c_1y_1'+ c_2'y_2+c_2y_2' \text{.} \end{equation*}

Before we go on, we're going to put a restriction on our mystery functions, namely \(c_1'y_1+c_2'y_2=0\text{.}\) That makes life a little easier. Continuing, we get

\begin{equation*} y_p'' = c_1'y_1'+ c_1y_1'' + c_2'y_2'+ c_2y_2'' \text{.} \end{equation*}

When we put all this mess into \(y''+By'+Cy=f\text{,}\) something cool happens:

\begin{equation*} c_1(y_1''+By_1'+Cy_1) + c_2(y_2''+By_2'+Cy_2) + c_1'y_1' + c_2'y_2' = f \text{.} \end{equation*}

Thanks to the null property of \(y_1\) and \(y_2\text{,}\) this boils down to \(c_1'y_1' + c_2'y_2' = f\text{.}\)

To summarize so far, we get a valid particular solution using two conditions,

\begin{align*} c_1'y_1+c_2'y_2 \amp = 0 \\ c_1'y_1' + c_2'y_2' \amp = f \text{.} \end{align*}

Stare at this long enough and you'll realize that it's just two linear equations for \(c_1'\) and \(c_2'\text{.}\) You can do some elimination to solve them (more on this in future chapters), and you wind up with

\begin{equation*} c_1' = -\frac{y_2f}{W}, \qquad c_2' = \frac{y_1f}{W}, \end{equation*}

with

\begin{equation*} W(t) = y_1y_2'-y_1'y_2\text{,} \end{equation*}

known as the Wronskian of \(y_1\) and \(y_2\text{.}\) It's easiest to remember as a determinant:

\begin{equation*} W = \begin{vmatrix} y_1 \amp y_2 \\ y_1' \amp y_2' \end{vmatrix}\text{.} \end{equation*}

Finally, "all" you have to do is integrate these expressions to get \(c_1\) and \(c_2\text{,}\) and from there you get \(y_p\text{.}\)

Example 2.7.5

Let's apply VP to \(y''-y=1\text{.}\) The null solutions are \(y_1=e^{-t}\) and \(y_2=e^t\text{,}\) which implies \(W=1+1=2\text{.}\) Then

\begin{equation*} c_1' = -\frac{y_2f}{W} = -\frac{1}{2}e^t, \qquad c_2' = \frac{y_1f}{W} = \frac{1}{2}e^{-t}\text{.} \end{equation*}

Now we integrate.

\begin{equation*} c_1(t) = -\frac{1}{2} \int_0^t e^s \, ds = \frac{1}{2}(1-e^{t}), \qquad c_2(t) = \frac{1}{2} \int_0^t e^{-s} \, ds = \frac{1}{2}(1-e^{-t})\text{.} \end{equation*}

In fact these are determined only up to integration constants (because particular solutions aren't unique), so we can drop the extra constants and get

\begin{equation*} y_p = -\frac{1}{2}e^t \cdot e^{-t} - \frac{1}{2} e^{-t} \cdot e^t = -1\text{.} \end{equation*}

This was a lot more difficult than UC on the same problem. However, now consider solving \(y''-y=H(t-T)\) for a step time \(T \gt 0\text{.}\) The VP equations are now \(c_1' = -\frac{1}{2}e^tH(t-T)\text{,}\) \(c_2'=\frac{1}{2}e^{-t}H(t-T)\text{.}\) From here we get

\begin{equation*} c_1(t) = \int_0^T -\frac{1}{2}e^s H(s-T)\, ds = -\frac{1}{2} H(t-T) \int_T^t e^s\, ds = \frac{1}{2} H(t-T) [e^T - e^t] \end{equation*}

and, similarly, \(c_2(t) = \frac{1}{2} H(t-T) [e^{-T}-e^{-t}]\text{.}\) This leads to

\begin{equation*} y_p = \frac{1}{2} H(t-T) \left[ e^{T-t} - 1 + e^{t-T} - 1 \right]\text{,} \end{equation*}

which is a lot less obvious in the UC method.

If you immediately wondered "what if the Wronskian is zero?" when you saw the VP formulas above, you have earned a special place in my heart. This possibility can be ruled out for some rather general theortical reasons, but we can also just calculate away the issue for equations with constant coefficients. If we have distinct characteristic roots \(s_1\) and \(s_2\text{,}\) then

\begin{equation*} W = (e^{s_1t})(s_2e^{s_2t}) - (s_1e^{s_1t})(e^{s_2t}) = (s_2-s_1)e^{(s_1+s_2)t}\text{.} \end{equation*}

Never zero, as promised. In the case of repeated root \(s_1\text{,}\) then

\begin{equation*} W = e^{s_1t}(s_1t +1)e^{s_1t} - (s_1 e^{s_1t})(t e^{s_1t}) = e^{2s_1t} \text{.} \end{equation*}

And here you were all worried.

Subsection 2.7.3 Fundamental solutions, again

If we continue to push ahead with the VP formulas, interesting things start to happen. For distinct roots,

\begin{equation*} c_1' = -\frac{\exp(s_2t)}{(s_2-s_1)\exp((s_1+s_2)t)} f(t) = \frac{f(t)e^{-s_1t}}{s_1-s_2}\text{,} \end{equation*}

and similarly \(c_2'= f(t)e^{-s_2t}/(s_2-s_1) \text{.}\) Then

\begin{align*} y_p(t) \amp = e^{s_1 t} \int_0^t \frac{f(\tau)e^{-s_1\tau}}{s_1-s_2}\, d\tau + e^{s_2 t} \int_0^t \frac{f(\tau)e^{-s_2\tau}}{s_2-s_1}\, d\tau \\ \amp = \int_0^t f(\tau) \frac{ e^{s_1(t-\tau)} - e^{s_2(t-\tau)}}{s_1-s_2}\, d\tau \\ \amp = \int_0^t f(\tau) g(t-\tau) \, d\tau \text{,} \end{align*}

with

\begin{equation*} g(t) = \frac{ e^{s_1t} - e^{s_2t} }{s_1-s_2} \text{.} \end{equation*}

For the case of a repeated root we can go though the VP formulas again, or just let \(s_2\to s_1\) and get

\begin{equation*} g(t) = t e^{s_1t} \text{.} \end{equation*}

The formula \(\int_0^t f(\tau) g(t-\tau) \, d\tau\) is reminiscent of the growth factor formula we got in first-order equations, which had \(\int_0^t G(s,t)f(s)\, ds\text{,}\) with \(G(s,t)=\exp[a(t-s)]\) for constant coefficients. Since we are no longer concerned only with growth processes, we now refer to such a \(G\) or \(g\) as a fundamental solution or Green function.

If we use an impulse forcing function \(f(t)=\delta(t)\) in the integral formula for \(y_p\text{,}\) then we get simply \(y_p(t)=g(t)\text{.}\) So the fundamental solution is also the impulse response. It's easy to see from the formula for \(g\) that \(g(0)=0\) and \(g'(0)=1\text{;}\) that is, the impulse forcing is equivalent to an instantaneous jump in \(y'\text{.}\)
