Section 4.5 Nullspaces

Â¶

From section 5.2

A little jargon we won't dwell on is a vector space. It's a set of vectors having a particular property: any linear combination of vectors in the set is also in the set (aka "closure" under addition and scalar multiplication). The space we will be interested in is called a nullspace of an \(n\times n\) matrix \(\bA\text{,}\) defined as

\begin{equation*} \mathcal{N}(\bA) = \{ \text{all $n$-vectors $\bv$ such that $\bA\bv=\bzero$} \}\text{.} \end{equation*}

To see that this has the required closure property, suppose \(\bv_1,\dots,\bv_k\) are in \(\mathcal{N}(\bA)\text{.}\) Then

\begin{equation*} \bA( c_1 \bv_1 + \dots + c_k\bv_k ) = c_1 \bA\bv_1 + \cdots + c_k \bA \bv_k = \bzero\text{,} \end{equation*}

so the linear combination is in \(\mathcal{N}(\bA)\text{,}\) too.

Subsection 4.5.1 Nullspaces from elimination

We can use regular elimination to find a nullspace. Since \(\bb=\bzero\) by definition, all the row operations leave it unchanged, and we can ignore it until the back substitution stage.

Example 4.5.1
\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \\ 2 \amp 2 \amp 1 \\ -1 \amp -1 \amp 4 \end{bmatrix} \mapsto \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 0 \amp 3 \\ 0 \amp 0 \amp 3 \end{bmatrix} \mapsto \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 0 \amp 3 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

The last row just says \(0=0\text{,}\) so we ignore it. The second row implies \(x_3=0\text{,}\) and then the first row implies \(x_1+x_2=0\text{.}\) We can regard \(x_2\) as a free variable; say \(x_2=\alpha\text{.}\) Then \(x_1=-\alpha\text{,}\) and the solution takes the form

\begin{equation*} \bx = \begin{bmatrix} -\alpha \\ \alpha \\ 0 \end{bmatrix} = \alpha \begin{bmatrix} -1 \\ 1 \\ 0 \end{bmatrix}\text{.} \end{equation*}

Thus the nullspace consists of all multiples of the constant vector \(\begin{bmatrix} -1 \\ 1 \\ 0 \end{bmatrix}\text{.}\) This representation isn't unique; any nonzero member of the space would serve as well.

Example 4.5.2

Suppose elimination ends with the matrix

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 4 \end{bmatrix}\text{.} \end{equation*}

Now back substitution implies \(x_3=0\text{,}\) then \(x_2=0\text{,}\) then \(x_1=0\text{.}\) So \(\mathcal{N}(\bA) = \{\bzero\}\text{.}\)

Example 4.5.3
\begin{equation*} \begin{bmatrix} 0 \amp 2 \amp -3 \\ 0 \amp 4 \amp -6 \\ 0 \amp -2 \amp 3 \end{bmatrix} \mapsto \begin{bmatrix} 0 \amp 2 \amp -3 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

The only condition here is \(2x_2-3x_3=0\text{.}\) Say \(x_3=\alpha\) and \(x_1=\beta\text{.}\) Then solutions take the form

\begin{equation*} \bx = \begin{bmatrix} \beta \\ 3\alpha/2 \\ \alpha \end{bmatrix} = \alpha \begin{bmatrix} 0 \\ 3/2 \\ 1 \end{bmatrix} + \beta \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}\text{.} \end{equation*}

The nullspace consists of all linear combinations of the two constant vectors shown above. Because two such vectors are needed, we say the nullspace is "two-dimensional."

Subsection 4.5.2 Row echelon form

The target of row elimination in the case of a nullspace can be described by three rules:

  1. The leftmost nonzero in any row is a 1 (called a leading one).
  2. The leading one in any row is to the right of any leading ones in higher rows.
  3. Any all-zero rows are at the bottom of the matrix.

We have not been following rule 1 up to now, but it's easy to satisfy; we just divide each row by its leading nonzero value. Such a matrix is said to be in row-echelon form (REF).

Example 4.5.4

REFs for the examples in the last subsection are

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix}, \quad \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 1 \end{bmatrix}, \quad \begin{bmatrix} 0 \amp 1 \amp -3/2 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

Once the matrix has been transformed to REF, you can divide the components of the solution \(\bx\) into two groups: the pivot variables are those whose columns have a leading one somewhere, and the free variables are the rest. The number of free variables is the dimension of the nullspace (also called the nullity of \(\bA\)). You can then find a representation of the nullspace by setting each of the free variables to one in turn (making the others zero) and solving for the resulting pivot variables. The nullspace is the set of linear combinations of those vectors.

Example 4.5.5

Given the REF

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{,} \end{equation*}

we see that \(x_2\) is the only free variable. When we set \(x_2=1\) during back substitution, we find that \(x_3=0\) and \(x_1=-1\text{.}\) Thus \(\mathcal{N}(\bA)\) consists of all linear combinations of \(\begin{bmatrix} -1 \\ 1 \\ 0 \end{bmatrix}\text{,}\) i.e., all multiples of that vector. With the REF

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{,} \end{equation*}

there are no free variables. The nullspace has dimension zero; i.e., it is the single point \(\bzero\text{.}\) Finally, with the REF

\begin{equation*} \begin{bmatrix} 0 \amp 1 \amp -3/2 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{,} \end{equation*}

the free variables are \(x_1\) and \(x_3\text{.}\) When \(x_1=1\) and \(x_3=0\text{,}\) we get \(\begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}\text{,}\) and when vice versa we get \(\begin{bmatrix} 0 \\ 3/2 \\ 1 \end{bmatrix}\text{.}\)
