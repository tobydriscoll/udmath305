Section 5.2 Special forms

Let's repeat the formulas defining our Fourier series:

\begin{equation*} u_k(x) = \frac{1}{\sqrt{2\pi}} e^{ikx}, \quad k=0,\pm 1,\pm 2,\dots\text{,} \end{equation*}
\begin{equation*} f(x) = \sum_{k=-\infty}^\infty c_k u_k(x),\quad c_k = \int_{-\pi}^\pi \overline{u_k}(x) f(x)\, dx\text{.} \end{equation*}

These were derived for a function \(f\) that is defined on \([-\pi,\pi]\text{.}\) If we instead have a function defined on \(\tilde{x}\in [a,b]\text{,}\) then we can define the change of variables

\begin{equation*} \tilde{x} = 2\pi \frac{x-a}{b-a} - \pi, \qquad \tilde{f}(\tilde{x}) = f(x)\text{,} \end{equation*}

and then apply the standard formulas to \(\tilde{f}(\tilde{x})\text{.}\)

Example 5.2.1

A common choice of domain is \(\tilde{x}\in [-L,L]\text{,}\) in which case

\begin{equation*} \tilde{x} = \frac{\pi x}{L}\text{.} \end{equation*}

Hence

\begin{align*} c_k \amp = \int_{-\pi}^\pi \overline{u_k}(\tilde{x}) \tilde{f}(\tilde{x})\, d\tilde{x}\\ \amp = \sqrt{\frac{\pi}{2L}} \int_{-L}^L e^{-ik\pi x/L} f(x)\, dx \text{,} \end{align*}

and the series is

\begin{equation*} f(x) = \tilde{f}(\tilde{x}) = \sum_{k=-\infty}^\infty c_k u_k(\tilde{x}) = \frac{1}{\sqrt{2\pi}} \sum_{k=-\infty}^\infty c_k e^{ik\pi x/L}\text{.} \end{equation*}

Subsection 5.2.1 Real Fourier series

The elegance of the complex Fourier series can't be beat. Still, some people will always be hating on complex numbers, so we ought to see how it looks in purely real terms. First, if \(f\) is a real function, then it's easy to decompose \(c_k\text{:}\)

\begin{align*} c_k \amp = \frac{1}{\sqrt{2\pi}} \int_{-\pi}^\pi \cos(kx) f(x)\, dx - i \cdot \frac{1}{\sqrt{2\pi}} \int_{-\pi}^\pi \sin(kx) f(x)\, dx\\ \amp = \alpha_k - i \beta_k\text{.} \end{align*}

Then we decompose the Fourier series itself:

\begin{align*} f(x) \amp = \frac{1}{\sqrt{2\pi}} \sum_{k=-\infty}^\infty (\alpha_k - i \beta_k) \bigl[ \cos(kx) + i\sin(kx) \bigr] \\ \amp = \frac{1}{\sqrt{2\pi}} \sum_{k=-\infty}^\infty \bigl[ \alpha_k \cos(kx) + \beta_k \sin(kx) \bigr] + i \bigl[\cdots \bigr] \text{,} \end{align*}

where we already know the missing part must work out to be zero, as it's the imaginary part of \(f\text{.}\) We'll also use the fact that the cosines and sines are even and odd functions,

\begin{equation*} \cos(-kx)=\cos(kx), \quad \sin(-kx) = -\sin(kx)\text{,} \end{equation*}

to break up the sum. The result is

\begin{align*} f(x) \amp = \frac{1}{\sqrt{2\pi}} \left[ \alpha_0 + \sum_{k=1}^\infty (\alpha_k+\alpha_{-k}) \cos(kx) + (\beta_k-\beta_{-k}) \sin(kx) \right] \\ \amp = \frac{1}{2} a_0 + \sum_{k=1}^\infty a_k \cos(kx) + b_k \sin(kx) \text{,} \end{align*}

where the definitions imply

\begin{align*} a_0 \amp = \frac{2}{\sqrt{2\pi}} \alpha_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x)\, dx\\ a_k \amp = \frac{1}{\sqrt{2\pi}} (\alpha_k + \alpha_{-k}) = \frac{1}{\pi} \int_{-\pi}^{\pi} \cos(kx) f(x)\, dx, \quad k=1,2,\ldots, \\ b_k \amp = \frac{1}{\sqrt{2\pi}} (\beta_k - \beta_{-k}) = \frac{1}{\pi} \int_{-\pi}^{\pi} \sin(kx) f(x)\, dx, \quad k=1,2,\ldots \text{.} \end{align*}

This is the Fourier series in real form.

Subsection 5.2.2 Sine and cosine series

There are some cases where the series simplifies a good deal. One of those is when the underlying function \(f\) is both real and odd, i.e., \(f(-x)=-f(x)\text{.}\) It becomes clear that \(a_k=0\) for all \(k\text{,}\) and that

\begin{equation*} b_k = \frac{1}{\pi} \int_{-\pi}^{\pi} \sin(kx) f(x)\, dx = \frac{2}{\pi} \int_{0}^{\pi} \sin(kx) f(x)\, dx \qquad \text{if f is odd,} \end{equation*}

because the product of odd functions is even. Similarly, if \(f\) is even, then \(b_k\equiv 0\) and

\begin{equation*} a_k = \frac{1}{\pi} \int_{-\pi}^{\pi} \cos(kx) f(x)\, dx = \frac{2}{\pi} \int_{0}^{\pi} \cos(kx) f(x)\, dx \qquad \text{if f is even} \text{.} \end{equation*}

In light of the above, one way to interpret the full Fourier series in real form is that \(f\) is expressed as the sum of an odd function and an even one. (This is less deep than it might seem, since for any function

\begin{equation*} f(x) = \frac{1}{2} [ f(x) - f(-x) ] + \frac{1}{2} [ f(x) + f(-x) ]\text{,} \end{equation*}

which is more or less the same odd/even decomposition.)

Subsection 5.2.3 Odd and even extensions

A curious aspect of the formulas for the pure sine and pure cosine series is that even though we started with a function defined on \([-\pi,\pi]\text{,}\) the formulas only access the values on \([0,\pi]\text{,}\) because we assumed a symmetry. But we can apply the formulas to any function that is given only on \([0,\pi]\text{.}\) For instance, we compute

\begin{equation*} a_k = \frac{2}{\pi} \int_{0}^{\pi} \cos(kx) f(x)\, dx\text{,} \end{equation*}

and then

\begin{equation*} \frac{1}{2} a_0 + \sum_{k=1}^\infty a_k \cos(kx) \end{equation*}

should represent \(f\) on \([0,\pi]\) and also be an even function. That is, it extends the definition of \(f\) to \([-\pi,0]\) by means of \(f(-x)=f(x)\) there. (This is a big part of the JPEG image compression algorithm.) We can similarly make an odd extension of \(f\) using the definitions of a sine series.

The odd case in particular brings up an interesting question. By definition, a true odd function has to satisfy \(f(0)=0\text{.}\) But suppose we start with an \(f\) that is nonzero at the origin and then compute its sine series? What should we expect to happen? Answering this kind of question means that we must look more carefully into how all of these series converge.
