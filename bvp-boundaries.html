Section 6.1 Boundaries

Let's return to the world of the second-order oscillator, \(y''+y=f(t)\text{.}\) We know the general solution to this problem has two arbitrary constants in it,

\begin{equation*} y(t) = c_1 \cos(t) + c_2 \sin(t) + y_p(t)\text{.} \end{equation*}

In some examples we found specific values for \(c_1\) and \(c_2\) as determined by the additional requirements \(y(0)=\alpha\) and \(y'(0)=\beta\text{.}\) We never stopped to ask whether this was always feasible. But note that

\begin{align*} \alpha \amp = y(0) = c_1 + y_p(0)\\ \beta \amp = y'(0) = c_2 + y_p'(0)\text{,} \end{align*}

so there is no difficulty. In most linear problems (those without a repeated root) we would have the general solution \(c_1 e^{s_1 t} + c_2 e^{s_2 t} + y_p(t)\text{,}\) and then the initial conditions imply

\begin{align*} \alpha \amp = y(0) = c_1 + c_2 + y_p(0)\\ \beta \amp = y'(0) = s_1 c_1 + s_2 c_2 + y_p'(0)\text{,} \end{align*}

which is a linear system with coefficient matrix \(\twomat{1}{1}{s_1}{s_2}\text{.}\) This matrix is nonsingular as long as \(s_1\neq s_2\text{,}\) so again we know there will be unique choices for \(c_1\) and \(c_2\text{.}\) (For a repeated root the general solution changes, but we end up at the same conclusion. In fact, this is exactly why having a nonzero Wronskian was important a while ago.)

In a physical oscillator such as a pendulum, the mathematics backs up our intuition. If you start a pendulum at a certain angle and angular velocity, only one outcome is possible, forever.

But physical reasoning also suggests that this conclusion may not completely generalize to using any two values to determine \(c_1\) and \(c_2\text{.}\) For instance, forget the initial conditions and instead consider using \(y(0)=y(7)=0\text{.}\) One obvious possibility is that the pendulum never moves. But you know that you can also tap the bob just so, so that it turns around at 3.5 seconds and then comes back to the lowest position at 7 seconds, thereby satisfying both side conditions. If you tap it a bit harder just so, it will go left, right, and then left again to reach the bottom spot again at 7 seconds, providing yet another solution. You see that we can generate infinitely many solutions to this problem!

A differential equation

\begin{equation*} ay''(x) + by'(x) + cy(x) = f(x)\text{,} \end{equation*}

meant to hold over the interval \([a,b]\text{,}\) together with the conditions \(y(a)=\alpha\text{,}\) \(y(b)=\beta\text{,}\) is called a two-point boundary value problem (TPBVP). You'll note that I changed the independent variable from \(t\) to \(x\text{,}\) which is just cosmetic. But it symbolizes the fact that many TPBVPs arise when a quantity varies over space, rather than time.

All our earned knowledge about the ODE itself will still apply here. But as the thought experiment above points out, some of our conclusions about the solutions will change, primarily the notion of uniqueness.
