Section 1.3 Vector algebra

¶

Subsection 1.3.1 Linear combinations

¶

We use \(\Rn{n}\) to denote the set of all vectors with \(n\) real elements. (Every vector is assumed to have a single-column shape.) If \(\bx\) is in \(\Rn{n}\text{,}\) then for any real number \(c\) we define \(c\bx\) by elementwise multiplication. Because this just scales the magnitude of the vector, a real value is often called a scalar. If \(\bx\) and \(\by\) are in \(\Rn{n}\text{,}\) then we can add or subtract them elementwise. (Addition of a scalar and a vector, and addition of vectors of different lengths, are undefined.)

Scaling and addition are often rolled together into a singly named operation.

Definition 1.3.1. Linear combination.

A linear combination of vectors \(\ba_1,\ldots,\ba_k\text{,}\) all in \(\Rn{n}\text{,}\) is

\begin{equation} x_1 \ba_1 + x_2 \ba_2 + \cdots + x_k \ba_k = \sum_{j=1}^k x_j \ba_j\text{,}\label{la-va-lincomb}\tag{1.3.1} \end{equation}

where the \(x_j\) are scalars known as coefficients (or weights).

One motivation for this definition is to express linear systems.

Example 1.3.2.

Consider the system

\begin{align*} x_1 - x_2 - x_3 \amp = 2 \\ 3x_1 - 2x_2 \amp = 9 \\ x_1 - 2x_2 - x_3 \amp = 5 \text{.} \end{align*}

Interpreting vector equalities elementwise, this is equivalent to

\begin{equation*} x_1 \threevec{1}{3}{1} + x_2 \threevec{-1}{-2}{-2} + x_3 \threevec{-1}{0}{-1} = \threevec{2}{9}{5}\text{.} \end{equation*}

We can identify the three vectors on the left side as the columns of the coefficient matrix \(\bA\text{,}\) and the one on the right is \(\bb\text{.}\) In an earlier example, we found the solution \(\bx=[1,-3,2]\text{.}\) Hence

\begin{equation*} \ba_1 - 3 \ba_2 + 2\ba_3 = \bb\text{,} \end{equation*}

Subsection 1.3.2 Matrix times vector

The linear combination definition (1.3.1) actually serves as the foundation of multiplication between a matrix and a vector. We use \(\Rmn{m}{n}\) to mean the set of all matrices with \(m\) rows and \(n\) columns.

Definition 1.3.4. Matrix times vector.

Given \(\bA\in\Rmn{m}{n}\) and \(\bx\in\Rn{n}\text{,}\) the product \(\bA\bx\) is defined as

\begin{equation} \bA\bx = x_1 \ba_1 + x_2 \ba_2 + \cdots + x_n \ba_n = \sum_{j=1}^n x_j \ba_j\text{.}\label{la-mv-matvec}\tag{1.3.2} \end{equation}
Example 1.3.5.

The product

\begin{equation*} \begin{bmatrix} 1 \amp -1 \amp -1 \\ 3 \amp -2 \amp 0 \\ 1 \amp -2 \amp -1 \end{bmatrix} \threevec{-1}{2}{-1} \end{equation*}

is equivalent to

\begin{equation*} (-1) \threevec{1}{3}{1} + (2) \threevec{-1}{-2}{-2} + (-1) \threevec{-1}{0}{-1} = \threevec{-2}{-7}{-4}\text{.} \end{equation*}

We often don't write out the product in this much detail. Instead we "zip together" the rows of the matrix with the entries of the vector:

\begin{equation*} \threevec{(-1)(1)+(2)(-1)+(-1)(-1)}{(-1)(3)+(2)(-2)+(-1)(0)}{(-1)(1)+(2)(-2)+(-1)(-1)} = \threevec{-2}{-7}{-4}\text{.} \end{equation*}

You might recognize the expressions in this vector as dot products from vector calculus, but that's a whole 'nother story.

What justifies calling this operation multiplication? In large part, it's the natural distributive properties

\begin{align*} \bA(\bx+\by) \amp = \bA\bx + \bA\by\\ (\bA+\bB)\bx \amp = \bA\bx + \bB\bx \end{align*}

(where matrix addition is also defined elementwise), which can be checked with just a little determination. It's also true that \(\bA(c\bx)=c(\bA\bx)\text{.}\) However, the product \(\bA\bx\) is not commutative; \(\bx\bA\) is not even defined.

As a matter of fact, these properties combine in a way worth noting. We can define a function \(L\) on vectors by \(L(\bx)=\bA\bx\text{.}\) It has very important properties that we want to single out by name.

Definition 1.3.6. Linear function.

A function \(L\) from \(\Rn{n}\) to \(\Rn{m}\) is a linear function if

\begin{equation*} L(c\bx + \by)=cL(\bx)+L(\by) \end{equation*}

for all \(n\)-vectors \(\bx,\by\) and scalars \(c\text{.}\)

This property would not hold for \(L(\bx)=x_1x_2\text{,}\) for instance. But that nonlinear function also cannot be described using a matrix-vector multiplication.
