Section 2.3 Homogeneous solutions

¶

Our first job is to figure out how to express all possible solutions of the homogeneous problem \(\opA[x]=0\text{,}\) or \(x'-a(t)x=0\text{.}\) As with linear systems, the goal is to find a basis, so that the general \(x_h\) is a linear combination of the basis functions. However, there is no analog of the RREF of a matrix to exploit here.

Fortunately we can just use some calculus. Define an integrating factor as

\begin{equation*} g(t) = \exp[ q(t) ]\text{,} \end{equation*}

for a \(q(t)\) to be defined momentarily. Then

\begin{equation*} g'(t) = q'(t) g(t)\text{.} \end{equation*}

So \(x=g\) is a solution of \(x'-a(t)x=0\) if \(a(t)=q'(t)\text{,}\) that is, \(q\) is an antiderivative of \(a\text{:}\)

\begin{equation} g(t) = \exp\left( \int a(t)\,dt \right)\text{.}\label{fo-eq-intfactor}\tag{2.3.1} \end{equation}

Thus, \(x_h=c_1g(t)\) is a homogeneous solution for any constant \(c_1\text{.}\) It's not an elementary fact that this is the general homogeneous solution.

The integrating factor \(g(t)\) is not uniquely defined, because it contains an indefinite integral. But adding \(C\)>to the integral is the same as multiplying \(g(t)\) by \(e^C\text{,}\) and the general homogeneous solution is not affected. This is the usual situation of scalar nonuniqueness for expressing the basis of a nullspace.

Example 2.3.1.

To solve \(x'=\sin(t) x\text{,}\) we integrate to get \(-\cos(t)\) and then exponentiate, so that

\begin{equation*} x_h(t) = c_1 \exp[ -\cos(t) ]\text{.} \end{equation*}
