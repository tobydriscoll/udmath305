The goal is to introduce zeros into the lower triangle of this matrix. By using only elementary row operations, we ensure that the matrix always represents a linear system that is equivalent to the original. We proceed from left to right and top to bottom. The first step is to put a zero in the (2,1) location using a multiple of row 1:
mult21 = S(2,1)/S(1,1)
mult21 = -1
S(2,:) = S(2,:) - mult21*S(1,:)

2.0000 0 4.0000 3.0000 4.0000 0 0 6.0000 -10.0000 44.0000 1.0000 15.0000 2.0000 -4.5000 29.0000 -4.0000 5.0000 -7.0000 -10.0000 9.0000
We repeat the process for the (3,1) and (4,1) entries.
mult31 = S(3,1)/S(1,1)
mult31 = 0.5000
S(3,:) = S(3,:) - mult31*S(1,:);
mult41 = S(4,1)/S(1,1)
mult41 = -2

At this point, everything in column 1 is set up to make an upper triangular matrix. Nothing we do from here on should change it any further. So we turn our attention to the second column. This has to get zeros in the (3,2) and (4,2) entries. We don't want to add in multiples of the first row any more, because that would wreck our beautiful zeros.
However, we can see that multiples of the second row won't work either. The zero in the (2,2) position (called the pivot element) is not able to change the numbers below it.
The fix is easy. We just swap two rows to bring in a nonzero pivot. In this case we choose to swap in the 4th row, but any row below the second would work.
S([2 4],:) = S([4 2],:)
S =
2 0 4 3 4 0 5 1 -4 17 0 15 0 -6 27 0 0 6 -10 44
Now we are ready to proceed with putting zeros below the diagonal in the second column.
mult32 = S(3,2)/S(2,2)
mult32 = 0
S(3,:) = S(3,:) - mult32*S(2,:);
mult42 = S(4,2)/S(2,2)
mult42 = 0
S(4,:) = S(4,:) - mult42*S(2,:);
S
S =
2 0 4 3 4 0 5 1 -4 17 0 0 -3 6 -24 0 0 6 -10 44
(We didn't change anything in the 4th row this time, because we didn't need to. But we ran the algorithm through it for generality.) Now columns 1 and 2 are both in good shape. We move to column 3. There is just one zero to introduce, at the (4,3) position.
mult43 = S(4,3)/S(3,3)
mult43 = -2

Voila! It's an upper-triangular matrix. To get here, we worked from the top left, down and to the right. Now we start at the bottom right and work back.
