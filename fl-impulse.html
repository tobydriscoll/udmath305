Section 2.7 Steps and impulses

¶

The following function is very useful in some modeling situations.

Definition 2.7.1. Unit step function.

The unit step function or Heaviside function \(H(t)\) is defined to be zero for \(t\le 0\) and one for \(t\gt 0\text{.}\)

The unit step function represents throwing on an idealized switch. To have the switch turn on at an arbitrary time \(T\text{,}\) we use \(H(t-T)\text{.}\) It's worth mentioning here the related idea of a window function, which turns on and then back off. To get a function that is one only between times \(S\) and \(T\text{,}\) for example, we can use \(f(t)=H(t-S)-H(t-T)\text{.}\)

It's relatively straightforward, if cumbersome, to incorporate step forcing into our solution methods used so far. The key fact to keep in mind is:

Fact 2.7.2.

The solution of \(x'-a(t)x=H(t-T)f(t)\) having continuous \(a(t)\) and \(f(t)\) is continuous.

As a result, we can solve an initial-value problem for \(x'-a(t)x=H(t-T)\) piecewise. Assuming the initial time is zero for simplicity, we first solve

\begin{equation*} x'-a(t)x=0 \end{equation*}

over \(0 \le t \le T\text{,}\) using the given value of \(x(0)\text{.}\) Then we solve

\begin{equation*} x'-a(t)x=1 \end{equation*}

for \(t\gt T\text{,}\) using the already-known \(x(T)\) as the “initial” value of this stage. If there are multiple step functions present, then there are more than just two pieces to integrate. At each jump point in the forcing function we reset the initial value and start again with a different nonhomogeneous forcing value. The process is the same for \(x'-ax=H(t-T)f(t)\text{.}\)

Example 2.7.3.

Let's solve \(x'-x=f(t)\text{,}\) \(x(0)=-2\text{,}\) where \(f(t)\) is 5 for \(3 \lt t \le 6\) and zero elsewhere. The homogeneous solution is \(x_h(t)=c_1e^{t}\text{.}\)

Over \(0 \le t \le 3\text{,}\) the homogeneous solution is the full solution. Applying \(x(0)=-2\) we get

\begin{equation*} x_{[0,3]}(t) = -2 e^t \end{equation*}

for this interval. Over \(3 \le t \le 6\text{,}\) the ODE is \(x'-x=5\text{.}\) Undetermined coefficients suggests the particular solution \(x_p(t)=A\text{,}\) and it's then clear that \(A=-5\text{.}\) Our general solution over this segment is therefore \(c_2e^t-5\text{.}\) Joining this to the previous segment at \(t=3\) requires

\begin{equation*} -2e^3 = c_2 e^3 - 5 \Rightarrow c_2 = 5e^{-3} -2\text{,} \end{equation*}

so that

\begin{equation*} x_{[3,6]}(t) = (5e^{-3} -2)e^t - 5 = -2 e^t + 5(e^{t-3}-1)\text{.} \end{equation*}

Finally, for \(t \gt 6\) the ODE is homogeneous again, so the general solution here is \(c_3e^t\text{.}\) Continuity at \(t=6\) requires

\begin{equation*} -2 e^6 + 5(e^3-1) = c_3e^6 \Rightarrow c_3 = -2 + 5(e^{-3}-e^{-6})\text{.} \end{equation*}

Hence the last phase of the solution is

\begin{equation*} x_{[6,\infty)}(t) = -2e^t + 5(e^{t-3} - e^{t-6})\text{.} \end{equation*}

Next imagine that we use a window function as the forcing term over ever shortening times, but in such a way that the integral under the curve remains constant. It's analogous to delivering a dose of medication via a slow IV drip versus injecting the same dose all at once directly. Define

\begin{equation*} \delta_\epsilon(t) = \frac{H(t)-H(t-\epsilon)}{\epsilon}\text{.} \end{equation*}

The area under the \(\delta_\epsilon(t)\) curve equals one for all \(\epsilon \gt 0\text{.}\) We now ask what happens as the window width \(\epsilon\) approaches zero.

Definition 2.7.5. Impulse response.

The impulse response of the linear operator \(\mathcal{A}[x]=x'-ax\) is the limit as \(\epsilon\to 0\) of the solution to \(\mathcal{A}[x]=\delta_\epsilon(t)\) with \(x(0)=0\text{.}\)

We often write the impulse response as the solution of \(x'-ax=\delta(t)\text{,}\) where \(\delta(t)\) is an impulse or a delta function. (A math pedant will never fail to remind you that \(\delta(t)\) isn't actually a function. Let's not go there.)

The solution to \(x'-ax=\delta_\epsilon(t)\text{,}\) \(x(0)=0\) is easily found for a constant \(a\) using the method described above, resulting in

\begin{equation*} x_\epsilon(t) = \frac{1-e^{-a\epsilon}}{a\epsilon} e^{at}\text{,} \end{equation*}

for \(t\gt \epsilon\text{.}\) As \(\epsilon\to 0\text{,}\) this is the only part of the solution we care about. Using L'Hôpital's Rule,

\begin{equation*} \lim_{\epsilon\to 0} x_\epsilon(t) = e^{at}, \quad t \gt 0\text{.} \end{equation*}

Hence the constant-coefficient impulse response is simply \(e^{at}\text{.}\) This is the same as the solution of the homogeneous \(x'-ax=0\) with initial value one rather than zero. This remarkable observation generalizes as follows.

Fact 2.7.6.

The solution of \(x'-a(t)x=f(t) + k \delta(t-T)\text{,}\) \(x(0)=x_0\text{,}\) where \(a(t)\) and \(f(t)\) are continuous, satisfies

\begin{equation*} x'-a(t)x=f(t), \; x(0)=x_0 \end{equation*}

for \(t\lt T\text{,}\) and

\begin{equation*} x'-a(t)x=f(t), \; x(T)=x(T^-)+k \end{equation*}

for \(t\gt T\text{,}\) where \(x(T^-)=\lim_{t\to T^-} x(t)\) comes from the first stage of the solution. If \(T=0\text{,}\) then

\begin{equation*} x'-a(t)x=f(t), \; x(0)=x_0+k \end{equation*}

for \(t\gt 0\text{.}\)

In words, an impulse in a first-order linear equation is equivalent to an instantaneous jump in the value of the solution.

Example 2.7.7.

Suppose we solve \(x'+5x=3\delta(t-2)\text{,}\) with \(x(0)=1\text{.}\) For \(0\lt t \lt 2\text{,}\) the equation is homogeneous and \(x(t)=e^{-5t}\text{.}\) Then the solution receives an instantaneous kick from \(e^{-10}\) to \(e^{-10}+3\) and continues evolving homogeneously. So for \(t\gt 2\text{,}\)

\begin{equation*} x(t) = (e^{-10}+3) e^{-5(t-2)} = e^{-5t} + 3 e^{-5(t-2)}\text{.} \end{equation*}

In fact we could write the solution at all times in one formula using step function notation:

\begin{equation*} x(t) = e^{-5t} + 3 H(t-2) e^{-5(t-2)}\text{.} \end{equation*}

Engineering problems are usually expressed in terms of physical units. In math we often like to go in the opposite direction, through a process called nondimensionalization. For \(x'=ax+f\) with positive \(a\text{,}\) it's as simple as defining a new dimensionless time variable \(T=at\text{,}\) because then

\begin{equation*} \frac{dx}{dt} = \frac{dx}{dT}\cdot \frac{dT}{dt} = a\frac{dx}{dT}\text{,} \end{equation*}

from which we get

\begin{equation*} \frac{dx}{dT} = x(T) + \frac{1}{a} f(T/a) = x(T) + F(T)\text{,} \end{equation*}

where \(F(T)=f(T/a)/a\text{,}\) unlike \(f\text{,}\) has the same units as \(x\text{.}\) (For negative \(a\text{,}\) we would use \(T=-at\) and get \(x'=-x+F\text{.}\))

The conclusion is that if we measure time as multiples of the characteristic time \(1/a\text{,}\) then the equation has no remaining coefficient in it. That simplifies the analysis and formulas we derive in terms of the new variables, though of course if we go back to the original variables we must get the same result as before. It may not be clear here that this transformation is worth the bother. But later we will have an equation with several dimensional coefficients, and reducing their number makes that equation a lot easier to talk about.
