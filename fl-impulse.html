Section 2.7 Steps and impulses

¶

The following function is very useful in some modeling situations.

Definition 2.7.1. Unit step function.

The unit step function or Heaviside function \(H(t)\) is defined to be zero for \(t\lt 0\) and one for \(t\ge 0\text{.}\)

The unit step function represents throwing on an idealized switch. To have the switch turn on at an arbitrary time \(T\text{,}\) we use \(H(t-T)\text{.}\) It's worth mentioning here the related idea of a window function, which turns on and then back off. To get a function that is one only between times \(S\) and \(T\text{,}\) for example, we can use \(H(t-S)-H(t-T)\text{.}\)

A key fact to keep in mind about step inputs is:

Fact 2.7.2.

If \(x(t)\) solves \(x'-a(t)x=H(t-T)f(t-T)\text{,}\) where \(a(t)\) and \(f(t)\) are continuous, then \(x(t)\) is continuous.

The reason for writing the forcing function as \(H(t-T)f(t-T)\) here is that it represents “do nothing until time \(T\text{,}\) then activate \(f\) as if starting from time zero”.

Subsection 2.7.1 Solution with piecewise constant forcing

¶

To keep things simple, we will consider problems \(x'-ax=f(t)\) in which \(a\) is constant and the forcing is piecewise constant, that is,

\begin{equation*} f(t) = k_1 H(t-T_1) + k_2 H(t-T_2) + \cdots + k_m H(t-T_m)\text{.} \end{equation*}

We will exploit the linearity of the problem. Returning to our operator notation \(\opA[x]=x'-ax\text{,}\) we solve the following problems:

\begin{align*} \opA[x_h] \amp= 0, \quad x_h(0)=x_0 , \\ \opA[x_1] \amp = H(t-T_1),\quad x_1(0)=0, \\ \amp \vdots \\ \opA[x_m] \amp = H(t-T_m),\quad x_m(0)=0. \end{align*}

Then we set \(x(t)=x_h(t)+k_1 x_1(t)+\cdots+k_m x_m(t)\text{.}\) As a result, \(x(0)=x_0+0+\cdots+0=x_0\) and

\begin{align*} \opA[x] \amp = \opA[x_h] + k_1 \opA[x_1] + \cdots + k_m \opA[x_m] \\ \amp = 0 + k_1[H(t-T_t)] + \cdots + k_m[H(t-T_m)], \end{align*}

which shows that \(x\) solves the original IVP. This technique of adding together solutions to linear equations is called superposition.

Really there are only two unique subproblems to solve. The homogeneous part is \(x_h(t)=e^{a t}x_0\text{.}\) To solve a step problem \(\opA[x_j]=H(t-T_j)\) with zero initial condition, we first note that \(x_j(t)=0\) for \(0\le t \le T_j\text{,}\) since there is no forcing over that time interval. Starting at time \(T_j\) we have

\begin{equation*} \opA[x_j]=1, \quad x_j(T_j)=0\text{.} \end{equation*}

This can be solved easily using undetermined coefficients with a constant particular solution. The result is \(x_j(t)=(e^{a (t-T_j)} - 1)/a\) for \(t \gt T_j\text{.}\) Putting both time intervals together, we get

\begin{equation*} x_j(t) = \frac{1}{a} H(t-T_j) (e^{a(t-T_j)} - 1)\text{.} \end{equation*}

We now have everything we need.

Example 2.7.3.

Let's solve \(x'-\frac{1}{10}x=w(t)\text{,}\) \(x(0)=-2\text{,}\) where \(w(t)\) is a window function that is 5 for \(3 \lt t \le 6\) and zero elsewhere. We can write the problem as

\begin{equation*} x' - \frac{1}{10}x = 5[H(t-3) - H(t-6)] = 5H(t-3) - 5H(t-6), \quad x(0)=-2\text{.} \end{equation*}

The solution therefore consists of three terms:

\begin{equation*} x(t) = -2 e^{t/10} + 50 H(t-3) (e^{(t-3)/10}-1) - 50 H(t-6)(e^{(t-6)/10}-1)\text{.} \end{equation*}

The first term is the homogeneous (or free) response, and the others switch on as the forcing changes its value. A piecewise expression of the solution is

\begin{equation*} x(t) = \begin{cases} -2e^{t/10}, \amp 0 \le t \lt 3, \\ -2e^{t/10} + 50(e^{(t-3)/10}-1), \amp 3 \le \lt 5, \\ -2e^{t/10} + 50(e^{(t-3)/10}-e^{(t-6)/10}), \amp 5 \le t \end{cases}\text{.} \end{equation*}

Subsection 2.7.2 Impulse forcing

¶

Next imagine that we use a window function as the forcing term over ever shortening times, but in such a way that the integral under the curve remains constant. It's analogous to delivering a dose of medication via a slow IV drip versus injecting the same dose all at once directly. Define

\begin{equation*} \delta_\epsilon(t) = \frac{1-H(t-\epsilon)}{\epsilon}\text{.} \end{equation*}

The area under the \(\delta_\epsilon(t)\) curve equals one for all \(\epsilon \gt 0\text{.}\) We now ask what happens as the window width \(\epsilon\) approaches zero.

Definition 2.7.5. Impulse response.

The impulse response of the linear operator \(\mathcal{A}[x]=x'-ax\) is the limit as \(\epsilon\to 0\) of the solution to \(\mathcal{A}[x]=\delta_\epsilon(t)\) with \(x(0)=0\text{.}\)

We often write the impulse response as the solution of \(x'-ax=\delta(t)\text{,}\) where \(\delta(t)\) is an impulse or a delta function. (A math pedant will never fail to remind you that \(\delta(t)\) isn't actually a function. Let's not go there.)

The solution to \(x'-ax=\delta_\epsilon(t)\text{,}\) \(x(0)=0\) is easily found for a constant \(a\) using the method described above, resulting in

\begin{equation*} x_\epsilon(t) = \frac{1}{a \epsilon} \left[ (e^{at}-1) - H(t-\epsilon) (e^{a(t-\epsilon)}-1) \right]\text{.} \end{equation*}

As \(\epsilon\to 0\text{,}\) we only care about what happens for \(t\gt \epsilon\text{,}\) which gives

\begin{equation*} x_\epsilon(t) = \frac{e^{a t}}{a \epsilon} \left( 1 - e^{-a\epsilon} \right), \quad t \gt \epsilon\text{.} \end{equation*}

Using L'Hôpital's Rule,

\begin{equation*} \lim_{\epsilon\to 0} x_\epsilon(t) =e^{at} \left[ \lim_{\epsilon\to 0} \frac{1 - e^{-a\epsilon}}{a \epsilon}\right] = e^{at}, \quad t \gt 0\text{.} \end{equation*}

Hence the constant-coefficient impulse response is simply \(e^{at}\text{.}\) This is the same as the solution of the homogeneous \(x'-ax=0\) with initial value one rather than zero. This remarkable observation generalizes as follows.

Fact 2.7.6.

The solution of \(x'-a(t)x=f(t) + k \delta(t-T)\text{,}\) \(x(0)=x_0\text{,}\) where \(a(t)\) and \(f(t)\) are continuous, satisfies

\begin{equation*} x'-a(t)x=f(t), \; x(0)=x_0 \end{equation*}

for \(t\lt T\text{,}\) and

\begin{equation*} x'-a(t)x=f(t), \; x(T)=x(T^-)+k \end{equation*}

for \(t\gt T\text{,}\) where \(x(T^-)=\lim_{t\to T^-} x(t)\) comes from the first stage of the solution. If \(T=0\text{,}\) then

\begin{equation*} x'-a(t)x=f(t), \; x(0)=x_0+k \end{equation*}

for \(t\gt 0\text{.}\)

In words, an impulse in a first-order linear equation is equivalent to an instantaneous jump in the value of the solution. As before, it's easy to use superposition to get a solution for the case of constant coefficients,

\begin{equation*} x' - ax = k \delta(t-T), \quad x(0)=x_0\text{.} \end{equation*}

The homogeneous part is, as before, \(x_h(t)=e^{a t}x_0\text{.}\) The forced response is the solution of

\begin{equation*} x' - ax = \delta(t-T), \quad x(0)=0\text{,} \end{equation*}

which, similarly to the derivation above, is \(x_d(t)=H(t-T) e^{a (t-T)}\text{.}\) Then the complete solution is the superposition

\begin{equation*} x_h(t) + k x_d(t) = e^{a t}x_0 + k H(t-T)e^{a (t-T)}\text{.} \end{equation*}

This technique extends naturally to multiple impulses of different amplitudes--in fact, to any linear combination of steps and impulses.

Example 2.7.7.

To solve \(x'+5x=3\delta(t-2)\text{,}\) with \(x(0)=1\text{,}\) we combine

\begin{equation*} x(t) = x_h(t) + 3H(t-2)x_d(t-2) = e^{-5 t} + 3H(t-2)e^{-5(t-2)}\text{.} \end{equation*}

Exercises 2.7.3 Practice for step and impulse forcing

¶

There are many different ways to deliver a drug to a person's bloodstream. For instance, we could use a single dose or multiple smaller doses. In first-order pharmacokinetics we assume that in the absence of an external input, the concentration of a drug in the bloodstream is cleared exponentially like \(x'=-r x\) for a rate constant \(r\text{.}\) We will also model an intravenous drip as a piecewise constant forcing function.

1. One dose.

Suppose the initial serum drug concentration \(x(t)\) is initially zero when an IV is given at a constant rate of 12 units per hour for 30 minutes. An IVP that models this case is

\begin{equation*} x'=-r x + 12 \left[ 1-H(t-0.5) \right], \quad x(0)=0, \end{equation*}

where \(t\) is measured in hours. Solve it to find the concentration of the drug over time.

Answer

\(\frac{12}{r}\left[ 1-e^{-r t} - H(t-0.5)\bigl(1-e^{-r(t-0.5)} \bigr)\right]\)

2. Four doses.

If the drug is given in four doses, each given at 3 units per hour over five minutes and separated by six hours, the model becomes

\begin{equation*} x' + r x = 3 \sum_{n=0}^3 \left[H(t-6n)-H\Bigl(t-6n-\tfrac{1}{12}\Bigr)\right]. \end{equation*}

Solve it for \(x(t)\text{.}\)

Answer
\begin{equation*} \frac{3}{r} \sum_{n=0}^3 H(t-6n) \Bigl(1 - e^{-r(t-6n)}\Bigr) - H\bigl(t-6n-\tfrac{1}{12}\bigr) \Bigl(1 - \exp\bigl(-r(t-6n-\tfrac{1}{12})\bigr) \Bigr) \end{equation*}

An alternative to an IV drip is an injection. This takes so short a time that we may as well model it as an impulse.

3. One injection.

Starting from an initial drug-free state, suppose we deliver an injection of 6 units. The model is now

\begin{equation*} x'+r x = 6\delta(t), \quad x(0)=0. \end{equation*}

What is the resulting \(x(t)\text{?}\)

Answer

\(6e^{-r t}\)

4. Four injections.

If we deliver four injections every 6 hours, each with 3 units of the drug, the model becomes

\begin{equation*} x' + r x = 3 \sum_{n=0}^3 \delta(t-6n), \quad x(0)=0. \end{equation*}

Solve it for \(x(t)\text{.}\)

Answer
\begin{equation*} 3\sum_{n=0}^3 H(t-6n) e^{-r (t-6n)} \end{equation*}
