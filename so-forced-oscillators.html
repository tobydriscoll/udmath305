Section 2.4 Exponentially forced oscillators

¶

This and the next section cover 2.4 in the text.

We now consider problems in the form \(Ay''+By'+Cy=f(t)\text{,}\) where the coefficients are constant, \(A\) and \(C\) are positive (oscillation), and the driving force is exponential, \(f(t)=e^{rt}\) for constant \(r\text{.}\)

There are plenty of details to sift through, but the essence is quite simple. The general solution of this problem has two contributions, \(y=y_n+y_p\text{.}\) The first, \(y_n\text{,}\) is the null solution to the associated homogeneous equation \(Ay''+By'+Cy=0\text{,}\) and the second, \(y_p\text{,}\) is any particular solution of the original problem. To see how this works, plug in:

\begin{align*} Ay'' + By' + Cy \amp = A(y_n+y_p)'' + B(y_n+y_p)' + C(y_n+y_p) \\ \amp = (Ay_n''+By_n'+Cy_n) + (Ay_p''+By_p'+Cy_p) \\ \amp = 0 + f = f\text{.} \end{align*}

The null solution contains two arbitrary constants that are used to soak up the initial conditions. We add that the particular solution is not unique, so the complete representation is not either. However, all such expressions are equivalent in the end, and the solution is always expressible in this form.

We covered finding the null solution in detail in Section 2.3. So the particular solution is our first item of business.

Subsection 2.4.1 Exponential response

¶

For \(f(t)=e^{rt}\) we let \(y_p=Y e^{rt}\) and plug into the ODE:

\begin{equation*} (Ar^2 + Br + C)Ye^{rt} = e^{rt}\text{.} \end{equation*}

This is identically satisfied if we choose

\begin{equation*} Y = \frac{1}{Ar^2 + Br + C}\text{.} \end{equation*}

This is called the transfer function. It's constant in time, but a function of the forcing exponent \(r\text{.}\) In fact we'll switch to using \(s\) in the transfer function (as we will see, it's all relatable to Laplace transforms).

Example 2.4.1

Let's solve \(y''+5y'+6y=4 e^{-t}\text{,}\) with \(y(0)=0\text{,}\) \(y'(0)=1\text{.}\) In stages:

  1. Null solution. The characteristic equation of \(y''+5y'+6y=0\) is \(s^2+5s+6=0\text{,}\) which has roots \(s_1=-2\text{,}\) \(s_2=-3\text{.}\) Thus \(y_n=c_1 e^{-2t}+c_2e^{-3t}\text{.}\)

  2. Particular solution. The transfer function is \(Y(s)=1/(s^2+5s+6)\text{,}\) and \(Y(-1)=1/2\text{.}\) Hence \(y_p=\frac{1}{2}\cdot 4 e^{-t}=2e^{-t}\text{.}\)

  3. Complete solution. \(y=y_n+y_p=c_1 e^{-2t}+c_2e^{-3t} + 2 e^{-t}\)

  4. Initial values. We get \(0=y(0)=c_1+c_2+2\) and \(1=y'(0)=-2c_1-3c_2-2\text{.}\) This implies \(c_1=-3\text{,}\) \(c_2=1\text{.}\) Finally, \(y=-3 e^{-2t}+e^{-3t} + 2 e^{-t}\text{.}\)

You may have noticed that the denominator of the transfer function is simply the characteristic polynomial of the homogeneous problem. That is always the case. The textbook uses \(Y(s)=1/P(s)\) in many spots to emphasize the connection.

You should be getting comfortable with the idea that exponential forcing includes sin and cos! We are simply interested in the transfer function for imaginary exponents, \(Y(i\omega)\) followed by the imaginary or real part of \(Ye^{i\omega t}\text{.}\) There is some complex-number-fu required here.

Example 2.4.2

Consider \(y''+y'=\sin(2t)\text{.}\) Since the forcing term is the imaginary part of \(e^{2it}\text{,}\) we use

\begin{equation*} Y(2i) = \frac{1}{(2i)^2+2i} = \frac{1}{2i-4}\text{.} \end{equation*}

We need to take the imaginary part of \(Y(2i)e^{2it}\) to get the particular solution we seek. The course of least resistance is to make the denominator of \(Y\) real by using a complex conjugate:

\begin{equation*} Y(2i) = \frac{1}{2i-4}\cdot \frac{2i+4}{2i+4} = \frac{2i+4}{4+16} = \frac{i+2}{10} = \frac{1}{5} + i\frac{1}{10}\text{.} \end{equation*}

There are two contributions to each of the real and imaginary parts of the product:

\begin{align*} Y(2i)e^{2it} \amp = \left(\frac{1}{5} + i\frac{1}{10}\right) \bigl( \cos(2t) + i \sin(2t) \bigr)\\ \amp = \Bigl[ \cdots \Bigr] + i \Bigl[ \frac{1}{5}\sin(2t) + \frac{1}{10}\cos(2t) \Bigr]\text{.} \end{align*}

(We don't need the real part, inside the first set of brackets.) Thus \(y_p = [2\sin(2t)+\cos(2t)]/10\text{.}\)

In retrospect we might have come at this problem from an entirely real standpoint. We could posit \(y_p=A\cos(2t) + B\sin(2t)\) and then plug in to get some algebraic conditions to determine \(A\) and \(B\text{.}\) There's still a fair amount of algebra and plenty of chances to make silly errors (see the derivation of formula (23) in section 2.4 of the textbook). I recommend that you give the complex method a fair try before retreating to the familiarity of the real-only method.

Subsection 2.4.2 Resonance

¶

Time for some deja vu. When we solved \(y'-at=e^{ct}\) in Subsection 1.4.4, we found that we got different formulas in the case \(c=a\text{.}\) We also needed a special formula for \(y_n\) when we got a double root in Subsection 2.3.1 The same sort of thing happens to \(y_p\) if we force an oscillator at one of the characteristic exponents. The reason is simple: if we try \(y_p=Ye^{rt}\) and \(r\) is a characteristic root, then we get \(0=f(t)\text{,}\) which can't be solved. Equivalently, the transfer function \(Y\) requires taking \(1/0\text{.}\)

Suppose we have a root \(s\) and that \(r\) is close to but not equal to \(s\text{.}\) Then \(y_p=e^{rt}/P(r)\) is a particular solution to the exponential forcing. We can add any null solution to this and it will remain a solution, thus

\begin{equation*} y=\frac{e^{rt}-e^{st}}{P(r)} \end{equation*}

is a valid particular solution as well. Since \(P(s)=0\) by assumption, we can also write

\begin{equation*} y=\frac{e^{rt}-e^{st}}{P(r)-P(s)}\text{.} \end{equation*}

Suddenly this becomes very interesting as \(r\to s\text{;}\) we get an indeterminate \(0/0\text{.}\) Using L'Hopital's rule (but with \(r\) as the variable, not \(t\)), we get the limit

\begin{equation*} y_p = \frac{te^{st}}{P'(s)}\text{,} \end{equation*}

which is a valid particular solution even when forcing exactly at exponent \(s\text{.}\)

Realistically we only care about this effect for harmonic forcing, \(f(t)=e^{i\omega t}\text{,}\) where we call it resonance. Since the characteristic polynomial \(P(s)=as^2+bs+c\) has imaginary roots only if \(b=0\text{,}\) we only see resonance in an undamped oscillator. The resonant particular solution has an amplitude that is linearly growing in time. The derivation above shows that though the solution formula (and unboundedness) is singular at the resonant frequency, the actual behavior is a continuous limit of near-resonant behavior.

Subsection 2.4.3 Gain and lag

¶

By now we have seen multiple times that forcing with \(\cos(\omega t)\) or \(\sin(\omega t)\) leads to solutions with a combination of both terms. We also know that such a result is equivalent to a shifted cosine in amplitude/phase form, and we turn our attention to this interpretation next.

With forcing \(f(t)=e^{i\omega t}\) we get a solution \(y_p=Y(i\omega) e^{i\omega t}\text{.}\) If we write the transfer function in polar form, we can express this as \(y_p = (G e^{-i\alpha }) e^{i\omega t} = Ge^{i(\omega t - \alpha)}\) for positive amplitude \(G\text{,}\) which we call the gain. It depends on the forcing frequency; in fact, \(G(\omega) = |Y(i\omega)|\text{.}\) The \(\alpha\) term is a phase lag that shows how the response may lag behind the driving force. It's usually not as interesting as the gain.

Example 2.4.3

The oscillator \(4y'' + y' + 36y\) has transfer function \(Y(s)=1/(4s^2+s+36)\text{.}\) At \(s=i\omega\) we get

\begin{equation*} Y(i\omega) = \frac{1}{ -4\omega^2 +i\omega + 36}\text{.} \end{equation*}

We can compute the gain using a complex conjugate.

\begin{equation*} G^2 = |Y|^2 = \frac{1}{ 36-4\omega^2 +i\omega} \frac{1}{ 36-4\omega^2 - i\omega} = \frac{1}{(36-4\omega^2)^2+\omega^2}\text{.} \end{equation*}

The gain is maximized when the denominator is minimized. This happens when \(2 (36-4\omega^2)(-8\omega)+2\omega = 0\text{,}\) or \(\omega^2=9-(1/32)\text{.}\) This is just slightly smaller than the natural undamped frequency \(\omega_n^2=36/4=9\text{.}\)

In the undamped limit, resonance produces an infinite gain. Damping reduces the peak and the maximal frequency, at first gradually and then dramatically.
