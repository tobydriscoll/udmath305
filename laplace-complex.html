Section 2.9 Complex and repeated poles

Â¶

In the previous section we saw that it's easy to turn a linear, constant-coefficient ODE into a Laplace transform \(Y(s)\) of a particular solution. The art comes from the inverse problem of recovering \(y(t)\) from there. With the real, simple poles we've seen so far, the algebra is quite modest, but things get dicier when the poles are complex or repeated.

Subsection 2.9.1 Imaginary poles

A transform having pure imaginary poles can be done easily if there are only two of them. As always, poles occur in conjugate pairs, so we first consider

\begin{equation*} Y(s) = \frac{A+Bs}{s^2+\omega^2} \end{equation*}

for some real constants \(A,B,\omega\text{.}\) This inverts easily to \(y(t)=(A/\omega)\sin(\omega t) + B\cos(\omega t)\text{,}\) and there is little else to say.

If \(Y\) has an additional real pole as well at \(s=c\text{,}\) then it can be written in the partial fraction form

\begin{equation*} Y(s) = \frac{A+Bs}{s^2+\omega^2} + \frac{C}{s-c} \text{.} \end{equation*}

This too is easy to invert once the constants are known. Finding them is tedious, but manageable.

Example 2.9.1

The undamped oscillator \(y''+16y=40e^{-2t}\) has a solution whose transform is

\begin{equation*} Y(s) = \frac{40}{(s+2)(s^2+16)\text{.} \end{equation*}

Hence

\begin{equation*} \frac{A+Bs}{s^2+16^2} + \frac{C}{s+2} = \frac{40}{(s+2)(s^2+16)\text{.} \end{equation*}

Clearing the denominators gives the identity

\begin{equation*} (A+Bs)(s+2) + C(s^2+16) = 40\text{.} \end{equation*}

We need three equations to determine the unknown coefficients. Setting \(s=-2\) quickly gives \(20C=40\text{,}\) so \(C=2\text{.}\) Then setting \(s=\pm 4\) gives the system

\begin{align*} (A+4B)(6) \amp = 40 - 64 = -24\\ (A-4B)(-2) \amp = -24\text{,} \end{align*}

or \(A+4B=-4\) and \(A-4B=12\text{.}\) We can add and subtract these equations to get \(2A=8\text{,}\) \(8B=-16\text{,}\) so finally

\begin{equation*} Y(s) = \frac{4-2s}{s^2+16^2} + \frac{2}{s+2} \text{.} \end{equation*}

Hence

\begin{equation*} y(t) = \sin(4t) - 2\cos(4 t) + 2e^{-2t}\text{.} \end{equation*}

Finally, consider the driven unforced oscillator \(y''+\omega_n^2 y= \cos(\omega t)\text{.}\) It leads to

\begin{equation*} Y(s) = \frac{s}{(s^2+\omega_n^2)(s^2+1)} = \frac{1}{\omega_n^2-\omega^2} \left( \frac{s}{s^2+\omega^2} - \frac{s}{s^2+\omega_n^2} \right)\text{,} \end{equation*}

which is easily inverted into a combination of \(\cos(\omega t)\) and \(\cos(\omega_n t)\text{.}\) However, at this point we should probably just be using the method of undetermined coefficients instead.

Subsection 2.9.2 Repeated poles

We know by now that repeated poles in \(Y(s)\) arise in two ways: from repeated characteristic roots (i.e. poles of the transfer function), or perfect resonance in the forcing function. We can anticipate the effect on the transform, because we know the answer in the time domain is something like \(te^{rt}\text{.}\)

In fact if we have any \(f(t)\) with transform \(F(s)\text{,}\) then from the transform definition we compute

\begin{align*} F'(s) \amp = \frac{d}{ds} \int_0^\infty f(t) e^{-st}\, dt\\ \amp = -\int_0^\infty t f(t) e^{-st}\, dt\\ \amp = - \mathcal{L}[ tf(t) ]\text{.} \end{align*}

Take a moment to appreciate this. Earlier we found that a derivative in \(t\) corresponds to multiplying by \(s\text{.}\) Up to a change in sign, a derivative in \(s\) corresponds to multiplying by \(t\text{!}\)

Example 2.9.2

The undamped oscillator \(y''+9y=\sin(3t)\) has a perfect resonance. The transform suggests the particular solution

\begin{equation*} Y(s) = \frac{3}{(s^2+9)^2}\text{.} \end{equation*}

This doesn't look like anything easy. However, note that

\begin{equation*} sY(s) = \frac{3s}{(s^2+9)^2} = -\frac{3}{2} \frad{d}{ds} \left[ \frac{1}{s^2+9} \right] \text{.} \end{equation*}

Therefore, \(sY(s)\) is the transform of \(\frac{1}{2} t \sin(3t)\text{.}\) So this must be \(y'(t)\text{.}\) An integration by parts then recovers

\begin{equation*} y(t) = \frac{1}{18} [ \sin(3t) - 3t\cos(3t) ]\text{.} \end{equation*}

If we keep differentiating in \(s\) we find

\begin{equation*} \frac{d^n}{ds^n} \mathcal{L}[ f(t) ] = (-1)^n \mathcal{L}[ t^n f(t) ]\text{.} \end{equation*}

Of interest is the special case of an exponential function \(e^{rt}\text{,}\) whose transform is \(1/s-r\text{:}\)

\begin{equation*} \mathcal{L} \left[ t^n e^{rt} \right] = \frac{n!}{(s-r)^{n+1}} \text{.} \end{equation*}

The even specialer case \(r=0\) allows us to transform any power of \(t\text{,}\) and therefore any polynomial.

Subsection 2.9.3 Complex poles

Complex poles in the transform correspond to complex exponentials in the time donmain. When they occur in conjugate pairs, the entire expression can be cast in real terms. We have

\begin{equation*} \frac{1}{s-(a+ib)} + \frac{1}{s-(a-ib)} = \frac{2(s-a)}{(s-a)^2+b^2} \text{,} \end{equation*}

so the right-hand side above is the transform of

\begin{equation*} e^{(a+ib)t} + e^{(a-ib)t} = 2 e^{at} \cos( b t)\text{.} \end{equation*}

In similar fashion we can show that \(e^{at} \cos( b t)\) transforms to \(b/[(s-a)^2+b^2]\text{.}\) In principle we can always find quadratic denominators like these for a partial fraction conversion.

Example 2.9.3

Suppose \(y''+2y'+10y=6\text{.}\) The very particular solution has \(Y(s)=6/(s^2+2s+10)\text{.}\) We can use the trick of completing the square to write this as \(Y(s)=6/[(s-1)^2+9]\text{,}\) and thus \(y(t) = 2e^{-t}\sin(3t)\text{.}\)

Things are much more complicated for a harmonically forced case such as \(y''+2y'+10y=\cos(2t)\text{.}\) Now

\begin{equation*} Y(s) = \frac{s}{(s^2+4)(s^2+2s+10)}\text{.} \end{equation*}

It is true that we have a decomposition in the form

\begin{equation*} Y(s) = \frac{A+Bs}{s^2+4} + \frac{C+Es}{(s-1)^2+9}\text{,} \end{equation*}

which is eminently invertible. But finding the four coefficients is a pretty heavy lift to do by hand.

The problem with Laplace transforms as a one-stop solver for linear, constant-coefficient problems is that the particular solution we find is always the "very particular" one that includes some terms from the null solution. That is because the poles of \(Y(s)\) include those of the transfer function and the forcing function. It's extra algebra to find the coefficients of the null/transient parts, but that step can't be skipped. Undetermined coefficients is a less sexy but simpler way to solve concrete problems in most cases.
