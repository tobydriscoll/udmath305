Section 1.8 Determinants

¶

You should have seen some \(2\times 2\) and \(3\times 3\) determinants before now, in vector calculus. The \(2\times 2\) case is easy to describe:

\begin{equation*} \det\left( \twomat{a}{b}{c}{d} \right) = \twodet{a}{b}{c}{d} = ad-bc\text{.} \end{equation*}

This can be extended to create a real-valued function for square matrices of any size. The formalities can be complicated, but we are going to use a practical definition.

Here and for the rest of this chapter, we are going to deal with square matrices only.

Definition 1.8.1. Determinant.

If \(\bA\in\Rmn{n}{n}\text{,}\) then

\begin{equation*} \det(\bA) = \sum (-1)^{i+j} a_{ij} \det( \mathbf{M}_{ij} )\text{,} \end{equation*}

where the sum is taken over any row or column of \(\bA\text{,}\) and \(\mathbf{M}_{ij}\) is the matrix that results from deleting row \(i\) and column \(j\) from \(\bA\text{.}\)

The definition, which is called cofactor expansion, is recursive: the \(n\times n\) case is defined in terms of the \((n-1)\times (n-1)\) case, and so on all the way back down to \(2\times 2\text{.}\)

Example 1.8.2.

Using expansion along the first row,

\begin{align*} \begin{vmatrix} 2 \amp 0 \amp -1 \\ -2 \amp 3 \amp -1 \\ 2 \amp 0 \amp 1 \end{vmatrix} \amp = (2) \twodet{3}{-1}{0}{1} - (0) \twodet{-2}{-1}{2}{1} + (-1)\twodet{-2}{3}{2}{0} \\ \amp = 2(3-0) + (-1)(0-6) = 12. \end{align*}

In this case it might have been a tad easier to exploit the zeros by expanding along the second column instead:

\begin{align*} \begin{vmatrix} 2 \amp 0 \amp -1 \\ -2 \amp 3 \amp -1 \\ 2 \amp 0 \amp 1 \end{vmatrix} \amp = -(0) \begin{vmatrix} \cdots \end{vmatrix} + (3) \twodet{2}{-1}{2}{1} - (0)\begin{vmatrix} \cdots \end{vmatrix} \\ \amp = 3(2+2) = 12. \end{align*}

There are a few facts about determinants that are good to know.

Fact 1.8.3.

Let \(\bA\) and \(\bB\) be \(n\times n\text{,}\) and let \(c\) be a scalar. Then

  1. \(\det(c\bA) = c^n \det(\bA)\text{,}\)
  2. \(\det(\bA\bB) = \det(\bA)\det(\bB)\text{,}\)
  3. \(\det(\bA)=0\) if and only if \(\bA\) is singular, and

It's the third one that we will be using. The determinant is often the easiest way to check for singularity of a small matrix by hand.

Subsection 1.8.1 Cramer's Rule

¶

Even though a 2x2 inverse is easy, it's still not the most convenient way to solve a linear system \(\bA\bx=\bb\text{.}\) There is an even faster equivalent shortcut known as Cramer's Rule:

\begin{align*} x_1 \amp = \frac{ \twodet{b_1}{a_{12}}{b_2}{a_{22}} }{ \det(\bA) }\\ x_2 \amp = \frac{ \twodet{a_{11}}{b_1}{a_{21}}{b_2} }{ \det(\bA) }\text{.} \end{align*}

Obviously this does not work if \(\det(\bA)=0\text{.}\) In that case the matrix is singular, and it's best to investigate by row elimination as usual.

Example 1.8.4.

For

\begin{align*} -x + 3y \amp = 1 \\ 3x + y \amp = 7\text{,} \end{align*}

Cramer's Rule says

\begin{align*} x \amp = \frac{ \twodet{1}{3}{7}{1} }{ \det(\bA) }= \frac{ \twodet{1}{3}{7}{1} }{ \twodet{-1}{3}{3}{1} } = \frac{-20}{-10} = 2 \\ y \amp = \frac{ \twodet{-1}{1}{3}{7} }{ \det(\bA) } = \frac{ \twodet{-1}{1}{3}{7} }{ \twodet{-1}{3}{3}{1} } = \frac{-10}{-10} = 1\text{.} \end{align*}
