Section 2.2 Linear ODEs

¶

Linear ODEs are the most fundamental type. If you could only get to ask one yes-or-no question about a new ODE, your question should be, "Is it linear?"

Definition 2.2.1. First-order linear ODE.

A first-order linear ODE is an equation of the form

\begin{equation} \dd{x}{t} = a(t)x + f(t)\text{.}\label{fl-eq-linear}\tag{2.2.1} \end{equation}

We often refer to \(a(t)\) as the coefficient function and \(f(t)\) as the forcing function. Note that it is not necessary for these to be linear functions of time, nor is the solution \(x(t)\) generally a linear function. The linearity being referred to is the fact that there are no terms such as \(\sin(x')\text{,}\) \(e^x\text{,}\) \(xx'\text{,}\) etc.

Subsection 2.2.1 General solutions

¶

All solutions to first-order linear problems share a basic structure. In fact, it's the same structure as the solutions of \(\bA\bx=\bb\text{.}\) In order to describe this, we're going to replace the vectors \(\bx\) and \(\bb\) with the functions \(x(t)\) and \(f(t)\text{,}\) and the matrix \(\bA\) by something analogous for functions.

Definition 2.2.2. Linear operator.

A linear operator \(\opA\) is a rule for transforming functions to other functions, such that

\begin{equation*} \opA[cx(t) + y(t)]=c\opA[x(t)] + \opA[y(t)] \end{equation*}

for all functions \(x,y\) and numbers \(c\text{.}\)

In the first chapter we pointed out that the linear function \(L\) defined by >\(L(\bx)=\bA\bx\) for a given matrix \(\bA\) has the same property. In the context of the first order linear ODE (2.2.1), the operator of interest to us is defined by

\begin{equation} \opA[x] = x' - a(t)x\text{,}\label{fl-eq-linop}\tag{2.2.2} \end{equation}

whose linearity you can easily validate for yourself. We can now express the ODE simply as \(\opA[x]=f\text{.}\)

As with linear systems, we have a special role for the homogeneous linear ODE \(\dd{x}{t} = a(t)x\text{,}\) that is, \(\opA[x]=0\text{,}\) or equation (2.2.1) with zero forcing.

Fact 2.2.3.

If \(x_1(t)\) and \(x_2(t)\) are solutions of \(\opA[x]=0\text{,}\) then so is any linear combination \(c_1x_1 + c_2x_2\) for constants \(c_1,c_2\text{.}\)

Quite simply, this is because

\begin{equation*} \opA[c_1x_1 + c_2x_2] = c_1\opA[x_1] + c_2\opA[x_2]=0+0=0\text{.} \end{equation*}

Clearly, we could extend the linear combination to any number of terms, with the same result. We can make a related observation:

Fact 2.2.4.

If \(x_h\) is a solution of the homogeneous \(\opA[x]=0\) and \(x_p\) is any particular solution of \(\opA[x]=f\text{,}\) then \(x=x_h+x_p\) is also a solution of \(\opA[x]=f\text{.}\)

This is just as easy as the last fact:

\begin{equation*} \opA[x_h+x_p] = \opA[x_h] + \opA[x_p]=0+f=f\text{.} \end{equation*}

As with linear systems, then, we can boil the general solution of the complete problem to finding the most general solution of the homogeneous problem, plus a particular solution of the original problem.

Theorem 2.2.5. General solutions (first-order linear ODEs).

Every solution of the ODE \(\opA[x]=f\) can be written in the form \(x=x_h+x_p\text{,}\) where \(x_h\) is the general solution of \(\opA[x]=0\) and \(x_p\) is any solution of \(\opA[x]=f\text{.}\)
