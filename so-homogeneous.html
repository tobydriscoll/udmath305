Section 4.2 Homogeneous problem

¶

Referring to (4.1.3), the homogeneous equation \(x''+bx'+cx=0\) can be recast as a first-order homogeneous system \(\bu'=\bA\bu\) through the definitions

\begin{equation*} \bu = \twovec{x}{x'},\qquad \bA = \twomat{0}{1}{-c}{-b}\text{.} \end{equation*}

Note that an initial-value problem for the system specifies \(\bu(0)\text{,}\) or equivalently, both \(x(0)\) and \(x'(0)\text{.}\) Hence a second-order scalar problem requires two initial conditions to have a unique solution.

The matrix in this system has the characteristic polynomial

\begin{equation*} |\bA-\lambda\bI| = \twodet{-\lambda}{1}{-c}{-b-\lambda} = \lambda^2 + b\lambda + c\text{.} \end{equation*}

This is easily remembered in terms of the original ODE \(x'' + bx' + cx=0\text{.}\) The eigenvalues can therefore be found from the quadratic formula.

Subsection 4.2.1 Distinct eigenvalues

In the nondefective case, we find the eigenvectors from the nullspaces of \(\bA-\lambda_j\bI\) for \(j=1,2\text{.}\) These have the basis vectors \([1,\,\lambda_j]\text{.}\) Thus the general homogeneous solution is

\begin{equation*} \bu_h(t) = c_1 e^{\lambda_1 t} \twovec{1}{\lambda_1} + c_2 e^{\lambda_2 t} \twovec{1}{\lambda_2}\text{.} \end{equation*}

In terms of the original variable \(x=u_1\text{,}\) this is more simply

\begin{equation} x_h(t) = c_1 e^{\lambda_1 t} + c_2 e^{\lambda_2 t}, \qquad \text{if } \lambda_1\neq\lambda_2\text{.}\label{so-eq-homogdistinct}\tag{4.2.1} \end{equation}

For real problems with complex eigenvalues, the second term in the sum is the conjugate of the first, and we could write instead

\begin{equation*} x_h(t) = \Re \left[ c_1 e^{\lambda_1 t} \right] \end{equation*}

We might want an alternative real-only form. Suppose the eigenvalues are written as \(\alpha \pm i \beta\text{.}\) Starting with

\begin{equation*} e^{(\alpha + i\beta)t} = e^{\alpha t} [\cos(\beta t) + i \sin(\beta t)], \end{equation*}

we can take real and imaginary parts to get independent real solutions. Thus

\begin{equation} x_h(t) = e^{\alpha t} \bigl[ a_1 \cos(\beta t) + a_2 \sin(\beta t) \bigr], \qquad \text{if } \lambda=\alpha \pm i\beta\text{.}\label{so-eq-homogcomplex}\tag{4.2.2} \end{equation}

There is a third equivalent form in the complex case: the phase-amplitude form,

\begin{equation} x_h(t) = R e^{\alpha t} \cos(\beta t - \theta)\text{.}\label{so-eq-homogphaseamp}\tag{4.2.3} \end{equation}

To see the equivalence to (4.2.2), we can use a trig identity on the phase-shifted cosine:

\begin{equation*} R e^{\alpha t} \cos(\beta t - \theta) = R e^{\alpha t} \cos(\theta) \cos(\beta t) + R e^{\alpha t} \sin(\theta) \sin(\beta t). \end{equation*}

We get back to (4.2.2) by choosing

\begin{equation*} R\cos(\theta) = a_1, \qquad R\sin(\theta) = a_2. \end{equation*}

More simply, we can think of \((a_1,a_2)\) as Cartesian coordinates and express the same point in polar form to get \((R,\theta)\text{.}\) Usually the amplitude, \(R=\sqrt{a_1^2+a_2^2}\text{,}\) is of the most interest. Whichever form is chosen for the general solution, it has two real constants that can be determined by the initial conditions, if supplied.

Example 4.2.1.

Let's find the general solution of \(x''+5x'+6x=0\text{.}\) The characteristic polynomial is \(\lambda^2+5\lambda+6\text{,}\) which has roots \(\lambda_1=-2\text{,}\) \(\lambda_2=-3\text{.}\) Thus \(x(t)=c_1 e^{-2t}+c_2e^{-3t}\text{.}\)

Example 4.2.2.

The characteristic polynomial of \(x''-2x'=0\) is \(\lambda^2 - 2\lambda\text{,}\) which has roots \(\lambda_1=0\text{,}\) \(\lambda_2=2\text{.}\) Thus \(x(t)=c_1 + c_2e^{2t}\text{.}\)

Example 4.2.3.

To solve \(3x''+6x'+30x=0\text{,}\) we first should divide through by 3 to get the standard form. Then the characteristic polynomial is \(\lambda^2 + 2\lambda + 10\text{,}\) which has roots \(-1\pm 3i\text{.}\) So \(x(t) = e^{-t}[c_1 e^{3it} + \overline{c_1}e^{-3it}]\text{.}\) Equivalently, \(x(t) = e^{-t}[a_1 \cos(3t) + a_2\sin(3t)]\text{,}\) or \(x(t) = R e^{-t}\cos(3t-\theta)\text{.}\)

Subsection 4.2.2 Repeated eigenvalue

If the companion matrix has a double eigenvalue, then it is defective (unless \(b=c=0\text{,}\) which leaves the truly trivial problem \(x''=0\)). In terms of the original ODE, this occurs when \(b^2=4c\text{,}\) and then \(\lambda_1=\lambda_2=-b/2\text{.}\) Recalling the trick used in Subsection 3.6.2, we define

\begin{equation*} \bB = \bA + \frac{b}{2}\bI = \twomat{b/2}{1}{-c}{-b/2} = \twomat{b/2}{1}{-b^2/4}{-b/2}\text{.} \end{equation*}

You can easily check that \(\bB^2=\bzero\text{.}\) Hence the power series for the matrix exponential has just two terms:

\begin{equation*} e^{t\bA} = e^{-(bt/2)\bI} e^{t\bB} = e^{-bt/2}(\bI + t\bB) \text{.} \end{equation*}

The general solution is \(e^{t\bA}\bc\text{,}\) but again we need only the first row to get the solution for the original \(x=u_1\text{:}\)

\begin{equation*} x_h(t) = e^{-bt/2} \left[ c_1 + t\left(c_2 + c_1\frac{b}{2}\right) \right] = e^{-bt/2} \left( \tilde{c}_1 + c_2 t\right)\text{,} \end{equation*}

where \(\tilde{c}_1\) is just a different arbitrary constant. We can rename the constants and express the result as

\begin{equation} x_h(t) = e^{-bt/2} \left( c_1 + c_2 t\right) , \qquad \text{if } \lambda_1 = \lambda_2 \neq 0.\label{so-eq-homogdefect}\tag{4.2.4} \end{equation}
Example 4.2.4.

The characteristic polynomial of \(x''-4x'+4x=0\) is \(\lambda^2 - 4\lambda + 4\text{,}\) which has a double root at 2. Hence \(x(t)=e^{2t}(c_1+c_2 t)\) is the general solution.
