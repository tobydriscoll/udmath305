Section 4.2 Elimination for linear systems

¶

From Section 4.2.

Whenever we are faced with a linear system of \(n\) equations in \(n\) variables, we can solve it by transforming it step by step into an equivalent system that is easier to solve. This process is most transparent for the two-dimensional case, so we start there.

Subsection 4.2.1 Two variables

Consider the system

\begin{align*} -x + 3y \amp = 1 \\ 3x + y \amp = 7\text{.} \end{align*}

We can use the first equation to eliminate \(x\) from the second. Specifically, we multiply the first equation by \(3\) and add it to the second to get the system

\begin{align*} -x + 3y \amp = 1 \\ 10 y \amp = 10\text{.} \end{align*}

It's now trivial to solve the second equation for \(y=1\text{.}\) With that known, the first equation can be solved to find \(x=-(1-3)=2\text{.}\)

The key to making the elimination work was the zero we forced into the second row of matrix representing the second system:

\begin{equation*} \begin{bmatrix} -1 \amp 3 \\ 0 \amp 10 \end{bmatrix}\text{.} \end{equation*}

This matrix has what we call and upper triangular structure. That structure is our goal, because then we can sort of unzip the solution by starting at the bottom and working our way up.

Subsection 4.2.2 More variables

Here's a system of four equations:

\begin{align*} 2x_1 + 0x_2 + 4x_3 + 3x_4 \amp = 4\\ -2x_1 + 0x_2 + 2x_3 - 13x_4 \amp = 40\\ x_1 + 15x_2 + 2x_3 - 4.5x_4 \amp = 29\\ -4x_1 + 5x_2 - 7x_3 - 10x_4 \amp = 9\text{.} \end{align*}

To begin with, all we really care about are the numbers (coefficients) in these equations. Everything else always remains the same. Let us define

\begin{equation*} \bA = \begin{bmatrix} 2 \amp 0 \amp 4 \amp 3 \\ -2 \amp 0 \amp 2 \amp -13 \\ 1 \amp 15 \amp 2 \amp -4.5 \\ -4 \amp 5 \amp -7 \amp -10 \end{bmatrix}, \qquad \bb = \begin{bmatrix} 4\\ 40\\ 29 \\9 \end{bmatrix}\text{.} \end{equation*}

Now we show how elimination proceeds on this matrix, but using matlab to handle the arithmetic for us.

To summarize, we proceed through the rows \(1,2,\ldots,n-1\text{.}\) For row \(i\) we seek to put zeros below the pivot element in the \((i,i)\) position. To do that we choose a multiplier for each \(k>i\) and add that multiple of row \(i\) to row \(k\text{.}\) The one thing that can stop us is if the pivot element is zero, because we can't find multipliers. In that event we swap row \(i\) with a lower row. After the eliminations are done, the system is in upper triangular form and can be solved by backward substitution.

Subsection 4.2.3 Breakdown \(=\) singularity

We have left one logical possibility unaddressed. What if we not only have a zero pivot, but all the candidate pivots below it are zero too? As a practical matter, this form of "breakdown" is benign; there is no need to eliminate the current variable because it is already eliminated! We can move down to the next row in the elimination process. To put it another way, we will end up with an upper triangular matrix with a zero in some \((i,i)\) position (on the diagonal of the matrix).

But there is a crucial theoretical implication too.

Theorem 4.2.1 Elimination breakdown equals singularity

The elimination process is unable to find a nonzero pivot at some point in the algorithm if and only if the original system matrix is singular.

This behavior is manifested once we move on to the backward substitution phase. If some \((i,i)\) element is zero in the triangular matrix, then we will be unable to solve for \(x_i\) when its turn comes. At that point, either the system holds true regardless of the value of \(x_i\) (which makes it a "free" variable), or the system is inconsistent and has no solution.

Subsection 4.2.4 Cramer's rule

There is a shortcut to avoid the effort of elimination for small systems. It's easiest to show and use in the case of two variables, such as

\begin{align*} -x + 3y \amp = 1 \\ 3x + y \amp = 7\text{.} \end{align*}

Cramer's rule says

\begin{align*} x \amp = \frac{ \twodet{1}{3}{7}{1} }{ \twodet{-1}{3}{3}{1} } = \frac{-20}{-10}\\ y \amp = \frac{ \twodet{-1}{1}{3}{7} }{ \twodet{-1}{3}{3}{1} } = \frac{-10}{-10}\text{.} \end{align*}
