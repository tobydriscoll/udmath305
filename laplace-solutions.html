Section 2.8 Laplace transform methods

Â¶

The correspondence between \(L[y]=Ay''+By'+Cy\) and \(P(s)=As^2+Bs+C\) is by now obvious. It came by way of assuming \(y(t)=Y(s)e^{st}\text{,}\) and we get \(Y(s)=1/P(s)\text{.}\) Factoring \(P\) tells us what we need to know. For example, if its roots are \(-2\) and \(-4\text{,}\) then we know that \(y(t)\) is a combination of \(e^{-2t}\) and \(e^{-4t}\text{.}\) If an exponential, polynomial, step, or impulse forcing term is present, it contributes to the solution in similarly predictable ways. The only mystery to work out is the coefficients.

There is a method of expressing all this knowledge called the Laplace transform. Given (almost) any function \(f(t)\text{,}\) there is a doppelganger function \(F(s)\) defined by an integral,

\begin{equation} F(s) = \int_0^\infty f(t) e^{-st}\, dt\text{.}\label{so-lap-lxform}\tag{2.8.1} \end{equation}

In this business, time starts at \(t=0\text{,}\) and nothing before then matters. One thing to note right away is that the Laplace transform is a linear operator. It's often written using a script L, as in \(F(s)=\mathcal{L}[f(t)]\text{.}\) The usual convention is that a lowercase function of \(t\) transforms to its uppercase namesake as a function of \(s\text{.}\)

Subsection 2.8.1 Derivatives

Applying integration by parts to the definition reveals the main reason Laplace transforms are useful in ODEs.

\begin{align*} \mathcal{L}[y'(t)] \amp = \int_0^\infty y'(t) e^{-st}\, dt \\ \amp = \left[ y(t)e^{-st}\right]_0^\infty - \int_0^\infty (-s) y(t) e^{-st}\, dt \\ \amp = -y(0) + s Y(s)\text{.} \end{align*}

Calculus (differentiation) is turned into algebra (multiplication). We can apply this rule repeatedly to transform higher derivatives:

\begin{equation*} \mathcal{L}[y''(t)] = s \mathcal{L}[y'(t)] - y'(0) = s^2Y(s) - sy(0) - y'(0)\text{.} \end{equation*}

So here is the plan of attack. Given an ODE \(Ay''+By'+Cy=f\text{,}\) we transform everything in sight (using linearity!) to get

\begin{equation*} A[s^2Y(s) - sy(0)-y'(0)] + B[sY(s)-y(0)] + CY(s) = F(s)\text{.} \end{equation*}

Now solve for the transform of the solution:

\begin{equation*} Y(s) = \frac{ F(s) + (As+B)y(0) - Ay'(0)}{As^2 + Bs + C}\text{.} \end{equation*}

Note that if we want just a particular solution, we can use \(y(0)=y'(0)=0\text{.}\) In any case, "all" we have to do know is to compute the inverse transform, that is, find \(y(t)\) given \(Y(s)\text{.}\) This is less straightforward than computing the forward transform, unfortunately. But we can make it managable in much the same way you learned indefinite integration, by learning some canonical cases and then matching patterns.

Subsection 2.8.2 Canonical transforms

  1. Exponential functions. This is straightforward to calculate from definition. For any complex number \(z\text{,}\)
    \begin{equation*} {\cal L}[e^{z t}] = \int_0^\infty e^{(z-s)t}\, dt = \left[ -\frac{1}{s-z} e^{(z-s)t} \right]_0^\infty = \frac{1}{s-z}\text{.} \end{equation*}
    (To make this valid requires that \({\text Re}(z)\lt s\text{,}\) but we are going to ignore such technicalities.) An important special case is when \(z=0\text{:}\) \({\cal L}[1]=1/s\text{.}\)
  2. Cos and sin. We can use an imaginary number in the exponential and take the real or imaginary part of the result. Thus,
    \begin{equation*} {\cal L}[\cos(\omega t)] = \text{Re}\left( \frac{1}{s-i\omega} \frac{s+i\omega}{s+i\omega} \right) = \frac{s}{s^2+\omega^2}\text{,} \end{equation*}
    and
    \begin{equation*} {\cal L}[\sin(\omega t)] = \text{Im}\left( \frac{1}{s-i\omega} \frac{s+i\omega}{s+i\omega} \right) = \frac{\omega}{s^2+\omega^2}\text{.} \end{equation*}

Subsection 2.8.3 The first-order case

Let's go back and revisit an old first-order linear friend, \(y'-ay=e^{ct}\text{.}\) Transforming both sides, we get

\begin{equation*} [sY(s)-y(0)] - aY(s) =\frac{1}{s-c}\text{,} \end{equation*}

and thus

\begin{equation*} Y(s) = \frac{y(0)}{s-a} + \frac{1}{(s-c)(s-a)}\text{.} \end{equation*}

The first part corresponds to \(y_n(t)=y(0)e^{at}\text{,}\) which is the null solution. To invert the transform of the second term, we need to express it using partial fractions. The text has a clever mnemonic for this:

\begin{equation*} \frac{1}{(s-c)(s-a)} = \frac{1}{(a-c)(s-a)} + \frac{1}{(s-c)(c-a)}\text{.} \end{equation*}

Thus we get a sum of two exponentials,

\begin{equation*} y_v(t) = \frac{1}{c-a}\left( e^{ct}-e^{at} \right) \text{.} \end{equation*}

This is the "very particular" solution that has \(y_v(0)=0\text{.}\)

There's no new result here, though the process is quite streamlined. The values \(s=a\) and \(s=c\) that are roots of the denominator of \(Y(s)\) are called poles of the transform. They signal clearly and quickly that every solution is a combination of the exponentials \(e^{at}\) and \(e^{ct}\text{.}\)

Subsection 2.8.4 Second-order equations

Let's break down what happens in a fundamental case, \(y''-y=e^{rt}\text{.}\) Transforming both sides gives \([s^2Y(s) - sy(0)-y'(0)] - Y(s) = 1/(s-r)\text{.}\) For reasons to be clarified later, let's skip to the "very particular" part with \(y_v(0)=y_v'(0)=0\text{,}\) for which

\begin{equation*} Y(s) = \frac{1}{(s^2-1)(s-r)} = \frac{1}{(s+1)(s-1)(s-r)}\text{.} \end{equation*}

This time there are three poles. Again the book's trick works well here:

\begin{equation*} \frac{1}{(s+1)(s-1)(s-r)} = \frac{1}{(s+1)(-1-1)(-1-r)} + \frac{1}{(1+1)(s-1)(1-r)} + \frac{1}{(r+1)(r-1)(s-r)} \text{.} \end{equation*}

It should be apparent that this works only if \(r\neq -1\) and \(r \neq 1\) (i.e., no resonance). From here we get three exponentials:

\begin{equation*} y_v(t) = \frac{1}{2(1+r)}e^{-t} + \frac{1}{2(1-r)}e^{t} + \frac{1}{r^2-1}e^{rt}\text{.} \end{equation*}

From here we can write out a general solution. The characteristic roots are \(\pm 1\text{,}\) so

\begin{equation*} y(t) = c_1 e^{-t} + c_2 e^{t} + y_v(t)\text{.} \end{equation*}

If desired, it's easy to solve for the constants in this, because \(y_v\) contributes nothing:

\begin{align*} y(0) \amp = c_1 + c_2 \\ y'(0) \amp = -c_1 + c_2 \text{.} \end{align*}

That's a lot less work than doing a separate PFD for the initial conditions as represented in the transform.
