Section 2.12 Laplace transform methods

Â¶

There is a shorthand method called the Laplace transform for deriving the solutions of linear, constant-coefficient equations that have forcing functions such as exponentials, steps, and impulses. Given (almost) any function \(f(t)\text{,}\) there is a doppelganger function \(F(s)\) defined by an integral,

\begin{equation} F(s) = \int_0^\infty f(t) e^{-st}\, dt\text{.}\label{so-lap-lxform}\tag{2.12.1} \end{equation}

In this business, time starts at \(t=0\text{,}\) and any reference to \(f(t)\) for negative \(t\) is defined to be zero. The Laplace transform is often written using a script L, as in

\begin{equation*} F(s)=\mathcal{L}[f(t)]\text{.} \end{equation*}

The usual convention is that a lowercase function of \(t\) transforms to its uppercase namesake as a function of \(s\text{.}\)

It should be clear from the definition that we have two important properties,

\begin{gather*} \mathcal{L}[ f(t) + g(t) ] = F(s) + G(s) \\ \mathcal{L}[ C f(t) ] = C F(s)\text{,} \end{gather*}

where \(C\) is any constant. In short, we can say that the Laplace transform is a linear operation on functions, just like differentiation.

Subsection 2.12.1 Derivatives

Applying integration by parts to the definition reveals the main reason Laplace transforms are useful in ODEs.

\begin{align*} \mathcal{L}[y'(t)] \amp = \int_0^\infty y'(t) e^{-st}\, dt \\ \amp = \left[ y(t)e^{-st}\right]_0^\infty - \int_0^\infty (-s) y(t) e^{-st}\, dt \\ \amp = -y(0) + s Y(s)\text{.} \end{align*}

Calculus (differentiation) is thereby turned into algebra (multiplication). So here is the plan of attack. Given an ODE \(y'=ay+q(t)\text{,}\) we transform everything in sight to get

\begin{equation*} sY(s)-y(0) = a Y(s) + F(s)\text{.} \end{equation*}

Now solve for the transform of the solution:

\begin{equation*} Y(s) = \frac{ F(s) + y(0)}{s-a}\text{.} \end{equation*}

Now "all" we have to do know is to compute the inverse transform, that is, find \(y(t)\) given \(Y(s)\text{.}\) This is less straightforward than computing the forward transform, unfortunately. But we can make it managable in much the same way you learned indefinite integration, by learning some canonical cases and then matching patterns.

Subsection 2.12.2 Exponentials

We can find the transform of an exponential function from the transform definition. For any complex number \(c\text{,}\)

\begin{equation*} {\cal L}[e^{c t}] = \int_0^\infty e^{(c-s)t}\, dt = \left[ -\frac{1}{s-c} e^{(c-s)t} \right]_0^\infty = \frac{1}{s-c}\text{.} \end{equation*}

(To make this valid requires that \({\text Re}(c)\lt s\text{,}\) but we are going to ignore such technicalities.) An important special case is when \(c=0\text{:}\) \({\cal L}[1]=1/s\text{.}\)

Let's consider \(y'-ay=e^{ct}\) in this light. Transforming both sides, we get

\begin{equation*} [sY(s)-y(0)] - aY(s) =\frac{1}{s-c}\text{,} \end{equation*}

and thus

\begin{equation*} Y(s) = \frac{y(0)}{s-a} + \frac{1}{(s-c)(s-a)}\text{.} \end{equation*}

We can immediately recognize the first term as the transform of \(e^{at}y(0)\text{.}\) To invert the transform of the second term, we need to express it using partial fractions. There is a general procedure for this, but it's worth knowing a simple mnemonic for this particular form:

\begin{equation*} \frac{1}{(s-c)(s-a)} = \frac{1}{(a-c)(s-a)} + \frac{1}{(s-c)(c-a)}\text{.} \end{equation*}

Thus the inverse transform of this fraction is a sum of two exponentials,

\begin{equation*} y_v(t) = \frac{1}{c-a}\left( e^{ct}-e^{at} \right) \text{.} \end{equation*}

This is the "very particular" solution that has \(y_v(0)=0\text{.}\) The general solution is \(e^{at}y(0) + y_v(t)\text{.}\) As we noted before, the resonant case follows from the limit \(c\to a\text{,}\) so that \(y_v=te^{at}\text{.}\)

From here we could include sinusoidal forcing functions via the parts of \(e^{i\omega t}\text{,}\) but we'll put that off for the next chapter.

Subsection 2.12.3 Impulses, steps, shifts

Since integration against an impulse is just point evaluation, finding the transform of an impulse is easy.

\begin{align*} {\cal L}[\delta(t)] \amp = 1 \\ {\cal L}[\delta(t-T)] \amp = e^{-sT}\text{.} \end{align*}

Finding the transform of a step is also pretty easy:

\begin{align*} {\cal L}[H(t-T)] \amp = \int_T^\infty e^{-st}\, dt \\ \amp = -\frac{1}{s} \Bigl[ e^{-st} \Bigr]_T^\infty = \frac{e^{-sT}}{s} \text{.} \end{align*}

A slight generalization of the calculation results in an important rule:

\begin{align*} {\cal L}[H(t-T)f(t-T)] \amp = \int_T^\infty f(t-T) e^{-st}\, dt \\ \amp = \int_0^\infty f(u) e^{-s(u+T)}\, du \\ \amp = e^{-sT} \int_0^\infty f(u) e^{-su}\, du = e^{-sT} {\cal L}[f(t)]\text{.} \end{align*}

This is sometimes called the shift theorem.

Example 2.12.1.

Let's apply the new rules to solve \(y'=ay + \delta(t-T)\text{,}\) with \(y(0)=0\text{.}\) Transform to get

\begin{equation*} [sY(s)-y(0)] - aY(s) = e^{-sT}\text{,} \end{equation*}

and thus

\begin{equation*} Y(s) = \frac{e^{-sT}}{s-a}\text{.} \end{equation*}

Notice that \(Y(s) = e^{-sT} F(s)\text{,}\) where \(F(s)=1/(s-a)\) is the transform of \(f(t)=e^{at}\text{.}\) By the shift theorem, it must be that

\begin{equation*} y(t) = H(t-T)f(t-T) = H(t-T)e^{a(t-T)}\text{.} \end{equation*}

Subsection 2.12.4 Transfer function

Perhaps it's clear by now that \(y'=ay+q(t)\) always leads to \(Y(s) = Q(s)/(s-a)\) as a particular solution. The term \(1/(s-a)\) is called the transfer function of the ODE. Values of \(s\) that are roots of the denominator are known as poles of the transform. They point out exponential solutions.

Recall that the impulse response is the case when \(q(t)=\delta(t)\text{,}\) so that \(Q(s)=1\text{.}\) This implies that the transfer function is the transform of the impulse response. We'll be encountering these ideas again in the next chapter.
