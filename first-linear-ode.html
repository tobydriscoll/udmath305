Section 1.3 Linear ODEs

Â¶

Refer to section 1.4 of the text (pp 17-18).

Linear ODEs are the most fundamental type. If you could only get one yes-or-no answer about a new ODE, your question should be, "Is it linear?" In the first-order case, all linear problems have the form

\begin{equation} \dd{y}{t} = a(t)y(t) + q(t) \text{.}\label{eq-first-linear}\tag{1.3.1} \end{equation}

Note that it is not necessary for \(a(t)\) and \(q(t)\) to be linear functions of time; nor is the solution \(y(t)\) generally a linear function. The linearity being referred to is the fact that there are no terms such as \(\sin(y')\text{,}\) \(e^y\text{,}\) \(yy'\text{,}\) etc.

All solutions to first-order linear problems share a basic structure. In fact there is a perfect (and rather deep) analogy to how one can specify a line using a direction parallel to the line, and one point on the line. For the line direction we could use a vector \(\vec{v}\text{.}\) Note that the set of all vectors \(\{C\vec{v}: C\in\mathbb{R}\}\) is a line through the origin. We can then translate this line by adding any constant vector \(\vec{p}\) representing a point on the line. Thus the set of all vectors representing points on the line is \(C\vec{v}+\vec{p}\text{.}\)

In the linear ODE case the role of \(\vec{v}\) is played by a solution of the homogeneous equation, which is (1.3.1) with \(q(t)\equiv 0\text{.}\) With such a solution \(y_h(t)\text{,}\) we define the homogeneous or null solution \(y_n(t)=Cy_h(t)\text{,}\) where it is understood that \(C\) is an arbitrary constant. Note that (due to the linearity and homogeneity) the null solution is always a solution of the homogeneous ODE for any choice of the constant..

Playing the role of a point on the line is a particular solution \(y_p(t)\text{.}\) As the name implies, it is one, any one, solution of the original (1.3.1). For definiteness the text uses \(y_p(0)=0\) to specify the particular solution uniquely (the "very particular" solution).

With these two special solutions in hand, we can write the general solution of (1.3.1) as \(y=y_n+y_p\text{.}\) Every solution of the ODE can be written in that form. This structure pertains only to linear problems. We don't always approach practically solving the problem this way, but we can always recognize the structure after the fact. It can be useful to know that the structure exists even when exact formulas can't be found for the null and particular solutions.

Before we go on to derive some exact solutions for important special cases, let's consider how to solve (1.3.1) with \(a(t)=\cos(2t)\) and \(q(t)=t\) numerically (approximately but to high accuracy) using MATLAB. For an ODE in the general form \(y'=f(t,y)\text{,}\) we have to define a function for \(f\text{.}\)

a = @(t) cos(2*t);
q = @(t) t;

f = @(t,y) a(t)*y + q(t);

(This could be done directly in one line, but we have stayed with the general linear form here for illustration.) Now we call the function ode45 to solve the problem, designating times of interest and an initial value.

t = linspace(0,5,500);
[t,y] = ode45(f,t,1);

The outputs here are vectors of equal length, giving the times we selected and values of the solution at those times. For instance, to get values at the final time we use

format long
[ t(end), y(end) ]

and get the response

ans =
   5.000000000000000  10.064799099008786

The outputs are also in a form easy to plot.

plot(t,y)
xlabel('t'), ylabel('y(t)')
title('Solution of a linear problem')
