Section 5.4 Two-dimensional systems

¶

An autonomous system in two dimensions has the particular form

\begin{align} \dd{x_1}{t} \amp = f_1(x_1,x_2), \label{nl-eq-system1}\tag{5.4.1}\\ \dd{x_2}{t} \amp = f_2(x_1,x_2). \label{nl-eq-system2}\tag{5.4.2} \end{align}

For such a system we can again use a direction field to visualize the global behavior. At a point in the \((x_1,x_2)\) plane, the vector \([f_1(x_1,x_2),f_2(x_1,x_2)]\) is tangent to the solution curve passing through that point.

Example 5.4.1. Direction field for a system of two equations.

Here is a direction field for the linear ODE system

\begin{align*} \dd{x}{t} \amp = -y\\ \dd{y}{t} \amp = x\text{.} \end{align*}

Here is a slope field in the first quadrant for the nonlinear system

\begin{align*} \dd{x}{t} \amp = 3x - \frac{1}{2}xy\\ \dd{y}{t} \amp = \frac{1}{4}xy - y\text{.} \end{align*}

Subsection 5.4.1 Linearization

¶

An equilibrium solution of the system (5.4.1)-(5.4.2) is a pair of values \((x_1^*,x_2^*)\) such that \(f_1(x_1^*,x_2^*)=f_2(x_1^*,x_2^*)=0\text{.}\) We can understand the dynamics near an equilibrium point by approximating the nonlinear system with a linear one. Specifically, we use the multivariate linearizations

\begin{align*} f_1(x_1,x_2) \amp \approx f_1(x_1^*,x_2^*) + \pp{f_1}{x_1} (x_1-x_1^*) + \pp{f_1}{x_2} (x_2-x_2^*)\\ f_2(x_1,x_2) \amp \approx f_2(x_1^*,x_2^*) + \pp{f_2}{x_1} (x_1-x_1^*) + \pp{f_2}{x_2} (x_2-x_2^*)\text{,} \end{align*}

where it is understood for brevity that the partial derivatives are all evaluated at \((x_1^*,x_2^*)\text{.}\) The first term in each expansion is zero due to the equilibrium assumption. Now define \(u_1(t)=x_1(t)-x_1^*\text{,}\) \(u_2(t)=x_2(t)-x_2^*\) to arrive at

\begin{align*} \dd{u_1}{t} \amp \approx \pp{f_1}{x_1} u_1 + \pp{f_1}{x_2} u_2, \\ \dd{u_2}{t} \amp \approx \pp{f_2}{x_1} u_1 + \pp{f_2}{x_2} u_2. \end{align*}

This motivates the following definition.

Definition 5.4.4. Jacobian matrix.

The Jacobian matrix of the system (5.4.1)-(5.4.2) is

\begin{equation} \bJ(x_1,x_2) = \twomat{\pp{f_1}{x_1}}{\pp{f_1}{x_2}}{\pp{f_2}{x_1}}{\pp{f_2}{x_2}}\text{.}\label{nl-eq-jacobian}\tag{5.4.3} \end{equation}

Just as a derivative of \(f(x)\) is a function of \(x\text{,}\) a Jacobian matrix is a function of the variables of the system.

Let's summarize. In the neighborhood of a fixed point \((x_1^*,x_2^*)\text{,}\) we can define the “deviation from equilibrium” as the variables \(u_i(t)=x_i(t)-x_i^*\) for \(i=1,2\text{.}\) These variables approximately satisfy \(\bu'=\bJ(x_1^*,x_2^*)\bu\text{,}\) which is a linear, constant-coefficient system in two dimensions. This is a huge revelation: near a steady state, dynamics are mostly linear.

We are now right back into the situation of Section 3.5. In particular:

Fact 5.4.5.

The stability of a steady state is (usually) determined by the eigenvalues of the Jacobian matrix at the steady state.

Example 5.4.6. Linearization of the pendulum.

Let's examine the steady states of a pendulum with damping,

\begin{equation*} \ddd{\theta}{t} + \gamma \theta' + \frac{g}{L} \sin(\theta) = 0\text{,} \end{equation*}

with \(\gamma \gt 0\text{.}\) It transforms into the first-order system

\begin{align*} \dd{x_1}{t} \amp = x_2 \\ \dd{x_2}{t} \amp = -\frac{g}{L}\sin(x_1) - \gamma x_2\text{.} \end{align*}

These define \(f_1\) and \(f_2\text{.}\) From here we note all the first partial derivatives:

\begin{align*} \pp{f_1}{x_1} = 0 \amp \qquad \pp{f_1}{x_2} = 1\\ \pp{f_2}{x_1} = - \frac{g}{L}\cos(x_1) \amp \qquad \pp{f_2}{x_2} = -\gamma\text{.} \end{align*}

The first steady state we consider is at the origin. Here the Jacobian matrix is

\begin{equation*} \begin{bmatrix} 0 \amp 1 \\ -g/L \amp -\gamma \end{bmatrix} \text{.} \end{equation*}

The eigenvalues are

\begin{equation*} \frac{-\gamma \pm \sqrt{\gamma^2 - 4(g/L)}}{2}. \end{equation*}

If the radical is imaginary, then the real part is \(-\gamma/2 \lt 0\text{.}\) If it is real, then it is in the interval \((0,\gamma)\text{,}\) so the eigenvalues are real and negative. Hence the origin is stable, as either a spiral point or a node. That comports with our expectations of a pendulum hanging straight down!

The other physically distinct equilibrium is at \((\pi,0)\text{,}\) i.e., the inverted position at rest. Now the Jacobian is

\begin{equation*} \begin{bmatrix} 0 \amp 1 \\ g/L \amp -\gamma \end{bmatrix}\text{,} \end{equation*}

with eigenvalues

\begin{equation*} \frac{-\gamma \pm \sqrt{\gamma^2 + 4(g/L)}}{2}. \end{equation*}

Now the radical is always real and greater than \(\gamma\text{,}\) so the eigenvalues are real and of opposite signs. Hence the inverted position is an unstable saddle point.

Example 5.4.7. van del Pol equation.

The van der Pol equation is a nonlinear oscillator in the form \(y''+\mu(y^2-1)y'+y=0\) for parameter \(\mu \gt 0\text{.}\) It is equivalent to the system

\begin{align*} x_1' \amp = x_2 \\ x_2' \amp = -x_1 - \mu(x_1^2-1)x_2, \end{align*}

which has the sole equilibrium point \((0,0)\text{.}\) The Jacobian matrix is

\begin{equation*} \twomat{0}{1}{-1-2\mu x_1 x_2}{-\mu(x_1^2-1)}, \end{equation*}

which at the origin is simply \(\twomat{0}{1}{-1}{\mu}\text{.}\) This has eigenvalues

\begin{equation*} \frac{1}{2} \left( {\mu} \pm \sqrt{\mu^2-4} \right). \end{equation*}

If \(0\lt \mu \lt 2\text{,}\) the origin is an unstable spiral point. If \(\mu > 2\text{,}\) the origin is an unstable node, as you can never achieve a negative eigenvalue.

The caveat on using eigenvalues for stability is when they both have zero real part, which is a neutrally stable center in the linear sense. In such cases the details of the nonlinear terms of the system can swing the stability either way.

Example 5.4.8. Linearization of predator-prey.

The system

\begin{align*} \frac{dx}{dt} \amp= 3x-\frac{xy}{2}\\ \frac{dy}{dt} \amp= -y+\frac{xy}{4} \end{align*}

is called a predator–prey equation. If species \(y\) is set to zero, species \(x\) would grow exponentially on its own (the prey). Similarly, species \(y\) would die off on its own (predator). We assume that encounters between the species are jointly proportional to the population of each, and they subtract from the prey and add to the predators.

We find fixed points when \(3x-(x y/2)=(x y/4)-y=0\text{,}\) which has two solutions, \((0,0)\) and \((4,6)\text{.}\) The Jacobian matrix of the system is

\begin{equation*} \mathbf{A} = \begin{bmatrix} 3-y/2 \amp -x/2 \\ y/4 \amp -1+x/4 \end{bmatrix}\text{.} \end{equation*}

At the origin, the Jacobian becomes

\begin{equation*} \begin{bmatrix} 3 \amp 0 \\ 0 \amp -1 \end{bmatrix}\text{.} \end{equation*}

The eigenvalues of a diagonal matrix are the diagonal entries. Hence the origin is a saddle point.

At the other steady state we have the Jacobian

\begin{equation*} \twomat{0}{-2}{3/2}{0}\text{.} \end{equation*}

The characteristic polynomial is \(\lambda^2 - 0\lambda + 3\text{,}\) so the eigenvalues are \(\pm i\sqrt{3}\text{.}\) Hence this fixed point is a center. Close to the point, the orbits are impossible to tell apart from those of a linear center:
