Section 2.4 Variation of parameters

¶

In this and the next section we turn to looking for particular solutions of the first-order linear ODE

\begin{equation} x'-a(t)x=f(t)\text{,}\label{fl-eq-folinrepeat}\tag{2.4.1} \end{equation}

or \(\opA[x]=f\text{.}\) We start in this section with an all-purpose method. In the next section we look at a method that's simpler for the most common problems.

We begin with the homogeneous solution

\begin{equation*} x_h(t) = c_1 g(t) = c_1 \exp\left[\int a(t)\, dt\right]\text{,} \end{equation*}

where \(g(t)\) is a shorthand for the exponential. The idea of variation of parameters is to posit (i.e., guess) that the particular solution can be written by replacing the constant with a function,

\begin{equation} x_p(t) = k(t) g(t)\text{,}\label{fl-eq-vofp1}\tag{2.4.2} \end{equation}

and then choose \(k(t)\) to make it so:

\begin{equation*} f(t) = x_p' - a(t) x_p = [k'g+kg'] - a k g = k'g + k[g'-ag]\text{,} \end{equation*}

which, because \(g(t)\) is a homogeneous solution, is simply \(f(t)=k'(t)g(t)\text{.}\) Hence we get a proper particular solution of the original problem provided \(f=k'g\text{,}\) or

\begin{equation} k(t) = \int \frac{f(t)}{g(t)}\, dt\text{.}\label{fl-eq-vofp2}\tag{2.4.3} \end{equation}

Step by step, we have to (1) integrate \(a(t)\) once to get \(g\text{,}\) (2) integrate \(f/g\) to get \(k\) in (2.4.3), and (3) multiply them together to get the particular solution in (2.4.2). It's also worth reiterating the implicit step (0), which is to first of all put the given problem in the standard form of (2.4.1); i.e., divide the equation through by any coefficient of \(x'\text{.}\)

The function \(1/g\) is often called an integrating factor. If you multiply the original equation by \(1/g\text{,}\) then the solution can be found by one more integration. It's the same result as we derived here.

Example 2.4.1.

Consider \(3x'=12x+24t\text{.}\) Rewriting it as \(x'-4x=8t\text{,}\) we identify \(a(t)=4\) and \(f(t)=8t\text{.}\) Then

\begin{equation*} g(t) = \exp\left[ \int 4\,dt \right] = e^{4t}\text{,} \end{equation*}

and

\begin{equation*} k(t) = \int \frac{8t}{e^{4t}}\, dt = -\frac{1}{2} (4t+1)e^{-4t}\text{,} \end{equation*}

where you need integration by parts (or a computer) to perform the integral. Hence

\begin{equation*} x_p(t) = \left[ \frac{1}{2} (4t+1)e^{-4t} \right] e^{4t} = -\frac{1}{2} (4t+1)\text{,} \end{equation*}

and the general solution is

\begin{equation*} x(t) = x_h(t) + x_p(t) = c_1 e^{4t} - \frac{1}{2} (4t+1)\text{.} \end{equation*}
Example 2.4.2.

To find a particular solution of \(x'=2tx + 6t\text{,}\) we start with

\begin{equation*} g(t) = \exp\left[ \int 2t\, dt \right] = e^{t^2} \text{.} \end{equation*}

Then we can compute

\begin{equation*} k(t) = \int \frac{6t}{e^{t^2}}\, dt = -3 e^{-t^2}\text{.} \end{equation*}

Finally,

\begin{equation*} x_p(t) = k(t) g(t) = -3\text{,} \end{equation*}

and the general solution is \(x(t)=c_1 e^{t^2}-3\text{.}\)

(It should be clear that we needed a very special \(f\) to make the second integral feasible. If not, try reworking it for \(f(t)=6\) and you will quickly find yourself stuck, at least while using familiar functions.)

Example 2.4.3.

For \((2+t) x'= x - 1\text{,}\) we first rewrite as

\begin{equation*} x' - \frac{1}{2+t} x = -\frac{1}{2+t}\text{.} \end{equation*}

Then

\begin{equation*} g(t) = \exp\left[ \int \frac{1}{2+t}\, dt \right] = \exp[ \ln(2+t) ] = 2+t\text{.} \end{equation*}

Next,

\begin{equation*} k(t) = \int \frac{-1}{2+t} (2+t)^{-1} \, dt = (2+t)^{-1}\text{,} \end{equation*}

so that

\begin{equation*} x(t) = c_1 (2+t) + (2+t)^{-1} (2+t) = c_1(2+t)+1\text{.} \end{equation*}

Combining the expressions for both the homogeneous and particular parts, we obtained the full general solution

\begin{equation} x(t) = c_1 \exp[q(t)] + \exp[q(t)] \int f(t) \exp[-q(t)]\, dt = \exp[q(t)] \left[ c_1 + \int f(t) \exp[-q(t)]\, dt \right]\text{,}\label{fl-eq-vofpgen}\tag{2.4.4} \end{equation}

where as before, \(q\) is any antiderivative of \(a(t)\text{.}\) Writing the solution this way clarifies that once again the “plus C” of indefinite integration doesn't really matter, as it gets absorbed into the final arbitrary \(c_1\text{.}\)

For initial-value problems, we have two options. The first is the default: find the entire general solution, and then apply the initial condition to get a specific value for the free constant.

Example 2.4.4.

Above we found that the general solution of \(x'=4x+8t\) is \(x(t)=c_1 e^{4t} - \frac{1}{2} (4t+1)\text{.}\) If we also have that \(x(0)=3/2\text{,}\) then we can conclude

\begin{equation*} \frac{3}{2} = x(0) = c_1 - \frac{1}{2}(1)\text{,} \end{equation*}

so \(c_1=2\text{.}\)

A slightly streamlined version of the process is available for IVPs. Given the initial condition \(x(t_0)=x_0\text{,}\) we rewrite the general solution (2.4.4) using definite integrals:

\begin{equation*} q_0(t) = \int_{t_0}^t a(s)\, ds, \end{equation*}

followed by

\begin{equation*} x(t) = \exp[q_0(t)] \left[ c_1 + \int_{t_0}^t f(s) \exp[-q_0(s)]\, ds \right]\text{.} \end{equation*}

By the properties of definite integrals, we know that \(q(t_0)=0\text{,}\) and hence

\begin{equation*} x_0 = x(t_0) = 1\cdot \left[ c_1 + 0 \right]\text{.} \end{equation*}

So there is really no need to solve for the constant; the initial value can be baked in from the beginning:

\begin{equation*} x(t) = \exp[q_0(t)] \left[ x_0 + \int_{t_0}^t f(s) \exp[-q_0(s)]\, ds \right]\text{.} \end{equation*}
Example 2.4.5.

Reconsider the previous example of \(x'=4x+8t\) with \(x(0)=3/2\text{.}\) Then

\begin{equation*} q_0(t) = \int_0^t 4s\, ds = 4t\text{,} \end{equation*}

and

\begin{equation*} x(t) = e^{4t} \left[ \frac{3}{2} + \int_{0}^t 8s e^{-4s}\, ds \right]. \end{equation*}

This evaluates to

\begin{equation*} x(t) = e^{4t} \left[ \frac{3}{2} + \frac{1}{2} \Bigl(1-e^{-4 t} (4 t+1)\Bigr) \right] = 2e^{4t} - \frac{1}{2}(4t+1)\text{,} \end{equation*}

just as before.

When the homogeneous and particular solutions are known or given, it's easier to write out the general solution and then find the constant in it. But starting from scratch, the definite integral approach is tidier. The overall difficulty is about the same.
