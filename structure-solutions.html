Section 1.3 Structure of linear ODE solutions

¶

Linear ODEs are the most fundamental type. If you could only get one yes-or-no answer about a new ODE, your question should be, "Is it linear?" In the first-order case, all linear problems have the form

\begin{equation*} \dd{y}{t} = a(t)y(t) + q(t) \text{.} \end{equation*}

Note that it is not necessary for \(a(t)\) and \(q(t)\) to be linear functions of time; nor is the solution \(y(t)\) generally a linear function. The linearity being referred to is the fact that there are no terms such as \(\sin(y')\text{,}\) \(e^y\text{,}\) \(yy'\text{,}\) etc. For this section and the next, we will limit ourselves to the important case when \(a\text{,}\) the growth/decay rate, is constant:

\begin{equation} \dd{y}{t} = ay(t) + q(t) \text{.}\label{eq-linear-first-const}\tag{1.3.1} \end{equation}

Subsection 1.3.1 Null and particular solutions

¶

All solutions to first-order linear problems share a basic structure. In fact there is a perfect (and deep) analogy to how one can specify a line using a direction parallel to the line, and one point on the line. For the line direction we could use a vector \(\vec{v}\text{.}\) Note that the set of all vectors \(\{C\vec{v}: C\in\mathbb{R}\}\) is a line through the origin. We can then translate this line by adding any constant vector \(\vec{p}\) representing a point on the line. Thus the set of all vectors representing points on the line is \(C\vec{v}+\vec{p}\text{.}\)

In the linear ODE case the role of \(\vec{v}\) is played by a solution of the homogeneous equation, which is (1.3.1) with \(q(t)\equiv 0\text{.}\) With such a solution \(y_h(t)\text{,}\) we define the homogeneous or null solution \(y_n(t)=Cy_h(t)\text{,}\) where it is understood that \(C\) is an arbitrary constant. Note that (due to the linearity and homogeneity) the null solution is always a solution of the homogeneous ODE for any choice of the constant..

Playing the role of a point on the line is a particular solution \(y_p(t)\text{.}\) As the name implies, it is one, any one, solution of the original (1.3.1). For definiteness the text uses \(y_p(0)=0\) to specify the particular solution uniquely (the "very particular" solution).

With these two special solutions in hand, we can write the general solution of (1.3.1) as \(y=y_n+y_p\text{.}\) Every solution of the ODE can be written in that form. This structure pertains only to linear problems (including the more general case with variable \(a(t)\)). We don't always approach solving the problem this way, but we can always recognize the structure after the fact. It can be useful to know that the structure exists even when exact formulas can't be found for the null and particular solutions.

Subsection 1.3.2 Integrating factor

¶

By multiplying (1.3.1) through by the integrating factor \(M(t)=e^{-at}\text{,}\) we can render it into a form easily solved by calculus:

\begin{align*} e^{-at} y'(t) - (a e^{-at})y(t) \amp = e^{-at}q(t) \\ \dd{}{t}\bigl[ e^{-at} y(t) \bigr] \amp = e^{-at}q(t) \\ e^{-at} y(t) \amp = C + \int e^{-at}q(t)\, dt \\ \amp = C + \int_0^t e^{-as}q(s)\, ds\text{.} \end{align*}

In the last step above we changed from an indefinite integral to a definite one whose value is zero when \(t=0\text{.}\) This makes it easy to evaluate the integration constant, leading to

\begin{equation} y(t) = e^{at}y(0) + \int_0^t e^{a(t-s)}q(s)\, ds\text{.}\label{eq-flc-solution}\tag{1.3.2} \end{equation}

We can dissect this expression in some detail. The null solution \(e^{at}y(0)\) is just what we found for \(y'=ay\) in the previous section. The particular solution represented by the integral is new. Think of the integral as a sum over all the moments \(s\lt t\text{.}\) At that moment there is an input to \(y'\) of size \(q(s)\text{.}\) That little input evolves all on its own for the remaining time \(t-s\) according to \(e^{a(t-s)}q(s)\text{.}\) So it's equivalent to having started a problem with initial value \(q(s)\) at that moment. That is, the complete solution combines the natural evolution of the true initial condition at time zero, plus the contributions of momentary "kicks" evolving from each instant up to the present. This idea proves to be quite fundamental to all linear problems.
