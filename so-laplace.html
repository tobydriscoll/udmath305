Section 2.8 Laplace transform methods

¶

This and the next section are from 2.7 in the text, but much of that is stuff we did in the previous chapter.

The correspondence between \(L[y]=Ay''+By'+Cy\) and \(P(s)=As^2+Bs+C\) is by now familiar. It came by way of assuming \(y(t)=Y(s)e^{st}\text{,}\) from which we get \(Y(s)=1/P(s)\text{.}\) Factoring \(P\) tells us what we need to know. For example, if its roots are \(-2\) and \(-4\text{,}\) then we know that \(y(t)\) is a combination of \(e^{-2t}\) and \(e^{-4t}\text{.}\) The only mystery to work out is the coefficients.

We can use the method of Laplace transforms to systematize the manipulations. As in first-order equations, these are useful mostly for linear, constant-coefficient problems.

Subsection 2.8.1 Second derivatives

Recall that the transform of \(y'(t)\) is \(sY(s)-y(0)\text{.}\) Applying this formula a second time shows that

\begin{equation*} \mathcal{L}[y''(t)] = s \mathcal{L}[y'(t)] - y'(0) = s^2Y(s) - sy(0) - y'(0)\text{.} \end{equation*}

Let's break down what happens in a fundamental case, \(y''-y=e^{rt}\text{.}\) Transforming both sides gives

\begin{equation*} [s^2Y(s) - sy(0)-y'(0)] - Y(s) = 1/(s-r)\text{.} \end{equation*}

For reasons to be clarified later, let's skip to the "very particular" part with zero initial conditions, for which

\begin{equation*} Y(s) = \frac{1}{(s^2-1)(s-r)} = \frac{1}{(s+1)(s-1)(s-r)}\text{.} \end{equation*}

This time there are three poles. Our mnemonic still works here,

\begin{equation*} \frac{1}{(s+1)(s-1)(s-r)} = \frac{1}{(s+1)(-1-1)(-1-r)} + \frac{1}{(1+1)(s-1)(1-r)} + \frac{1}{(r+1)(r-1)(s-r)} \text{.} \end{equation*}

Note that this requires \(r\neq -1\) and \(r \neq 1\) (i.e., no resonance). From here we get three exponentials:

\begin{equation*} y_v(t) = \frac{1}{2(1+r)}e^{-t} + \frac{1}{2(1-r)}e^{t} + \frac{1}{r^2-1}e^{rt}\text{.} \end{equation*}

Now we can write out a general solution. The characteristic roots are \(\pm 1\text{,}\) so

\begin{equation*} y(t) = c_1 e^{-t} + c_2 e^{t} + y_v(t)\text{.} \end{equation*}

If desired, it's easy to solve for the constants in this, because \(y_v\) contributes nothing:

\begin{align*} y(0) \amp = c_1 + c_2 \\ y'(0) \amp = -c_1 + c_2 \text{.} \end{align*}

That's a lot less work than doing a separate PFD needed when the initial conditions are kept in the transform. That's why it often pays to just ignore those parts when using the transform method, putting them back in at the very end.

Subsection 2.8.2 Imaginary poles

If we transform \(e^{i\omega t}\text{,}\) we can take the real and imaginary parts to get some new formulas:

\begin{equation*} {\cal L}[\cos(\omega t)] = \text{Re}\left( \frac{1}{s-i\omega} \frac{s+i\omega}{s+i\omega} \right) = \frac{s}{s^2+\omega^2}\text{,} \end{equation*}

and

\begin{equation*} {\cal L}[\sin(\omega t)] = \text{Im}\left( \frac{1}{s-i\omega} \frac{s+i\omega}{s+i\omega} \right) = \frac{\omega}{s^2+\omega^2}\text{.} \end{equation*}

Conversely, a transform \(Y(s)\) that has \(s^2+\omega^2\) as a factor in the denominator has poles at \(\pm i\omega\text{.}\) The simplest case is

\begin{equation*} Y(s) = \frac{A+Bs}{s^2+\omega^2} \quad \Rightarrow \quad y(t)=(A/\omega)\sin(\omega t) + B\cos(\omega t)\text{.} \end{equation*}
Example 2.8.1

Let's solve \(y''+9y=\delta(t-2)\text{.}\) A particular solution has transform

\begin{equation*} Y(s) = \frac{e^{-2s}}{s^2+9} = e^{-2s} F(s)\text{,} \end{equation*}

where \(F(s)\) is the transform of \(\sin(3t)/3\text{.}\) So the shift theorem implies

\begin{equation*} y(t) = \frac{1}{3} H(t-2) \sin(3t-6)\text{.} \end{equation*}

Things are not too bad if there is an additional real pole:

\begin{equation*} Y(s) = \frac{A+Bs}{s^2+\omega^2} + \frac{C}{s-c} \quad y(t)=(A/\omega)\sin(\omega t) + B\cos(\omega t) + C e^{ct}\text{.} \end{equation*}

Usually there is some grunt work involved in finding the values of \(A\text{,}\) \(B\text{,}\) and \(C\text{.}\)

Example 2.8.2

The undamped oscillator \(y''+16y=40e^{-2t}\) has a solution whose transform is

\begin{equation*} Y(s) = \frac{40}{(s+2)(s^2+16)} = \frac{A+Bs}{s^2+16^2} + \frac{C}{s+2}\text{.} \end{equation*}

Clearing the denominators gives the identity

\begin{equation*} (A+Bs)(s+2) + C(s^2+16) = 40\text{.} \end{equation*}

We need three equations to determine the unknown coefficients. Setting \(s=-2\) quickly gives \(20C=40\text{,}\) so \(C=2\text{.}\) Then setting \(s=\pm 4\) gives the system

\begin{align*} (A+4B)(6) \amp = 40 - 64 = -24\\ (A-4B)(-2) \amp = -24\text{,} \end{align*}

or \(A+4B=-4\) and \(A-4B=12\text{.}\) We can add and subtract these equations to get \(2A=8\text{,}\) \(8B=-16\text{,}\) so finally

\begin{equation*} Y(s) = \frac{4-2s}{s^2+16^2} + \frac{2}{s+2} \text{.} \end{equation*}

Hence

\begin{equation*} y(t) = \sin(4t) - 2\cos(4 t) + 2e^{-2t}\text{.} \end{equation*}

Finally, consider the driven, undamped oscillator \(y''+\omega_n^2 y= \cos(\omega t)\text{.}\) It leads to

\begin{equation*} Y(s) = \frac{s}{(s^2+\omega_n^2)(s^2+1)} = \frac{1}{\omega_n^2-\omega^2} \left( \frac{s}{s^2+\omega^2} - \frac{s}{s^2+\omega_n^2} \right)\text{,} \end{equation*}

which is readily inverted into a combination of \(\cos(\omega t)\) and \(\cos(\omega_n t)\text{.}\) However, at this point we are probably better off using the method of undetermined coefficients instead.
