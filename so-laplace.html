Section 4.6 Laplace transform methods

¶

We return to our linear, second-order, constant-coefficient problem in the simpler notation

\begin{equation*} x'' + bx' + cx = f(t)\text{.} \end{equation*}

We cast this problem as a first-order system in two dimensions, found that the eigenvalues are the roots of \(\lambda^2+b\lambda + c\text{,}\) and then constructed exponential solutions for the homogeneous part (except for the double eigenvalue case). We also found a particular solution for the case of an exponential \(f(t)\text{.}\) There is an alternative way to derive the same results using the method of Laplace transforms to systematize the manipulations.

Subsection 4.6.1 Second derivatives

Recall that the transform of \(x'(t)\) is \(sX(s)-x(0)\text{.}\) Applying this formula twice shows that

\begin{equation*} \lx[x''(t)] = s \lx[x'(t)] - x'(0) = s^2X(s) - sx(0) - x'(0)\text{.} \end{equation*}

Let's look at an example first.

Example 4.6.1.

Let's observe what happens in a fundamental case, \(x''-x=e^{rt}\text{.}\) Transforming both sides gives

\begin{equation*} [s^2X(s) - sx(0)-x'(0)] - X(s) = \frac{1}{s-r}\text{.} \end{equation*}

We easily solve for \(X\text{:}\)

\begin{equation*} X(s) = x'(0)\frac{1}{(s^2-1)} + x(0)\frac{s}{(s^2-1)} + \frac{1}{(s^2-1)(s-r)} \text{.} \end{equation*}

This is doable, but we are going to lighten the load a bit. Even before we took the transform, we knew how to find the general homogeneous solution, \(c_1e^t+c_2e^{-t}\text{.}\) So let's use the transform to find any particular solution of the nonhomogeneous problem. Based on the expression above, we're best off with finding the particular solution with \(x(0)=x'(0)=0\text{,}\) leaving us with

\begin{equation*} X_p(s) = \frac{1}{(s+1)(s-1)(s-r)} \text{.} \end{equation*}

The partial fraction mneomic gives us

\begin{equation*} \frac{1}{(s+1)(s-1)(s-r)} = \frac{1}{(s+1)(-1-1)(-1-r)} + \frac{1}{(1+1)(s-1)(1-r)} + \frac{1}{(r+1)(r-1)(s-r)} \text{.} \end{equation*}

Note that this requires \(r\neq -1\) and \(r \neq 1\) (i.e., no resonance). From here we get

\begin{equation*} x_p(t) = \frac{1}{2(1+r)}e^{-t} + \frac{1}{2(1-r)}e^{t} + \frac{1}{r^2-1}e^{rt}\text{.} \end{equation*}

With this we can construct a general solution \(x_n+x_p\text{,}\) and then apply initial conditions to solve for the constants, if those are given.

This example is the template for our use of Laplace transforms. In the general case we get

\begin{equation*} [s^2X(s) - sx(0)-x'(0)] + b[sX(s)-x(0)] + cX(s) = F(s)\text{.} \end{equation*}

We choose to find a particular solution with zero initial conditions, so that

\begin{equation*} X_p(s) = \frac{F(s)}{s^2+bs+c}\text{.} \end{equation*}

The real effort comes in finding the inverse transform of the right-hand side. Note that the denominator has roots at the eigenvalues of the homogeneous problem. These are called poles of the transform \(X_p\text{.}\) Typically \(F(s)\) will contribute one or more of its own poles. Each pole at \(s=r\) corresponds to an exponential solution \(e^{rt}\) in time, but the exact expression depends on a partial fraction decomposition and perhaps other manipulations.

Subsection 4.6.2 Imaginary poles

If we transform \(e^{i\omega t}\text{,}\) we can take the real and imaginary parts to get some new formulas:

\begin{equation*} \lx[\cos(\omega t)] = \text{Re}\left( \frac{1}{s-i\omega} \frac{s+i\omega}{s+i\omega} \right) = \frac{s}{s^2+\omega^2}\text{,} \end{equation*}

and

\begin{equation*} \lx[\sin(\omega t)] = \text{Im}\left( \frac{1}{s-i\omega} \frac{s+i\omega}{s+i\omega} \right) = \frac{\omega}{s^2+\omega^2}\text{.} \end{equation*}

Conversely, a transform \(X(s)\) that has \(s^2+\omega^2\) as a factor in the denominator has poles at \(\pm i\omega\text{.}\) The simplest case is

\begin{equation*} X_p(s) = \frac{A+Bs}{s^2+\omega^2} \quad \Rightarrow \quad x_p(t)=(A/\omega)\sin(\omega t) + B\cos(\omega t)\text{.} \end{equation*}
Example 4.6.2.

Let's solve \(x''+9x=\delta(t-2)\text{.}\) A particular solution has transform

\begin{equation*} X_p(s) = \frac{e^{-2s}}{s^2+9} = e^{-2s} F(s)\text{,} \end{equation*}

where \(F(s)\) is the transform of \(\sin(3t)/3\text{.}\) So the shift theorem implies

\begin{equation*} x_p(t) = \frac{1}{3} H(t-2) \sin(3t-6)\text{.} \end{equation*}

Things are not too bad if there is an additional real pole:

\begin{equation*} X_p(s) = \frac{A+Bs}{s^2+\omega^2} + \frac{C}{s-c} \quad x_p(t)=(A/\omega)\sin(\omega t) + B\cos(\omega t) + C e^{ct}\text{.} \end{equation*}

Usually there is some grunt (or computer) work involved in finding the values of \(A\text{,}\) \(B\text{,}\) and \(C\text{.}\)

Example 4.6.3.

The undamped oscillator \(x''+16x=40e^{-2t}\) has a particular solution whose transform is

\begin{equation*} X_p(s) = \frac{40}{(s+2)(s^2+16)} = \frac{A+Bs}{s^2+16^2} + \frac{C}{s+2}\text{.} \end{equation*}

Clearing the denominators gives the identity

\begin{equation*} (A+Bs)(s+2) + C(s^2+16) = 40\text{.} \end{equation*}

We need three equations to determine the unknown coefficients. Setting \(s=-2\) quickly gives \(20C=40\text{,}\) so \(C=2\text{.}\) Then setting \(s=\pm 4\) gives the system

\begin{align*} (A+4B)(6) \amp = 40 - 64 = -24\\ (A-4B)(-2) \amp = -24\text{,} \end{align*}

or \(A+4B=-4\) and \(A-4B=12\text{.}\) We can add and subtract these equations to get \(2A=8\text{,}\) \(8B=-16\text{,}\) so finally

\begin{equation*} X_p(s) = \frac{4-2s}{s^2+16^2} + \frac{2}{s+2} \text{.} \end{equation*}

Hence

\begin{equation*} x_p(t) = \sin(4t) - 2\cos(4 t) + 2e^{-2t}\text{.} \end{equation*}

Finally, consider the driven, undamped oscillator \(x''+\omega_0^2 x = \cos(\omega t)\text{.}\) It leads to

\begin{equation*} X_p(s) = \frac{s}{(s^2+\omega_0^2)(s^2+1)} = \frac{1}{\omega_0^2-\omega^2} \left( \frac{s}{s^2+\omega^2} - \frac{s}{s^2+\omega_n^2} \right)\text{,} \end{equation*}

which is readily inverted into a combination of \(\cos(\omega t)\) and \(\cos(\omega_0 t)\text{.}\) However, at this point we are probably better off using the method of undetermined coefficients instead.
