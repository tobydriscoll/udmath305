We have some things left to learn about Laplace transforms.

Subsection 2.10.1 Shifts and steps

If we know the transform of some \(f(t)\text{,}\) then we can easily get the transform of the shifted/delayed version \(H(t-T)f(t-T)\text{:}\)

\begin{align*} {\cal L}[H(t-T)f(t-T)] \amp = \int_T^\infty f(t-T) e^{-st}\, dt \\ \amp = \int_0^\infty f(u) e^{-s(u+T)}\, du \\ \amp = e^{-sT} \int_0^\infty f(u) e^{-su}\, du = e^{-sT} {\cal L}[f(t)]\text{.} \end{align*}

An important special case is when \(f(t)\equiv 1\text{,}\) and we get the transform of a step as

\begin{equation*} {\cal L}[H(t-T)] = \frac{e^{-sT}}{s}\text{.} \end{equation*}
Example 2.10.1

Let's transform the function

\begin{equation*} f(t) = t + (1-t) H(t-2) + (e^{-2t}-1) H(t-3)\text{.} \end{equation*}

The tricky bit is that to apply the transform formula for \(H(t-T)f(t-T)\text{,}\) we need the same shift to appear in both terms. We use a classic add-and-subtract trick:

\begin{align*} f(t) \amp = t + (1-(t-2+2)) H(t-2) + (e^{-2[(t-3)+3]}-1) H(t-3)\\ \amp = t + (-1-(t-2)) H(t-2) + (e^{-6}e^{-2(t-3)}-1) H(t-3)\\ \amp = t - H(t-2) -(t-2) H(t-2) + e^{-6}e^{-2(t-3)}H(t-3) - H(t-3)\text{.} \end{align*}

Now we're ready to read off the transform.

\begin{equation*} F(s) = \frac{1}{s^2} - \frac{e^{-2s}}{s} - \frac{e^{-2s}}{s^2} + e^{-6}\frac{e^{-3s}}{s+2} - \frac{e^{-3s}}{s}\text{.} \end{equation*}

We have a new addition to the inversion toolkit: exponentials in \(s\text{.}\)

Example 2.10.2

Let's find the inverse of \(F(s) = \frac{6 e^{-5s}}{s^2+s-2}\text{.}\) We write it as \(F(s) = 6^{-5s} G(s)\text{,}\) where

\begin{equation*} G(s) = \frac{1}{(s+2)(s-1)} = \frac{1}{(1+2)(s-1)} + \frac{1}{(s+2)(-2-1)}\text{.} \end{equation*}

This inverts to give \(g(t) = e{t}/3 - e^{-2t}/3\text{.}\) Thus

\begin{equation*} f(t) = 6 H(t-5)g(t-5) = 2 H(t-5) \left[ e^{t-5} - e^{-2t+10} \right] \text{.} \end{equation*}

Subsection 2.10.2 Impulses

Since impulses cause evaluation, finding their transforms is easy.

\begin{align*} {\cal L}[\delta(t)] \amp = 1 \\ {\cal L}[\delta(t-T)] \amp = e^{-sT}\text{.} \end{align*}
Example 2.10.3

Let's solve \(y''+9y=\delta(t-2)\text{.}\) A particular solution has transform

\begin{equation*} Y(s) = \frac{e^{-2s}}{s^2+9} = e^{-2s} F(s)\text{,} \end{equation*}

where \(F(s)\) is the transform of \(\sin(3t)/3\text{.}\) So the shift formula implies

\begin{equation*} y(t) = \frac{1}{3} H(t-2) \sin(3t-6)\text{.} \end{equation*}
