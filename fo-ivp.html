Section 2.4 Initial value problems

¶

The general solution of \(x'=a(t)x+f(t)\) includes the general homogeneous solution, which contains an arbitrary multiplicative constant. This “solution” is therefore really a family of solutions. Many models include one piece of additional information that specifies the solution uniquely.

Definition 2.4.1. Initial value problem (IVP).

A first-order initial value problem is a first-order ODE together with an initial condition \(x(t_0)=x_0\text{.}\)

The definition of IVP, unlike most of what we have done so far, applies to nonlinear ODEs as well as linear ones.

Solving an IVP typically involves first finding the general solution of the ODE, then applying the initial condition to determine a specific value for the integration constant.

Example 2.4.2.

We found earlier that the general solution of \(x'=\sin(t) x\) is

\begin{equation*} x(t) = c_1 \exp[ -\cos(t) ]\text{.} \end{equation*}

If we are also given the initial condition \(x(0)=1\text{,}\) then we substitute to determine

\begin{equation*} 1 = x(0) = c_1 e^{-\cos(0)} = \frac{c_1}{e}\text{,} \end{equation*}

so that \(c_1=e\text{.}\) We then write

\begin{equation*} x(t) = e \exp[ -\cos(t) ] = \exp[1-\cos(t)]\text{,} \end{equation*}

which has no arbitrary constants left.

Subsection 2.4.1 Growth factor

¶

Another option is to rewrite some formulas for the IVP case. Consider again the integrating factor,

\begin{equation*} g(t) = \exp\left[ \int a(t)\, dt \right]\text{.} \end{equation*}

It is not uniquely defined thanks to the integration constant. For an ODE alone this is natural, as the solution itself is not unique. But for the IVP we can define

\begin{equation} G(s,t) = \exp\left[ \int_{s}^t a(u)\, du \right]\text{,}\label{fo-eq-growthfactor}\tag{2.4.1} \end{equation}

which we call the growth factor. It has some interesting and useful properties.

Theorem 2.4.3. Growth factor.
  1. \begin{equation*} G(t,t)=1 \end{equation*}
  2. \begin{equation*} \frac{d}{dt} G(t_0,t) = a(t) G(t_0,t) \end{equation*}
  3. \begin{equation*} G(s,r)G(r,t) = G(s,t) \end{equation*}
  4. \begin{equation*} \frac{1}{G(s,t)}=G(t,s) \end{equation*}
  5. The unique solution of \(x'-ax=0\text{,}\) \(x(t_0)=x_0\) is \(G(t_0,t)x_0\text{.}\)
Example 2.4.4.

Reconsider the previous IVP \(x'=\sin(t) x\text{,}\) \(x(0)=1\text{.}\) First we find

\begin{equation*} G(s,t) = \exp \left[ \int_{s}^t \sin(u)\, du \right] = \exp[-\cos(t)+\cos(s)] \text{.} \end{equation*}

The solution of the IVP is therefore

\begin{equation*} x(t) = G(0,t)\cdot 1 = \exp[1-\cos(t)]\text{,} \end{equation*}

which is what we found before.

The growth factor has a simple interpretation: multiplication by \(G(s,t)\) represents the evolution of \(x\) from a state at time \(s\) to the state at time \(t\text{.}\) That makes the first three properties of Theorem 2.4.3 seem pretty obvious.

Subsection 2.4.2 Numerical solutions

¶

Because an IVP has a unique solution, it is a suitable target for an approximate numerical solution.

Example 2.4.5.

We look again at \(x'=\sin(t) x\text{,}\) \(x(0)=1\text{.}\) Most numerical routines, like those in MATLAB, are coded to solve an ODE given in the general form \(x'=F(t,x)\text{.}\) Therefore we have to define a function for \(F\text{.}\)

F = @(t,x) sin(t)*x;

Now we call the function ode45 to solve the problem, designating times of interest and the initial value.

t = linspace(0,5,500);
[t,x] = ode45(F,t,1);

The outputs here are vectors of equal length, giving the times we selected and values of the solution at those times. For instance, to get values at the final time we use

format long
[ t(end), x(end) ]

and get the response

ans =
5.000000000000000   2.046942183624863

The outputs are also in a form easy to plot.

plot(t,x)
xlabel('t'), ylabel('x(t)')
title('Solution of a linear problem')
