Section 2.8 Laplace transform methods

¶

There is a shorthand method for deriving the solutions of linear, constant-coefficient equations that have forcing functions such as steps, impulses, and elementary functions. Given (almost) any function \(f(t)\text{,}\) there is a doppelganger function \(F(s)\) defined by an integral.

Definition 2.8.1. Laplace transform.

The Laplace transform of a function \(f(t)\) is defined as

\begin{equation} \lx[f(t)] = F(s) = \int_0^\infty f(t) e^{-st}\, dt\text{.}\label{so-lap-lxform}\tag{2.8.1} \end{equation}

In this business, time starts at \(t=0\text{,}\) and any reference to \(f(t)\) for negative \(t\) is defined to be zero. A convention is that a lowercase function of \(t\) transforms to its uppercase namesake as a function of \(s\text{.}\) (Another convention is that \(\lx\) is used with curly braces, but we have already committed to using square brackets for operators.)

Fact 2.8.2.

The Laplace transform is a linear operator. That is,

\begin{gather*} \lx[ f(t) + g(t) ] = F(s) + G(s) \\ \lx[ c f(t) ] = c F(s)\text{,} \end{gather*}

where \(c\) is any constant.

Subsection 2.8.1 Derivatives

Applying integration by parts to the definition of \(\lx[x'(t)]\) reveals the main reason Laplace transforms are useful in ODEs:

\begin{align*} \lx[x'(t)] \amp = \int_0^\infty x'(t) e^{-st}\, dt \\ \amp = \left[ x(t)e^{-st}\right]_0^\infty - \int_0^\infty (-s) x(t) e^{-st}\, dt \\ \amp = -x(0) + s X(s)\text{.} \end{align*}

Calculus is turned into algebra by the Laplace transform. So here is the plan of attack. Given an ODE \(x'-ax=f(t)\text{,}\) we transform everything in sight to get

\begin{equation*} sX(s)-x(0) - a X(s) = F(s)\text{.} \end{equation*}

Now solve for the transform of the solution:

\begin{equation*} X(s) = \frac{ F(s) + x(0)}{s-a}\text{.} \end{equation*}

Now “all” we have to do know is to compute the inverse transform, that is, find \(x(t)\) given \(X(s)\text{.}\) This is a lot less straightforward than computing the forward transform, unfortunately. But we can make it manageable in much the same way you learned indefinite integration: by learning some canonical cases and then matching patterns.

Subsection 2.8.2 Basic transforms

We can find the transform of an exponential function from the transform definition. For any complex number \(c\text{,}\)

\begin{equation*} \lx[e^{c t}] = \int_0^\infty e^{(c-s)t}\, dt = \left[ -\frac{1}{s-c} e^{(c-s)t} \right]_0^\infty = \frac{1}{s-c}\text{.} \end{equation*}

(To make this valid requires that \({\text Re}(c)\lt s\text{,}\) but we are going to ignore all such technicalities.)

Fact 2.8.3. Transform of an exponential.
\begin{equation} \lx[e^{c t}] = \frac{1}{s-c}\text{.}\label{fl-eq-ltexp}\tag{2.8.2} \end{equation}

An important special case is \(c=0\text{:}\)

\begin{equation*} \lx[1]=\frac{1}{s}\text{.} \end{equation*}
Example 2.8.4.

Let's consider \(x'-ax=e^{ct}\text{,}\) where \(a,c\) are different constants. Transforming both sides, we get

\begin{equation*} [sX(s)-x(0)] - aX(s) =\frac{1}{s-c}\text{,} \end{equation*}

and thus

\begin{equation*} X(s) = \frac{x(0)}{s-a} + \frac{1}{(s-c)(s-a)}\text{.} \end{equation*}

We can immediately recognize the first term as the transform of \(e^{at}x(0)\text{.}\) To invert the transform of the second term, we need to express it using partial fractions. There is a general procedure for this, but it's worth knowing a simple mnemonic for this particular form:

\begin{equation*} \frac{1}{(s-c)(s-a)} = \frac{1}{(a-c)(s-a)} + \frac{1}{(s-c)(c-a)}\text{.} \end{equation*}

Thus the inverse transform of this fraction is a sum of two exponentials,

\begin{equation*} x_p(t) = \frac{1}{c-a}\left( e^{ct}-e^{at} \right)\text{.} \end{equation*}

This is a particular solution that has \(x_p(0)=0\text{.}\) The general solution is \(e^{at}x(0) + x_p(t)\text{.}\)

Finding the transform of a step is also pretty easy:

\begin{align*} \lx[H(t-T)] \amp = \int_T^\infty e^{-st}\, dt \\ \amp = -\frac{1}{s} \Bigl[ e^{-st} \Bigr]_T^\infty = \frac{e^{-sT}}{s} \text{.} \end{align*}
Fact 2.8.5. Transform of a step.
\begin{equation} \lx[H(t-T)] = \frac{e^{-sT}}{s}\text{.}\label{fl-eq-ltstep}\tag{2.8.3} \end{equation}

A generalization of the same calculation results in a very important rule.

Theorem 2.8.6. Shift theorem.
\begin{equation} \lx[H(t-T)f(t-T)] = e^{-sT} \lx[f(t)]\text{.}\label{fl-eq-ltshift}\tag{2.8.4} \end{equation}

To get the transform of an impulse, we can play the same game as before. The transform of our narrow step \(\delta_\epsilon\) is

\begin{equation*} \lx[ \delta_\epsilon(t-T)] = \frac{1}{\epsilon} \left( \lx[H(t-T)] - \lx[H(t-T-\epsilon) \right) = \frac{1 - e^{\epsilon s}}{\epsilon s}e^{-sT}\text{.} \end{equation*}

Taking the limit as \(\epsilon\to 0\) gives the following.

Fact 2.8.7. Transform of an impulse.
\begin{equation} \lx[\delta(t-T)] = e^{-sT}\text{.}\label{fl-eq-ltimpulse}\tag{2.8.5} \end{equation}

An important special case is \(T=0\text{:}\)

\begin{equation*} \lx[\delta(t)] = 1\text{.} \end{equation*}
Example 2.8.8.

Let's consider \(x'=ax + \delta(t-T)\text{,}\) with \(x(0)=0\text{.}\) Transform to get

\begin{equation*} [sX(s)-x(0)] - aX(s) = e^{-sT}\text{,} \end{equation*}

and thus

\begin{equation*} X(s) = \frac{e^{-sT}}{s-a}\text{.} \end{equation*}

Notice that \(X(s) = e^{-sT} Y(s)\text{,}\) where \(Y(s)=1/(s-a)\) is the transform of \(y(t)=e^{at}\text{.}\) By the shift theorem, it must be that

\begin{equation*} x(t) = H(t-T)y(t-T) = H(t-T)e^{a(t-T)}\text{.} \end{equation*}

Subsection 2.8.3 Transfer function

Perhaps it's clear by now that \(x'-ax=f(t)\) always leads to \(X(s) = F(s)/(s-a)\) as a particular solution. The term \(1/(s-a)\) is called the transfer function of the ODE. Values of \(s\) that are roots of the denominator are known as poles of the transform. They point out exponential solutions.

The impulse response is the case when \(f(t)=\delta(t)\text{,}\) so that \(F(s)=1\text{.}\) This implies the following.

Fact 2.8.9.

The transfer function is the Laplace transform of the impulse response.
