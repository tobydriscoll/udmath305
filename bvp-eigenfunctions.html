Section 6.2 Eigenfunctions

A few examples get us pointed in the right direction.

Example 6.2.1

Our first TPBVP is the rather sedate-looking \(y''+y=0\text{,}\) \(y(0)=y(1)=0\text{.}\) (Note that the domain for \(x\) is implied by the locations of the boundaries.) The general solution is \(y=c_1\cos(x) + c_2\sin(x)\text{,}\) and then the boundary conditions imply

\begin{align*} 0 \amp = y(0) = c_1\\ 0 \amp = y(1) = c_1\cos(1) + c_2\sin(1)\text{.} \end{align*}

The coefficient matrix of the linear system here has determinant \(\sin(1)\neq 0\text{,}\) so the unique solution of the linear system is \(c_1=c_2=0\text{.}\) So the only solution of the TPBVP is \(y(x)\equiv 0\text{.}\)

Example 6.2.2

Now let's consider \(y''+y=0\text{,}\) \(y(0)=y(\pi)=0\text{.}\) The general solution is still \(y=c_1\cos(x) + c_2\sin(x)\text{,}\) and the boundary conditions imply

\begin{align*} 0 \amp = y(0) = c_1\\ 0 \amp = y(1) = c_1\cos(\pi) + c_2\sin(\pi) = -c_1\text{.} \end{align*}

We can require only \(c_1=0\) , and \(c_2\) remains free. So \(y(x)=C \sin(x)\) is a solution for arbitrary \(C\text{.}\)

Example 6.2.3

Here's a third go at almost the same problem, but this time we allow a parameter: \(y''+\mu^2 y=0\text{,}\) \(y(0)=y(\pi)=0\text{,}\) where \(\mu^2\) is an unspecified nonnegative constant (hence we require only that \(\mu\) is real). The general solution here is \(y(x)=c_1\cos(\mu x) + c_2\sin(\mu x)\text{.}\) The linear system arising from the initial conditions is now

\begin{equation*} \twomat{1}{0}{\cos(\mu \pi)}{\sin(\mu \pi)} \twovec{c_1}{c_2} = \bzero\text{.} \end{equation*}

The matrix is singular if and only if \(\sin(\mu\pi)=0\text{,}\) or \(\mu = 0,\pm1,\pm2,\dots\text{.}\) At each of these values of \(\mu\text{,}\) every multiple of \(\sin(\mu x)\) is a solution. The zero function is the only solution at any other choice of \(\mu\text{.}\) In fact we ought to rule out \(\mu=0\) as well, because then the solution is again identically zero.

The last example above implied that the TPBVP \(y''+\lambda y=0\text{,}\) \(y(0)=y(\pi)=0\) is special when \(\lambda=\mu^2=1^2,2^2,3^2,\ldots\text{.}\) We call these the eigenvalues of the BVP, and the associated nonzero solutions are called eigenfunctions. The analogy with matrices here is very close. An eigenfunction is a nonzero member of the "nullspace" of the BVP, and these nullspaces are nontrivial only for certain discrete values of \(\lambda\text{.}\) One difference here is that the BVP has infinitely many eigenvalues, which turns out to be typical.

In the above we did not allow the possibility of negative eigenvalues. So let us try for \(y''-\mu^2 y=0\text{,}\) \(y(0)=y(\pi)=0\text{,}\) where \(\mu\) is real and nonzero. Now the general solution is \(y(x)=c_1e^{-\mu x} + c_2 e^{\mu x}\text{,}\) and the system for the boundary conditions is

\begin{equation*} \twomat{1}{1}{e^{-\mu \pi}}{e^{\mu \pi}} \twovec{c_1}{c_2} = \bzero\text{.} \end{equation*}

Singularity now requires \(e^{\mu \pi}-{e^{\mu \pi}=0\text{,}\) or \(e^{2\mu \pi}=1\text{,}\) or \(\mu=0\text{.}\) So we can't find any negative eigenvalues.

Let's close this section with a broadening observation. If we consider just \(y''+\lambda y=0\) alone without any boundaries at all, then the solutions are combinations of \(\exp(i\sqrt{\lambda} x)\) and \(\exp(-i\sqrt{\lambda} x)\text{.}\) Potentially every number is an eigenvalue! But it's when we start adding conditions that the possibilities get narrowed. For instance, if we want the eigenfunctions to be bounded as \(x\to\pm\infty\text{,}\) then we must demand that \(\lambda \gt 0\text{,}\) so that the exponents are pure imaginary. Thus the eigenfunctions are \(e^{\pm i\omega x}\) for real \(\omega\text{.}\) If we also want them to be \(2\pi\)-periodic, then we must also require that \(\omega=k\text{,}\) a positive integer. Then the eigenfunctions are our orthogonal friends, \(e^{\pm i k x}\text{.}\) Finally, if we want to force them to be zero at two specific points, the "two-dimensional" combination possibilities collapse down to only \(e^{ikx}-e^{-ikx}\text{,}\) or \(\sin(kx)\text{.}\) The differential operator \(L[y]=y''\) suggests a universe of possible eigenfunctions, but the extra conditions from the boundaries cut down the options.
