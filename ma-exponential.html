Section 4.9 Matrix exponential

¶

From section 6.4.

Let's take another look at \(\by'=\bA\by\) in the diagonalizable case with \(\bA=\bV\bD\bV^{-1}\text{.}\) Again we define \(\bu=\bV^{-1}\by\) and get the diagonal system \(\bu'=\bD\bu\text{,}\) with solution

\begin{equation*} \bu(t) = \begin{bmatrix} e^{\lambda_1 t} u_1(0) \\ e^{\lambda_2 t} u_2(0) \\ \vdots \\ e^{\lambda_n t} u_n(0) \end{bmatrix} = \begin{bmatrix} e^{\lambda_1 t} \amp \amp \amp \\ \amp e^{\lambda_2 t} \amp \amp \\ \amp \amp \ddots \amp \\ \amp \amp \amp e^{\lambda_n t} \end{bmatrix} \bu(0)\text{.} \end{equation*}

Diagonal matrices are a snap to multiply:

\begin{equation*} \bD^k = \fourdiag{\lambda_1^k}{\lambda_2^k}{\lambda_n^k}, \qquad k=1,2,3,\ldots\text{.} \end{equation*}

From here we can make a remarkable leap to a matrix power series,

\begin{equation*} \bI + \bD + \frac{1}{2!}\bD^2 + \frac{1}{3!}\bD^3 + \cdots = \fourdiag{e^\lambda_1}{e^\lambda_2}{e^\lambda_n}\text{.} \end{equation*}

It's irresistable to conclude that we have found the exponential of the diagonal matrix \(\bD\text{:}\)

\begin{equation*} e^\bD = \fourdiag{e^\lambda_1}{e^\lambda_2}{e^\lambda_n}\text{.} \end{equation*}

What's more, we can now write simply,

\begin{equation*} \bu(t) = e^{t\bD} \bu(0)\text{.} \end{equation*}

Returning to the original ODE variable \(\by=\bV\bu\) leads to

\begin{equation*} \by(t) = \bV e^{t\bD} \bV^{-1} \by(0) \text{.} \end{equation*}

Could it be, then, that

\begin{equation*} e^{t\bA} = \bV e^{t\bD} \bV^{-1}\text{?} \end{equation*}

Let's consult the power series:

\begin{align*} e^{t\bA} \amp = \bI + t\bA + \frac{1}{2!}t^2\bA^2 + \frac{1}{3!}t^3\bA^3 + \cdots\\ \amp = \bV\bV^{-1} + t(\bV\bD\bV^{-1}) + \frac{1}{2!}t^2 (\bV\bD\bV^{-1})(\bV\bD\bV^{-1}) + \frac{1}{3!}t^3 (\bV\bD\bV^{-1})(\bV\bD\bV^{-1})(\bV\bD\bV^{-1}) + \cdots \\ \amp = \bV \bigl( \bI + t\bD + \frac{1}{2!}t^2\bD^2 + \frac{1}{3!}t^3\bD^3 + \cdots \bigr) \bV^{-1}\text{,} \end{align*}

and we have arrived. Therefore,

\begin{equation*} \by(t) = e^{t\bA} \by(0) \end{equation*}

is the solution of \(\by'=\bA\by\text{.}\)

Example 4.9.1

We look again at an example fron the previous section,

\begin{equation*} \bA = \begin{bmatrix} 1 \amp 1 \\ 4 \amp 1 \end{bmatrix}\text{,} \end{equation*}

in which \(\lambda_1=3\text{,}\) \(\bv_1 = \twovec{1}{2}\text{,}\) and \(\lambda_2=-1\text{,}\) \(\bv_2=\twovec{1}{-2}\) are eigenpairs. We define

\begin{equation*} \bV = \twomat{1}{1}{2}{-2} \end{equation*}

and compute

\begin{equation*} \bV^{-1} = \frac{1}{4}\twomat{2}{1}{2}{-1}\text{.} \end{equation*}

Hence

\begin{equation*} \by(t) = \frac{1}{4} \twomat{1}{1}{2}{-2} \twodiag{e^{3t}}{e^{-t}} \twomat{2}{1}{2}{-1} \by(0)\text{.} \end{equation*}

Subsection 4.9.1 Properties

Many of the things you know about scalar exponential functions hold true for matrices as well.

  • \(e^\bzero = \bI\)
  • \(\bigl(e^{t\bA}\bigr)^{-1} = e^{-t\bA}\)
  • \(\dd{}{t} e^{t\bA} = \bA e^{t\bA} = e^{t\bA} \bA\)

However, there is one property we have to let go of, at least partly:

\begin{equation*} e^{\bA+\bB} \neq e^{\bA} e^{\bB} \quad \text{ unless }\quad \bA\bB=\bB\bA\text{.} \end{equation*}

This has some far-reaching consequences that we won't get a chance to explore.

Subsection 4.9.2 Complex and defective cases

Example 4.9.2

It's routine to show that

\begin{equation*} \bA = \twomat{0}{1}{-1}{0} = \twomat{1}{1}{i}{-i} \twodiag{i}{-i} \frac{1}{2} \twomat{1}{-i}{1}{i}\text{.} \end{equation*}

From here we find that

\begin{align*} e^{t\bA} \amp = \twomat{1}{1}{i}{-i} \twodiag{e^{it}}{e^{-it}} \frac{1}{2} \twomat{1}{-i}{1}{i}\\ \amp = \frac{1}{2} \twomat{e^{it}}{e^{-it}}{ie^{it}}{-ie^{-it}} \twomat{1}{-i}{1}{i} \\ \amp = \frac{1}{2} \twomat{2\cos(t)}{-i[2i\sin(t)]}{i[2i\sin(t)]}{2\cos(t)}\\ \amp = \twomat{\cos(t)}{\sin(t)}{-\sin(t)}{\cos(t)}\text{.} \end{align*}

This matrix applies a rotation of an angle \(t\) about the origin to all vectors in the plane. Thus, the phase portrait consists of concentric circles and the origin is a center.

Example 4.9.3

The problem \(\mathbf{y}' = \begin{bmatrix} -1 \amp 1 \\ 0 \amp -1 \end{bmatrix} \mathbf{y}\) has a defective matrix, so we cannot use a diagonalization to compute the matrix exponential. Surprisingly, though, the result can be even easier to find!

We will use the fact that

\begin{equation*} \exp(\bA t+\bI t) = \exp(\bA t) \exp(\bI t) = e^{t} e^{\bA t}\text{.} \end{equation*}

Again, we cannot do the first step above for general matrices, but it is clearly true that \((\bA t)(\bI t)=(\bI t)(\bA t)\text{,}\) so it's okay here. The second step above just uses \(e^{t\bI} = e^t \bI\text{.}\)

Now for the fun part. By the power series definition,

\begin{equation*} \exp(\bA t+\bI t) = \bI + t (\bA+\bI) + \frac{1}{2}t^2 (\bA+\bI)^2 + \cdots\text{.} \end{equation*}

But note that

\begin{equation*} (\bA+\bI)^2 = \twomat{0}{1}{0}{0} \twomat{0}{1}{0}{0} = \bzero\text{.} \end{equation*}

So all the higher matrix powers vanish too, and

\begin{equation*} \exp(\bA t+\bI t) = \bI + t (\bA+\bI) = \twomat{1}{t}{0}{1}\text{.} \end{equation*}

Finally, then,

\begin{equation*} e^{t\bA} = e^{-t} e^{t\bA+t\bI} = e^{-t} \twomat{1}{t}{0}{1}\text{.} \end{equation*}

We had a double eigenvalue at \(-1\) and, to nobody's surprise, \(te^{-t}\) came out to play.

Subsection 4.9.3 Equation with forcing term

Suppose we now have \(\by'=\bA\by + \bff(t)\) for a constant matrix \(\bA\text{.}\) We can solve this problem using our old friend, the integrating factor. Define \(\mathbf{M}(t) = e^{-t \bA}\text{,}\) and multiply through by it. Then

\begin{align*} \dd{}{t} \bigl[ e^{-t \bA} \by(t) \bigr] \amp = e^{-t \bA} \bff(t)\text{,} \end{align*}

and we solve to get

\begin{equation*} \by(t) = e^{t \bA}\by(0) + \int_0^t e^{(t-s) \bA} \bff(s)\, ds\text{,} \end{equation*}

which is a lot like our growth-factor formula for first-order scalar equations.

This seems like a good place to stop with the matrices.
