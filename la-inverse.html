Section 1.7 Inverses

¶

Subsection 1.7.1 Inverse matrix

For scalars, the multiplicative inverse of \(a\) is the reciprocal, \(a^{-1}\text{,}\) except that zero has no inverse. Now we investigate the same question for square matrices.

Definition 1.7.1. Inverse matrix.

The inverse of an \(n\times n\) matrix \(\bA\) is a matrix \(\bC\in\Rmn{n}{n}\) such that \(\bA\bC = \bC\bA = \bI \text{.}\) If such a \(\bC\) exists, we write \(\bA^{-1}=\bC\) and say that \(\bA\) is invertible; otherwise, it is singular.

Inverses are possible for square matrices only. The concept does not directly translate to rectangular matrices.

From the start there are two important things to appreciate about inverses.

Fact 1.7.2.

  1. If a matrix has an inverse, it is unique. (So it makes sense to talk about “the” inverse of a matrix.)

  2. \(\bA\mathbf{Z}=\bI\) if and only if \(\mathbf{Z}\bA=\bI\text{.}\) (So we don't need to separately think about left-inverse and right-inverse.)

Because the zero matrix gives zero for every multiplicative partner, it cannot have an inverse. However, unlike scalars, there are nonzero matrices that are singular. The simplest is

\begin{equation*} \bA = \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

(Proof: The first column of \(\bC\bA\) is zero for any square \(\bC\text{.}\))

Subsection 1.7.2 Inverses and linear systems

If \(\bA\in\Rmn{n}{n}\) is invertible, then we can use its inverse to solve a linear system \(\bA\bx=\bb\text{:}\)

\begin{equation*} \bA^{-1} \bb = \bA^{-1} (\bA \bx) = \bigl( \bA^{-1} \bA \bigr) \bx = \bI \bx = \bx\text{.} \end{equation*}

The uniqueness of the inverse means that this solution is unique as well. We will see later on that the converse is also true: unique solution implies invertibility.

It's one thing to say “the inverse exists” and quite another to say “here it is.” Finding the inverse of a given matrix is in principle a matter of solving \(n\) linear systems. That's because if we let \(\mathbf{e}_j\) be the \(j\)th column of \(\bI\text{,}\) then according to the columnwise interpretation of \(\bA\bA^{-1}\text{,}\) the solution of \(\bA \bx = \mathbf{e}_j\) is the \(j\)th column of \(\bA^{-1}\text{.}\)

Row elimination can be used to solve those systems simultaneously. However, it's all beside the point. If our goal is to solve a linear system \(\bA\bx=\bb\text{,}\) then there is little reason to solve \(n\) other systems first to get the inverse! The inverse tends to be much more important theoretically than for practical computation.

Example 1.7.3.

MATLAB does have an inv command to find an inverse. However, it's not considered the best way to solve a linear system (it will be much slower for large matrices).

MATLAB works with 16-digit approximations of real numbers, so things that are truly equal in exact arithmetic may only agree to many significant digits. Usually it's best to take the difference of two things to test near-equality.

Subsection 1.7.3 \(2\times 2\) inverses

There does happen to be a convenient formula for the matrix inverse in the \(2\times 2\) case:

\begin{equation} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}^{-1} = \frac{1}{ad-bc} \begin{bmatrix} d \amp -b \\ -c \amp a \end{bmatrix}\text{.}\label{la-eq-twotwoinv}\tag{1.7.1} \end{equation}

This formula breaks down if \(ad=bc\text{,}\) in which case the matrix is singular. You might remember that this is the same algebraic condition quoted for the \(2\times 2\) linear system \(\bA\bx=\bb\) not having a unique solution.
