Section 1.5 General and particular solutions

¶

Our next step is to observe that solutions to \(\bA\bx=\bb\) can all be expressed as \(\bx=\bx_h+\bx_p\text{,}\) with the two parts belonging to different sets. The same thing is going to happen when we study differential equations, pretty much for exactly the same reasons.

Subsection 1.5.1 Homogeneous solutions

The linear system \(\bA\bx=\bzero\) plays a special role and is called a homogeneous linear system.

Theorem 1.5.1. Homogeneous linear systems.

If \(\bx_1,\ldots,\bx_k\) are solutions of a homogeneous system \(\bA\bx=\bzero\text{,}\) then every linear combination of them is also a solution.

Proof.

Let \(c_1\bx_1 + \cdots + c_k\bx_k\) be such a linear combination. Then

\begin{align*} \bA(c_1\bx_1 + \cdots + c_k\bx_k) \amp = \bA(c_1 \bx_1 ) + \cdots + \bA(c_k \bx_k) \\ \amp = c_1\bA\bx_1 + \cdots + c_k\bA \bx_k \\ \amp = \bzero\text{.} \end{align*}

By itself this is nice, but what makes this theorem really important is that every homogeneous solution can obtained this way.

Example 1.5.2.

Suppose

\begin{equation*} \bA = \begin{bmatrix} 0 \amp -2 \amp 4 \\ 0 \amp 1 \amp -2 \\ 0 \amp 3 \amp -6 \end{bmatrix}\text{.} \end{equation*}

The RREF of the augmented matrix \([\bA,\bzero]\) is found to be

\begin{equation*} \bA = \begin{bmatrix} 0 \amp 1 \amp -2 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

Hence the only pivot column is column 2, and all solutions are of the form

\begin{equation*} \threevec{s}{2t}{t} = s \threevec{1}{0}{0} + t \threevec{0}{2}{1}\text{,} \end{equation*}

where \(s\) and \(t\) are arbitrary scalars. You can easily check that \([1,0,0]\) and \([0,2,1]\) are each solutions of \(\bA\bx=\bzero\text{.}\)

Motivated by the theorem and example above, we make a new definition.

Definition 1.5.3. General solution of a linear system (homogeneous).

We say that \(\bx_h = c_1\bx_1 + \cdots + c_k\bx_k\) is the general solution of \(\bA\bx=\bzero\) if every solution to the system can be so written for some choice of the coefficients.

Subsection 1.5.2 Particular solutions

¶

We now come to one of the simplest yet most confusing terms in the course. A particular solution of the linear system \(\bA\bx=\bb\) is just any one solution of the problem. The only reason the term exists is to distinguish it from the general solution, which (as above) is an expression for every possible solution of the system.

Theorem 1.5.4. General solution of a linear system.

All solutions of \(\bA\bx=\bb\) may be written as

\begin{equation*} \bx = \bx_h + \bx_p\text{,} \end{equation*}

where \(\bx_h\) is the general solution of \(\bA\bx=\bzero\) and \(\bx_p\) is any particular solution of \(\bA\bx=\bb\text{.}\) We call this the general solution of \(\bA\bx=\bb\text{.}\)

Proof.

Let \(\bx_p\) be a particular solution, and suppose \(\bu\) is another solution. Then

\begin{equation*} \bA(\bu-\bx_p)=\bA\bu - \bA\bx_p = \bb - \bb = \bzero\text{.} \end{equation*}

Hence \(\bu-\bx_p\) is a homogeneous solution, which means that we can write it as \(\bx_h\text{.}\)

Thus, one way to attack \(\bA\bx=\bb\) is to first find the general homogeneous solution, then find any particular solution of the nonhomogeneous problem. You will do this many times over for differential equations in future chapters.

Example 1.5.5.

We continue with the matrix \(\bA\) from Example 1.5.2, now with \(\bb=[-8,4,12]\text{.}\) The RREF of the augmented matrix works out to be

\begin{equation*} \begin{bmatrix} 0 \amp 1 \amp -2 \amp 4 \\ 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

The general homogeneous solution is found, as before, by replacing the last column with zeros to get

\begin{equation*} \bx_h = s \threevec{1}{0}{0} + t \threevec{0}{2}{1}\text{.} \end{equation*}

For the particular solution we have free variables from columns 1 and 3. But since we can use any particular solution, we will take the easy route and set the free variables to zero. Then the first row tells us that \(x_2=4\text{,}\) and we get the general solution

\begin{equation*} \bx = \threevec{0}{4}{0} + s \threevec{1}{0}{0} + t \threevec{0}{2}{1}\text{.} \end{equation*}

This process doesn't really save us any algebraic work right now, because we would have gotten the same expression for the solution directly from the RREF. So you might be wondering what the point of this section is. Have patience, grasshopper.
