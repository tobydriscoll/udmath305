We can find the transform of an exponential function from the transform definition. For any complex number \(c\text{,}\)

\begin{equation*} {\cal L}[e^{c t}] = \int_0^\infty e^{(c-s)t}\, dt = \left[ -\frac{1}{s-c} e^{(c-s)t} \right]_0^\infty = \frac{1}{s-c}\text{.} \end{equation*}

(To make this valid requires that \({\text Re}(c)\lt s\text{,}\) but we are going to ignore such technicalities.) An important special case is when \(c=0\text{:}\) \({\cal L}[1]=1/s\text{.}\)
