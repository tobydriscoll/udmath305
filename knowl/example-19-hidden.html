Suppose \(y'=t^2/(y^3-2)\text{.}\) Separation and integration lead to

\begin{equation*} \int y^3\, dy = \int t^2\, dt\text{,} \end{equation*}

or \(\frac{1}{4}y^4 - 2y = C + \frac{1}{3}t^3\text{.}\) We could work hard to try to solve explicitly for \(y\text{,}\) but it's probably best to leave it in implicit form. This is a common limitation to separable "solutions".

Even in implicit form, we can solve for the arbitrary constant if given an initial condition. For instance, suppose \(y(0)=4\) is given for this ODE. Then \(4^4/4-2\cdot4 = C+0\text{,}\) so \(C=56\text{.}\)
