As a matter of fact, these properties combine in a way worth noting. We can define a function \(L\) on vectors by \(L(\bx)=\bA\bx\text{.}\) This is what we call a linear function, because \(L(c\bx + \by)=cL(\bx)+L(\by)\) for all \(n\)-vectors \(\bx,\by\) and scalars \(c\text{.}\) This would not be true for, for example, \(L(\bx)=x_1x_2\text{.}\)
