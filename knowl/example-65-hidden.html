To solve \(3x''+6x'+30x=0\text{,}\) we first should divide through by 3 to get the standard form. Then the characteristic polynomial is \(\lambda^2 + 2\lambda + 10\text{,}\) which has roots \(-1\pm 3i\text{.}\) So \(x(t) = e^{-t}[c_1 e^{3it} + \overline{c_1}e^{-3it}]\text{.}\) Equivalently, \(x(t) = e^{-t}[a_1 \cos(3t) + a_2\sin(3t)]\text{,}\) or \(x(t) = R e^{-t}\cos(3t-\theta)\text{.}\)
