Let's solve the system having augmented matrix

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \amp 4 \\ 1 \amp 2 \amp 2 \amp 3 \\ 2 \amp 1 \amp -5 \amp 9 \end{bmatrix}\text{.} \end{equation*}

Use multiples of the first row to eliminate below it.

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \amp 4 \\ 0 \amp 1 \amp 3 \amp -1 \\ 0 \amp -1 \amp -3 \amp 1 \end{bmatrix} \end{equation*}

Use a multiple of the second row to eliminate below that.

\begin{equation*} \begin{bmatrix} 1 \amp 1 \amp -1 \amp 4 \\ 0 \amp 1 \amp 3 \amp -1 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix} \end{equation*}

We have reached the last row, and it's time to turn around and eliminate upwards. But we can't, because the last row is entirely zero. So skip that and move up to the next row, eliminating upwards again in the second column:

\begin{equation*} \begin{bmatrix} 1 \amp 0 \amp -1 \amp 5 \\ 0 \amp 1 \amp 3 \amp -1 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}

This is as simple as we can get things. The last row is telling us the astounding fact that \(0=0\text{,}\) i.e., nothing at all. The fact that there was nothing there to determine \(x_3\) means that we can set it to be anything we like, say \(x_3=s\) for arbitrary \(s\text{.}\) Now the second row says

\begin{equation*} x_2 = -1 - 3s\text{,} \end{equation*}

which is as specific as we can get. Finally, the first row says

\begin{equation*} x_1 = 5 + s\text{.} \end{equation*}

So \([5+s,-1-3s,s]\) is a solution for every \(s\text{,}\) and we have infinitely many solutions.
