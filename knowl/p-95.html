We're about to embark on some nonlinear ODEs. First consider a generic first-order problem in the form \(y'=f(y)\text{.}\) Even before attempting a full solution of the problem, we can see that a value \(y_0\) that satisfies \(f(y_0)=0\) is special, because then we get \(y'=0\) and the solution is constant. Such a value is called a steady state or equilibrium.
