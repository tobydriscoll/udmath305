There is a shorthand method for deriving the solutions of linear, constant-coefficient equations that have forcing functions such as exponentials, steps, and impulses. Given (almost) any function \(f(t)\text{,}\) there is a doppelganger function \(F(s)\) defined by an integral.

Definition 2.9.1. Laplace transform.

The Laplace transform of a function \(f(t)\) is defined as

\begin{equation} \lx[f(t)] = F(s) = \int_0^\infty f(t) e^{-st}\, dt\text{.}\tag{2.9.1} \end{equation}
In this business, time starts at \(t=0\text{,}\) and any reference to \(f(t)\) for negative \(t\) is defined to be zero. A convention is that a lowercase function of \(t\) transforms to its uppercase namesake as a function of \(s\text{.}\) (Another convention is that \(\lx\) is used with curly braces, but we have already commited to using square brackets for operators.)
