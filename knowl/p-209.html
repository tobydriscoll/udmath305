We can go far with some simple calculus. Let's try an exponential solution,

\begin{equation*} x(t) = \exp[ q(t) ]\text{,} \end{equation*}

for a \(q(t)\) to be defined momentarily. Then by the chain rule,

\begin{equation*} x'(t) = q'(t) x(t)\text{.} \end{equation*}

So \(x\) is a solution of \(x'-a(t)x=0\) if \(a(t)=q'(t)\text{,}\) that is, \(q\) is an antiderivative of \(a\text{.}\) The general homogeneous solution is thus

\begin{equation} x_h(t) = c_1 \exp\left( \int a(t)\,dt \right)\text{,}\tag{2.3.1} \end{equation}

for arbitrary constant \(c_1\text{.}\) It's implicit that any antiderivative can be used here; i.e., the “plus C” you usually have to include can just be left out, because it's absorbed into \(c_1\) anyway.
