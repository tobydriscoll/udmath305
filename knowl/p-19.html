Now, at this point we don't know much else about \(f(t)\text{.}\) But we can show next using the ODE that it has the key property of an exponential function: \(f(A+B)=f(A)f(B)\) for any values of \(A\) and \(B\text{.}\) To see this, start with \(y(A+B)=y(0)f(A+B)\text{.}\) Now imagine letting the ODE solution evolve from \(t=0\) to \(t=A\text{,}\) where we freeze time for a moment. We have that \(y(A)=y(0)f(A)\text{,}\) of course. Now suppose we let things continue to evolve by \(B\) additional time units. The solution becomes \(y(A)f(B)=y(0)f(A)f(B)\text{.}\) But this is the same as \(y(A+B)\text{,}\) and we are done.
