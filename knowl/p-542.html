\begin{equation*} \dfrac{As+B}{s^2+\omega^2} = \frac{C}{s-i\omega} + \frac{D}{s+i\omega} \end{equation*}

leads to

\begin{equation*} As + B = C(s+i\omega) + D(s-i\omega). \end{equation*}

Setting \(s=i\omega\) gives

\begin{equation*} B + i\omega A = 2i\omega C\text{,} \end{equation*}

or

\begin{equation*} C = \frac{1}{2}\,A - \frac{i}{2\omega} \, B\text{.} \end{equation*}

Using the other pole (or conjugation) gives \(D=\frac{A}{2} + \frac{iB}{2\omega}\text{.}\) Upon taking the inverse transform we get

\begin{equation*} \left( \frac{A}{2} - \frac{iB}{2\omega} \right) e^{i\omega t} + \left( \frac{A}{2} + \frac{iB}{2\omega} \right) e^{-i\omega t} \text{,} \end{equation*}

which rearranges to

\begin{equation*} \frac{A}{2} \left( e^{i\omega t} + e^{-i\omega t}\right) - i \frac{B}{2\omega} \left( e^{i\omega t} - e^{-i\omega t}\right)\text{.} \end{equation*}

Euler's identity finishes the job.
