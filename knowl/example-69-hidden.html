For \(x''+x'=\sin(2t)\text{,}\) we use

\begin{equation*} x_p = A\cos(2t) + B\sin(2t)\text{.} \end{equation*}

Inserting this into the ODE leads to

\begin{equation*} [-4A\cos(2t) - 4B\sin(2t)] + [-2A\sin(2t) + 2B\cos(2t)] = \sin(2t)\text{.} \end{equation*}

This is true for all time if and only if we match the coefficients of like trig functions:

\begin{equation*} -4A + 2B = 0, \qquad -4B-2A = 1 \text{.} \end{equation*}

The solution of these equations is \(A=-1/10\text{,}\) \(B=-1/5\text{.}\)
