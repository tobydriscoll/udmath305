With forcing \(f(t)=e^{i\omega t}\) we get a solution \(y_p=Y(i\omega) e^{i\omega t}\text{.}\) If we write the transfer function in polar form, we can express this as \(y_p = (G e^{-i\alpha }) e^{i\omega t} = Ge^{i(\omega t - \alpha)}\) for positive amplitude \(G\text{,}\) which we call the gain. It depends on the forcing frequency; in fact, \(G(\omega) = |Y(i\omega)|\text{.}\) The \(\alpha\) term is a phase lag that shows how the response may lag behind the driving force. It's usually not as interesting as the gain.
