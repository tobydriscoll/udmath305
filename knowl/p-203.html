You're familiar with a function as a rule mapping a value (number) to another value. A rule for mapping functions to other functions is called an operator. The first nontrivial operator you ever encountered was differentiation, the map from \(u(x)\) to \(u'(x)\text{.}\) We sometimes write this operator as a capital D: \(u'=D[u]\text{.}\) (Many times we leave out the square brackets, too.)
