Time for some deja vu. When we solved \(y'-at=e^{ct}\) in Subsection 1.4.4, we found that we got different formulas in the case \(c=a\text{.}\) We also needed a special formula for \(y_n\) when we got a double root in Subsection 2.3.1 The same sort of thing happens to \(y_p\) if we force an oscillator at one of the characteristic exponents. The reason is simple: if we try \(y_p=Ye^{rt}\) and \(r\) is a characteristic root, then we get \(0=f(t)\text{,}\) which can't be solved. Equivalently, the transfer function \(Y\) requires taking \(1/0\text{.}\)
