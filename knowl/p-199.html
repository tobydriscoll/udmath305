The mechanical oscillator \(ay'' + by' + cy = f(t)\) is more conveniently expressed as

\begin{equation*} y'' + \frac{b}{a} y' + \frac{c}{a} y + \frac{c}{a} F(t)\text{.} \end{equation*}

One reason to prefer this form is that \(y\) must have the same units as the forcing \(F\text{.}\) Furthermore, we see the undamped natural frequency \(\omega_n=\sqrt{c/a}\text{,}\) and this has units of inverse time only. The same units apply to \(b/a\text{,}\) so we introduce the new dimensionless parameter

\begin{equation*} \zeta = \frac{1}{2} \frac{b/a}{\omega_n} = \frac{b}{\sqrt{4ac}}\text{.} \end{equation*}

Now the ODE becomes

\begin{equation*} y'' + 2 \zeta \omega_n y' + \omega_n^2 y + \omega_n^2 F(t)\text{.} \end{equation*}
