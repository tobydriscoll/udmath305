For \(f(t)=e^{rt}\) we let \(y_p=Y e^{rt}\) and plug into the ODE:

\begin{equation*} (ar^2 + br + c)Ye^{rt} = e^{rt}\text{.} \end{equation*}

This is identically satisfied if we choose

\begin{equation*} Y = \frac{1}{ar^2 + br + c}\text{.} \end{equation*}

In various contexts we call this the exponential response function or the transfer function. It's constant in time, but a function of the forcing exponent \(r\) (which is usually called \(s\) in transfer functions).
