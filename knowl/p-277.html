A linear system of ODEs is an equation of the form

\begin{equation} \bx' = \bA(t)\bx + \bff(t)\text{,}\tag{3.1.1} \end{equation}

where \(\bx\) is an \(n\)-dimensional state vector, \(\bA\) is an \(n\times n\) coefficient matrix, and \(\bff(t)\) is an \(n\)-dimensional forcing function. If given, an initial condition of the system is a time \(t_0\) and vector \(\bx_0\) such that \(\bx(t_0)=\bx_0\text{.}\)
