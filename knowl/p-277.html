In this business, time starts at \(t=0\text{,}\) and any reference to \(f(t)\) for negative \(t\) is defined to be zero. A convention is that a lowercase function of \(t\) transforms to its uppercase namesake as a function of \(s\text{.}\) (Another convention is that \(\lx\) is used with curly braces, but we have already committed to using square brackets for operators.)
