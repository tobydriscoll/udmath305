First note that we can recast the nonlinear problem as a first-order system in two variables. Define \(x=\theta\) and \(y=\theta'\text{.}\) Then

\begin{align*} \dd{x}{t} \amp = y \\ \dd{y}{t} \amp = -\frac{g}{L}\sin(x)\text{.} \end{align*}

This trick works for any nonlinear second-order equation \(\theta''=g(t,\theta,\theta')\text{.}\) Thus we can focus on problems in the general form

\begin{align*} \dd{x}{t} \amp = F(x,y) \\ \dd{y}{t} \amp = G(x,y)\text{,} \end{align*}

or \(\bu' = \bff(\bu)\) for 2-vector \(\bu(t)\text{.}\) Note that (as we saw before), the vector \(\bu\) represents the state of the system.
