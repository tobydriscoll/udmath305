With these two special solutions in hand, we can write the general solution of (1.3.1) as \(y=y_n+y_p\text{.}\) Every solution of the ODE can be written in that form. This structure pertains only to linear problems. We don't always approach practically solving the problem this way, but we can always recognize the structure after the fact. It can be useful to know that the structure exists even when exact formulas can't be found for the null and particular solutions.
