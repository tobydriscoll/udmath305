In the linear ODE case the role of \(\vec{v}\) is played by a solution of the homogeneous equation, which is (1.3.1) with \(q(t)\equiv 0\text{.}\) With such a solution \(y_h(t)\text{,}\) we define the null solution \(y_n(t)=Cy_h(t)\text{,}\) where it is understood that \(C\) is an arbitrary constant. That is, the null solution is a solution of the homogeneous ODE for any choice of the constant.
