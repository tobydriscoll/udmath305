The undamped oscillator \(x''+9x=\sin(3t)\) has a perfect resonance. The transform suggests the particular solution

\begin{equation*} X_p(s) = \frac{3}{(s^2+9)^2}\text{.} \end{equation*}

We are close to using the fact that

\begin{equation*} \frac{d}{ds} \left[ \frac{3}{s^2+9} \right] = \frac{-6s}{(s^2+9)^2}. \end{equation*}

In fact, if we designate \(G(s)=2sX_p(s)\text{,}\) then we know that

\begin{equation*} \lx[ t \sin(3t) ] = -\frac{d}{ds} \left[ \frac{3}{s^2+9} \right] = G(s), \end{equation*}

and therefore \(g(t)=t\sin(3t)\text{.}\) Next, note that

\begin{equation*} 2\lx[x_p'(t)] = 2[sX_p(s)-X_p(0)] = G(s). \end{equation*}

Hence \(x_p'(t) = \frac{1}{2} t \sin(3t)\text{,}\) and integration by parts yields

\begin{equation*} x_p(t) = \frac{1}{18} [ \sin(3t) - 3t\cos(3t) ]\text{.} \end{equation*}
