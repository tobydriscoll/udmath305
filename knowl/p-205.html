In an initial value problem, we can use the growth factor to modify the variation of parameters formula:

\begin{equation} x(t) = G(t_0,t)x_0 + \int_{t_0}^t G(s,t) f(s)\, ds\text{.}\tag{2.5.1} \end{equation}

We can use TheoremÂ 2.4.3 to show that this is the desired solution. First, \(x(t_0)=G(t_0,t_0)x_0 + 0 = x_0\text{,}\) as required. The derivative is a little tricky, as it requries an extension of the Fundamental Theorem of Calculus:

\begin{equation*} \frac{d}{dt} \int_{t_0}^t G(s,t) f(s)\, ds = G(t,t)f(t) + \int_{t_0}^t \frac{d}{dt} \left[ G(s,t)f(s) \right]\, ds = f(t) + a(t) \int_{t_0}^t G(s,t) f(s)\, ds\text{.} \end{equation*}
