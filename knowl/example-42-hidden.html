Suppose we solve \(x'+5y=3\delta(t-2)\text{,}\) with \(x(0)=1\text{.}\) Start with

\begin{equation*} x(t) = e^{-5t} + 3 \int_0^t e^{-5(t-s)}\delta(s-2)\, ds\text{.} \end{equation*}

If \(t \lt 2\text{,}\) the integration interval does not contain the impulse, so the solution is just \(e^{-5t}\text{.}\) But if \(t \gt 2\text{,}\) \(x(t) = e^{-5t} + 3 e^{-5(t-2)}\text{.}\)

dydt = @(t,y) -5*y;
[t1,y1] = ode45(dydt,[0,2],1);
[t2,y2] = ode45(dydt,[2,6],3+y1(end));
plot([t1;t2],[y1;y2])
xlabel('t'), ylabel('y(t)')
title('Impulse forcing')
semilogy([t1;t2],[y1;y2])
xlabel('t'), ylabel('y(t)')
title('Impulse forcing')
