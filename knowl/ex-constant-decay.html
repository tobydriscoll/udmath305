Example 2.1.3. Constant decay rate.
\begin{equation*} \dd{x}{t} = -y \end{equation*}

Now the model states that the rate of change is negative when \(x\) is positive (and vice versa). So a positive starting value will decrease, but the rate of decrease will then lessen, etc. From this qualitative description alone it's impossible to tell whether the solution ever reaches zero--at which point its rate of change would be zero.

But we can end the suspense by easily verifying that \(x=e^{-t}\) is a solution. This function asymptotically approaches zero as \(t\to\infty\text{,}\) which we call exponential decay. In other, related problems we will find that solutions can decay (or "relax") to any value, not just zero.

x = @(t) exp(-t);
fplot(x,[0,4]), set(gca,'yscale','log')
xlabel('t'), ylabel('x(t)')
title('Exponential decay')
