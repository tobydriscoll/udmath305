Example 4.3.1.

Let's find a particular solution of \(x'' +4x'+4x=8t^2\text{.}\) The correct form of \(x_p\) is a quadratic polynomial, so

\begin{equation*} x_p = A + Bt + Ct^2\text{.} \end{equation*}

Plugging that into the ODE yields

\begin{align*} 2C + 4(B+2Ct) + 4(A+Bt+Ct^2) \amp = 8t^2\\ 4C t^2 + (8C+4B) + (2C+4B+4A) \amp = 8t^2\text{.} \end{align*}

This has to be an identity for all \(t\text{.}\) Matching like powers, we can read off \(C=2\text{,}\) then \(B=-4\text{,}\) and finally \(A=3\text{.}\) This provides us with \(x_p=2t^2+4t+3\text{.}\)
