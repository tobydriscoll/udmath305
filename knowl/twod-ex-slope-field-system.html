Example 3.1.2 Slope fields for a system of two equations

Here is a slope field for the linear ODE system

\begin{align*} \dd{x}{t} \amp = -y\\ \dd{y}{t} \amp = x\text{.} \end{align*} 
F = @(x,y) -y;   G = @(x,y) x;

slopefield(F,G,[-2 2],[-2 2])

Here is a slope field in the first quadrant for the nonlinear system

\begin{align*} \dd{x}{t} \amp = 3x - \frac{1}{2}xy\\ \dd{y}{t} \amp = \frac{1}{4}xy - y\text{.} \end{align*} 
F = @(x,y) 3.*x-x.*y/2;
G = @(x,y) -y + x.*y/4;

slopefield(F,G,[0 10],[0 12])
