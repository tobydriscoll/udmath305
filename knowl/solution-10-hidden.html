The correct IVP is

\begin{equation*} x' + r x = 3 \sum_{n=0}^3 \delta(t-6n), \quad x(0)=0. \end{equation*}

Now we can apply the machinery from this section. With a zero initial value, the homogeneous part of the solution is identically zero. Using delayed impulse responses, term \(n\) from the sum leads to

\begin{equation*} x_n(t) = H(t-6n) e^{-r (t-6n)}. \end{equation*}

So the total solution is

\begin{equation*} x(t) = 3\sum_{n=0}^3 H(t-6n) e^{-r (t-6n)}. \end{equation*}
