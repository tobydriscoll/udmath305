The operator \(L[y]=y'''+2y''+y'+2y\) has characteristic roots \(-2,\pm i\text{.}\) Now note that for any function \(y\text{,}\)

\begin{align*} (D+2)(D-i)(D+i)y \amp = (D+2)(D-i)(y'+iy) \\ \amp = (D+2)\Bigl((y''+iy')-i(y'+iy)\Bigr) \\ \amp = (D+2)(y''+y) = y''' +y' +2y'' + 2y = L[y]\text{.} \end{align*}

We can write the factors in any order, and even things like \(L[y]=(D^2+1)(D-2)\text{.}\)
