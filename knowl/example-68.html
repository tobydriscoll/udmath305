Example 4.7.2.

Suppose \(x''+2x'+10x=6\delta(t-1)\text{.}\) A particular solution has

\begin{equation*} X_p(s) = \frac{6e^{-s}}{s^2+2s+10}\text{.} \end{equation*}

We can use the trick of completing the square to write this as

\begin{equation*} X_p(s) = e^{-s} \frac{6}{(s+1)^2+9} = e^{-s} F(s)\text{,} \end{equation*}

where we can now see that \(f(t) = 2e^{-t}\sin(3t)\text{.}\) By the shift theorem, \(x_p(t)=H(t-1)f(t-1)\text{.}\)

Now consider \(x''+2x'+10x=40\text{.}\) We get

\begin{equation*} X_p(s) = \frac{40}{s(s^2+2s+10)} = \frac{40}{s[(s+1)^2+9]}\text{.} \end{equation*}

We decompose into

\begin{equation*} X_p(s) = \frac{A}{s} + \frac{3B+C(s+1)}{(s+1)^2+9}\text{.} \end{equation*}

Once we find the constants, we know that the inverse of this is \(x_p(t) = A + Be^{-t}\sin(3t) + C e^{-t}\cos(3t)\text{.}\) To find the constants we clear denominators and get

\begin{equation*} 40 = A[(s+1)^2+9] + (3B+Cs+C) s\text{.} \end{equation*}

With \(s=0\) we conclude that \(A=4\text{.}\) From this we find that \(40 = Cs^2+(3B+C)s + 4s^2 +8s + 40\text{,}\) which tells us \(C=-4\) and \(B=-4/3\text{.}\) Hence

\begin{equation*} x_p(t) = 4 - \frac{4}{3} e^{-t} \sin(3t) - 4 e^{-t} \cos(3t)\text{.} \end{equation*}

For a harmonically forced case such as \(x''+2x'+10x=\cos(2t)\text{,}\) we have

\begin{equation*} X_p(s) = \frac{s}{(s^2+4)(s^2+2s+10)}\text{.} \end{equation*}

We could find a decomposition in the form

\begin{equation*} X_p(s) = \frac{2A+Bs}{s^2+4} + \frac{3C+E(s+1)}{(s+1)^2+9}\text{,} \end{equation*}

which is eminently invertible. The details are left to the proverbial reader.
