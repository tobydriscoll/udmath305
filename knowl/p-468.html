We can ease the formulas a little if we define

\begin{equation*} \bu_k = \frac{\bv_k}{\|\bv_k\|}, \quad k=1,\dots,n\text{,} \end{equation*}

that is, we normalize the original orthogonal set of vectors. Then

\begin{equation*} \bx = \inprod{\bu_1}{\bx} \bu_1 + \cdots + \inprod{\bu_n}{\bx} \bu_n\text{.} \end{equation*}

So far this might not look like a compelling idea. But we're about to do the exact same representation with functions.
