Next, the linear system \(( \bA -3 \mathbf{I}) \bv = \boldsymbol{0} \) looks like

\begin{equation*} \begin{bmatrix} -2 \amp 1 \\ 4 \amp -2 \end{bmatrix} \begin{bmatrix} v_1 \\ v_2 \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \end{bmatrix}\text{.} \end{equation*}

By design, this is a singular system, and since zero is a solution, we expect infinitely many solutions. The singularity means that the last row is redundant, so we can focus on solving only \(-2v_1+v_2=0\text{.}\) I.e., any nonzero vector with \(v_2=2v_1\) is an eigenvector. These are all vectors of the form \([c,2c]^T=c[1,2]^T\) for a nonzero scalar \(c\text{.}\)
