Let's consider \(y'-ay=e^{ct}\) in this light. Transforming both sides, we get

\begin{equation*} [sY(s)-y(0)] - aY(s) =\frac{1}{s-c}\text{,} \end{equation*}

and thus

\begin{equation*} Y(s) = \frac{y(0)}{s-a} + \frac{1}{(s-c)(s-a)}\text{.} \end{equation*}

We can immediately recognize the first term as the transform of \(e^{at}y(0)\text{.}\) To invert the transform of the second term, we need to express it using partial fractions. There is a general procedure for this, but it's worth knowing a simple mnemonic for this particular form:

\begin{equation*} \frac{1}{(s-c)(s-a)} = \frac{1}{(a-c)(s-a)} + \frac{1}{(s-c)(c-a)}\text{.} \end{equation*}

Thus the inverse transform of this fraction is a sum of two exponentials,

\begin{equation*} y_v(t) = \frac{1}{c-a}\left( e^{ct}-e^{at} \right) \text{.} \end{equation*}

This is the "very particular" solution that has \(y_v(0)=0\text{.}\) The general solution is \(e^{at}y(0) + y_v(t)\text{.}\) As we noted before, the resonant case follows from the limit \(c\to a\text{,}\) so that \(y_v=te^{at}\text{.}\)
