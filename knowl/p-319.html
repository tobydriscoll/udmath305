We are now going to shift away from talking about the second-order problem to the more general \(\mathbf{u}' = \mathbf{A} \mathbf{u}\text{.}\) It's fair to ask what kinds of phase portraits we can get for this type of problem. The answer is: same as the above. The extra generality in the first row of the matrix can change the details, but there are no new major kinds of behavior beyond sink, source, saddle, and center.
