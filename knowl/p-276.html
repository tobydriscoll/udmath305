Using the given assumptions and mass balances we obtain the system

\begin{align*} \frac{dx}{dt} \amp =-k_1 x, \quad x(0)=A,\\ \frac{dy}{dt} \amp = k_1 x - k_2y, \quad y(0)=0. \end{align*}

Although this takes the form of a system of two equations for two dependent variables, we can solve the first one for \(x\) independently and then put that result into the second ODE. Specifically, \(x(t)=A\exp(-k_1 t)\text{,}\) so

\begin{equation*} y' + k_2 y = k_1 A e^{-k_1t}. \end{equation*}

Let us use undetermined coefficients here. The homogeneous solution is \(y_h(t)=c_1e^{-k_2 t}\text{,}\) and the particular solution has to take the form \(y_p(t)=B e^{-k_1 t}\text{.}\) We find the coefficient through placement into the ODE:

\begin{equation*} (-k_1B e^{-k_1t}) + k_2 (B e^{-k_1 t}) = k_1 A e^{-k_1t}, \end{equation*}

which implies \(B=k_1A/(k_2-k_1)\text{,}\) provided the rate constants are different. (Whatever solution method is chosen would have to deal with that case separately.) Thus

\begin{equation*} y(t) = c_1e^{-k_2 t} + \frac{k_1 A}{k_2-k_1} e^{-k_1 t}. \end{equation*}

We find the unknown \(c_1\) through the condition

\begin{equation*} 0 = y(0) = c_1 + \frac{k_1 A}{k_2-k_1}, \end{equation*}

and finally

\begin{equation*} y(t) = \frac{k_1 A}{k_2-k_1} \left( e^{-k_1 t} - e^{-k_2 t} \right). \end{equation*}
