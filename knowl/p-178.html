Let's put all this together:

\begin{equation*} \bA \bx = (\bV \bD \bV^{-1})\bx = \bV \bD (\bV^{-1}\bx)\text{,} \end{equation*}

which implies that

\begin{equation*} \bV^{-1} ( \bA\bx) = \bD (\bV^{-1}\bx)\text{,} \end{equation*}

which states that the action of \(\bA\) on any vector is diagonal when cast into eigenvector coordinates.
