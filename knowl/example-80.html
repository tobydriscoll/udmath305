Example 4.7.4.

Suppose \(x''+2x'+10x=6\delta(t-1)\text{.}\) A particular solution with zero initial conditions has

\begin{equation*} X_p(s) = \frac{6e^{-s}}{s^2+2s+10}\text{.} \end{equation*}

The quadratic formula tells us the homogeneous eigenvalues \(-1\pm 3i\text{,}\) so we assume a shift of \(-1\) and designate \(X_p(s)=G(s+1)\text{.}\) Thus

\begin{equation*} G(s) = X_p(s-1) = 6e \frac{e^{-s}}{s^2-2s+1+2s-2+10} = 6e \frac{e^{-s}}{s^2+9}. \end{equation*}

Now we can apply the Shift Theorem to \(G\) and get

\begin{equation*} g(t) = 6e H(t-1) v(t-1), \end{equation*}

where \(v(t)=\sin(3t)/3\text{.}\) When we unwind the definitions, we get

\begin{equation*} x_p(t) = e^{-t} g(t) = 2 e^{-t+1} H(t-1)\sin(3t-3). \end{equation*}

Are we having fun yet?

Now consider \(x''+2x'+10x=40\text{.}\) We get

\begin{equation*} X_p(s) = \frac{40}{s(s^2+2s+10)} \text{.} \end{equation*}

We have the same eigenvalues and so define \(X_p(s)=G(s+1)\) to get

\begin{equation*} G(s) = X_p(s-1) = \frac{40}{(s-1)(s^2+9)}. \end{equation*}

This is amenable to the PFD

\begin{equation*} G(s) = \frac{A}{s-1} + \frac{B+Cs}{s^2+9}\text{.} \end{equation*}

Clearing denominators, we get

\begin{equation*} 40 = A(s^2+9) + (B+Cs) (s-1)\text{.} \end{equation*}

If we set \(s=1\) we immediately see that \(A=4\text{.}\) That leaves

\begin{equation*} 40 = (4+C)s^2 +(B-C)s + 36-B\text{.} \end{equation*}

Now we can read off \(B=-4\text{,}\) \(C=-4\text{.}\) Thus

\begin{equation*} g(t) = 4e^{t} - 4 \cos(3t) - \frac{4}{3}\sin(3t). \end{equation*}

Finally,

\begin{equation*} x_p(t) = e^{-t}g(t) = 4 - 4 e^{-t}\cos(3t) - \frac{4}{3} e^{-t}\sin(3t). \end{equation*}

For a harmonically forced case such as \(x''+2x'+10x=\cos(2t)\text{,}\) we have

\begin{equation*} X_p(s) = \frac{s}{(s^2+4)(s^2+2s+10)}\text{.} \end{equation*}

The best first step is to decompose

\begin{equation*} X_p(s) = \frac{A+Bs}{s^2+4} + \frac{C+Es}{s^2+2s+10}\text{.} \end{equation*}

The first term leads directly to \(\sin(2t)\) and \(\cos(2t)\text{,}\) while the second would once again be shifted to get \(s^2+9\) in the denominator. The details are left to the proverbial reader.
