Our first observation is that this problem can always be converted to a first-order problem in two dimensions. Define a vector \(\bu\) by \(u_1=x\text{,}\) \(u_2=x'\text{.}\) By definition, \(u_1'=u_2\text{.}\) Since \(u_2'=x''=f(t,x,x')\text{,}\) we can get the system

\begin{align*} u_1' \amp = u_2\\ u_2' \amp = f(t,u_1,u_2). \end{align*}

This is a first-order system \(\bu'=\mathbf{g}(t,\bu)\) of two scalar equations for the two components.
