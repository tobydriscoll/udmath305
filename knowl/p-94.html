You should have seen some \(2\times 2\) and \(3\times 3\) determinants before now, in vector calculus. The \(2\times 2\) case is easy to describe:

\begin{equation*} \det\left( \twomat{a}{b}{c}{d} \right) = \twodet{a}{b}{c}{d} = ad-bc\text{.} \end{equation*}

This can be extended to create a real-valued function for square matrices of any size. The formalities can be complicated, but we are going to use a practical definition.
