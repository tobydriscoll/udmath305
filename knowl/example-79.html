Example 4.7.3.

Again we consider how to invert

\begin{equation*} X_p(s) = \frac{3}{(s^2+9)^2} = \frac{3}{(s-3i)^2(s+3i)^2} \text{.} \end{equation*}

The PFD has four terms that work out as

\begin{equation*} X_p(s) = \frac{1}{36} \left[ \frac{i}{s+3i} - \frac{i}{s-3i} - \frac{3}{(s+3i)^2} - \frac{3}{(s-3i)^2} \right]. \end{equation*}

These easily invert using complex exponentials:

\begin{equation*} x_p(t) = \frac{1}{36} \left[ i\bigl(e^{-3it}-e^{3it}\bigr) - 3 t \bigl(e^{-3it}+e^{3it}\bigr) \right]. \end{equation*}

Now Euler' formula leads back to the same result,

\begin{equation*} x_p(t) = \frac{1}{18} [ \sin(3t) - 3t\cos(3t) ]\text{.} \end{equation*}
