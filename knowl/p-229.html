A slightly streamlined version of the process is available for IVPs. Given the initial condition \(x(t_0)=x_0\text{,}\) we rewrite the general solution (2.4.4) using definite integrals:

\begin{equation*} q_0(t) = \int_{t_0}^t a(s)\, ds, \end{equation*}

followed by

\begin{equation*} x(t) = \exp[q_0(t)] \left[ c_1 + \int_{t_0}^t f(s) \exp[-q_0(s)]\, ds \right]\text{.} \end{equation*}

By the properties of definite integrals, we know that \(q(t_0)=0\text{,}\) and hence

\begin{equation*} x_0 = x(t_0) = 1\cdot \left[ c_1 + 0 \right]\text{.} \end{equation*}

So there is really no need to solve for the constant; the initial value can be baked in from the beginning:

\begin{equation*} x(t) = \exp[q_0(t)] \left[ x_0 + \int_{t_0}^t f(s) \exp[-q_0(s)]\, ds \right]\text{.} \end{equation*}
