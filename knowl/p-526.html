Now consider \(x''+2x'+10x=40\text{.}\) We get

\begin{equation*} X_p(s) = \frac{40}{s(s^2+2s+10)} \text{.} \end{equation*}

We have the same eigenvalues and so define \(X_p(s)=G(s+1)\) to get

\begin{equation*} G(s) = X_p(s-1) = \frac{40}{(s-1)(s^2+9)}. \end{equation*}

This is amenable to the PFD

\begin{equation*} G(s) = \frac{A}{s-1} + \frac{B+Cs}{s^2+9}\text{.} \end{equation*}

Clearing denominators, we get

\begin{equation*} 40 = A(s^2+9) + (B+Cs) (s-1)\text{.} \end{equation*}

If we set \(s=1\) we immediately see that \(A=4\text{.}\) That leaves

\begin{equation*} 40 = (4+C)s^2 +(B-C)s + 36-B\text{.} \end{equation*}

Now we can read off \(B=-4\text{,}\) \(C=-4\text{.}\) Thus

\begin{equation*} g(t) = 4e^{t} - 4 \cos(3t) - \frac{4}{3}\sin(3t). \end{equation*}

Finally,

\begin{equation*} x_p(t) = e^{-t}g(t) = 4 - 4 e^{-t}\cos(3t) - \frac{4}{3} e^{-t}\sin(3t). \end{equation*}
