Example 4.6.10.

The undamped oscillator \(x''+9x=\sin(3t)\) has a perfect resonance. From zero initial conditions we have

\begin{equation*} X(s) = \frac{3}{(s^2+9)^2}\text{.} \end{equation*}

We are going to use

\begin{equation*} \frac{d}{ds} \left[ \frac{3}{s^2+9} \right] = \frac{-6s}{(s^2+9)^2}. \end{equation*}

Specifically, if we designate \(G(s)=2sX(s)\text{,}\) then we know that

\begin{equation*} \lx[ t \sin(3t) ] = -\frac{d}{ds} \left[ \frac{3}{s^2+9} \right] = G(s), \end{equation*}

and therefore \(g(t)=t\sin(3t)\text{.}\) Next, note that

\begin{equation*} 2\lx[x'(t)] = 2[sX(s)-X(0)] = G(s). \end{equation*}

Hence \(x'(t) = \frac{1}{2} t \sin(3t)\text{,}\) and integration by parts yields

\begin{equation*} x(t) = \frac{1}{18} [ \sin(3t) - 3t\cos(3t) ]\text{.} \end{equation*}
