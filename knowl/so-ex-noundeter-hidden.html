The equation \(x''+x=\cos(\omega t)\) suggests the particular solution \(x_p=A\cos(\omega t)+B\sin(\omega t)\text{.}\) Upon substitution,

\begin{align*} [-\omega^2 A\cos(\omega t) - \omega^2 B\sin(\omega t) ] + [ A\cos(\omega t) + B\sin(\omega t)] \amp =\cos(\omega t),\text{.} \end{align*}

which leads to the conclusion that \(B=0\) and, if \(\omega^2 \neq 1\text{,}\) \(A=1/(1-\omega^2)\text{.}\) However, if \(\omega = 1\text{,}\) the substitution would leave us with \(0=\cos(t)\text{,}\) which is impossible to satisfy for all \(t\text{.}\)
