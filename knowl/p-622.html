The van der Pol equation is a nonlinear oscillator in the form \(y''+\mu(y^2-1)y'+y=0\) for parameter \(\mu \gt 0\text{.}\) It is equivalent to the system

\begin{align*} x_1' \amp = x_2 \\ x_2' \amp = -x_1 - \mu(x_1^2-1)x_2, \end{align*}

which has the sole equilibrium point \((0,0)\text{.}\) The Jacobian matrix is

\begin{equation*} \twomat{0}{1}{-1-2\mu x_1 x_2}{-\mu(x_1^2-1)}, \end{equation*}

which at the origin is simply \(\twomat{0}{1}{-1}{\mu}\text{.}\) This has eigenvalues

\begin{equation*} \frac{1}{2} \left( {\mu} \pm \sqrt{\mu^2-4} \right). \end{equation*}

If \(0\lt \mu \lt 2\text{,}\) the origin is an unstable spiral point. If \(\mu > 2\text{,}\) the origin is an unstable node, as you can never achieve a negative eigenvalue.
