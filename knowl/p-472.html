In order to normalize the orthogonal complex exponential functions, we need to calculate the inner product in the case of \(j=k\text{:}\)

\begin{align*} \inprod{e^{ijx}}{e^{ijx}} \amp = \int_{-\pi}^\pi e^{-ijx}\cdot e^{ijx}\, dx\\ \amp = 2\pi \end{align*}

So we define

\begin{equation*} u_k(x) = \frac{1}{\sqrt{2\pi}} e^{ikx}, \quad k=0,\pm 1,\pm 2,\dots\text{.} \end{equation*}

(Does it bother you that this set has infinitely many members? Sorry, but functions are naturally infinite-dimensional.) Then this suggests the formulas

\begin{equation*} f(x) = \sum_{k=-\infty}^\infty c_k u_k(x),\quad c_k = \int_{-\pi}^\pi \overline{u_k}(x) f(x)\, dx\text{.} \end{equation*}

This is our first encounter with what is called a Fourier series for \(f\text{.}\) We do have to be cautious when we approach infinite sums. While this formula isn't straightforwardly true for every function you could think of, it's true enough to be one of the most monumental developments in the history of the analysis of functions.
