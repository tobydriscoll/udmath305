That derivation may have seemed complicated, but the conclusion is not. To simplify it we will restate it for the special case in which the functions in the ODE are all constants. Thus, to solve

\begin{equation*} \opA[x] = x' - a x = H(t-T), \quad x(0)=x_0\text{,} \end{equation*}

we first solve \(x' - a x = 0\) with \(x(0)=x_0\) to get

\begin{equation*} x_h(t) = e^{a t} x_0\text{.} \end{equation*}

Then we solve \(x' - a x = 1\) with \(x(0)=0\) to get

\begin{equation*} x_s(t) = e^{a t} - 1\text{.} \end{equation*}

Then the solution to the original IVP is

\begin{equation*} x(t) = x_h(t) + H(t-T)x_s(t-T) = e^{a t} x_0 + H(t-T)(e^{a(t-T)}-1)\text{.} \end{equation*}

This technique of adding together solutions to linear equations is called superposition. It nicely extends to having multiple step functions that define a piecewise constant forcing. For example, to solve

\begin{equation*} x'- a x = k_1 H(t-T_1) + k_2 H(t-T_2), \quad x(0)=x_0\text{,} \end{equation*}

we would use

\begin{equation*} x(t)=x_h(t) + k_1 H(t-T_1)x_s(t-T_1) + k_2 H(t-T_2)x_s(t-T_2)\text{,} \end{equation*}

and so on.
