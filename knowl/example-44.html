Example 3.2.2 Phase portrait of a saddle

Here we give a phase portrait for the equation \(y''+6y'-3y=0\text{.}\)

A = 1;  B = 6;  C = -3;
s = sort(roots([A B C]),'descend');

plot(0,0,'r.','markersize',18)
    
% Plot trajectories
hold on
colr = get(gca,'colororder');
t = linspace(0,15,300)';
for theta = 2*pi*(0:23)/24
    c = [1 1;s(1) s(2)] \ [ cos(theta); sin(theta) ];
    y = c(1)*exp(s(1)*t) + c(2)*exp(s(2)*t);
    dydt = s(1)*c(1)*exp(s(1)*t) + s(2)*c(2)*exp(s(2)*t);
    plot(y,dydt,'linew',1.5,'color',colr(1,:))
end
y = 2*exp(s(2)*t); dydt = 2*s(2)*exp(s(2)*t);
plot([y -y],[dydt -dydt],'k','linew',1.5)
y = 2*exp(-s(1)*t); dydt = 2*s(1)*exp(-s(1)*t);
plot([y -y],[dydt -dydt],'k','linew',1.5)

axis([-1.5 1.5 -1.5 1.5]), axis square
title('saddle')
xlabel('y')
ylabel('y''')

In this case all solutions are of the form \(y=c_1 e^{s_1t} + c_2 e^{s_2 t}\) where \(s_2\lt 0 \lt s_1\text{.}\) As time advances, \(e^{s_2 t}\) decays to zero while the other exponential grows, so again \(y\approx c_1 e^{s_1 t}\) and \(y'\approx c_1 s_1 e^{s_1 t}\text{.}\) That is, solutions in the phase plane asymptotically move away from the origin parallel to the vector \((1,s_1)\text{.}\) The only exception is if \(c_1=0\text{,}\) in which case solutions go straight toward the origin forever. (Solutions with \(c_2=0\) go toward the origin in "backward time.") In this situation we say that the origin is a saddle.
