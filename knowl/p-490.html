The ODE for the mass-spring system is

\begin{align*} 5 x'' + 8x' + 11 x \amp = 0 \\ x'' + 1.6 x' + 2.2 x \amp = 0, \end{align*}

from which we identify the natural frequency

\begin{equation*} \omega_0 = \sqrt{2.2} \approx 1.483 \text{s}^{-1}, \qquad \end{equation*}

The damping coefficient is therefore

\begin{equation*} Z = \frac{1.6}{2\omega_0} \approx 0.539. \end{equation*}

Since this value is less than one, the system is underdamped.
