It should be clear from the definition that we have two important properties,

\begin{gather*} \mathcal{L}[ f(t) + g(t) ] = F(s) + G(s) \\ \mathcal{L}[ C f(t) ] = C F(s)\text{,} \end{gather*}

where \(C\) is any constant. In short, we can say that the Laplace transform is a linear operation on functions, just like differentiation.
