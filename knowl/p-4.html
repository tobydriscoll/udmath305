It might feel silly, but let's review what it means to solve the linear equation

\begin{equation*} ax = b \end{equation*}

for \(x\text{.}\)

  1. If \(a\neq 0\text{,}\) there is a single solution, \(x=b/a\text{.}\)

  2. Otherwise:

    1. If also \(b=0\text{,}\) then every value of \(x\) is a valid solution.
    2. Otherwise, there are no solutions.

To summarize, for nonzero \(a\) there is exactly one solution, and otherwise there are infinitely many or zero solutions (depending on \(b\)). A linear system with no solutions is called inconsistent , with the opposite being consistent.
