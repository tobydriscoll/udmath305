To summarize, we proceed through the rows \(1,2,\ldots,n-1\text{.}\) For row \(i\) we seek to put zeros below the pivot element in the \((i,i)\) position. To do that we choose a multiplier for each \(k>i\) and add that multiple of row \(i\) to row \(k\text{.}\) The one thing that can stop us is if the pivot element is zero, because we can't find multipliers. In that event we swap row \(i\) with a lower row. After the eliminations are done, the system is in upper triangular form and can be solved by backward substitution.
