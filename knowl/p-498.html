Here's a third go at almost the same problem, but this time we allow a parameter: \(y''+\lambda^2 y=0\text{,}\) \(y(0)=y(\pi)=0\text{,}\) where \(\lambda^2\) is an unspecified nonnegative constant (hence we require only that \(\lambda\) is real). The general solution here is \(y(x)=c_1\cos(\lambda x) + c_2\sin(\lambda x)\text{.}\) The linear system arising from the initial conditions is now

\begin{equation*} \twomat{1}{0}{\cos(\lambda \pi)}{\sin(\lambda \pi)} \twovec{c_1}{c_2} = \bzero\text{.} \end{equation*}

The matrix is singular if and only if \(\sin(\lambda\pi)=0\text{,}\) or \(\lambda = 0,\pm1,\pm2,\dots\text{.}\) At each of these values of \(\lambda\text{,}\) every multiple of \(\sin(\lambda x)\) is a solution. The zero function is the only solution at any other choice of \(\lambda\text{.}\) In fact we ought to rule out \(\lambda=0\) as well, because then the solution is again identically zero.
