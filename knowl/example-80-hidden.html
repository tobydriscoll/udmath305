Suppose \(x''+2x'+10x=6\delta(t-1)\text{,}\) with zero initial conditions. Then

\begin{equation*} X(s) = \frac{6e^{-s}}{s^2+2s+10}\text{.} \end{equation*}

The quadratic formula tells us the homogeneous eigenvalues \(-1\pm 3i\text{,}\) which will contribute an amplitude \(e^{-t}\) in time. This corresponds to a shift \(a=-1\text{,}\) so we designate \(X(s)=F(s+1)\text{.}\) Thus

\begin{equation*} F(s) = X(s-1) = 6e \frac{e^{-s}}{s^2-2s+1+2s-2+10} = 6e \frac{e^{-s}}{s^2+9}. \end{equation*}

Now we can apply the Shift Theorem to \(F\) using the definition \(V(s)=1/(s^2+9)\) and get

\begin{equation*} f(t) = 6e H(t-1) v(t-1), \end{equation*}

where \(v(t)=\frac{1}{3}\sin(3t)\text{.}\) When we unwind the definitions, we get

\begin{equation*} x(t) = e^{-t} f(t) = 2 e^{-t+1} H(t-1)\sin(3t-3). \end{equation*}
