Let's solve \(x'-x=w(t)\text{,}\) \(x(0)=-2\text{,}\) where \(w(t)\) is a window function that is 5 for \(3 \lt t \le 6\) and zero elsewhere. We can write the problem as

\begin{equation*} x' - x = 5[H(t-3) - H(t-6)] = 5H(t-3) - 5H(t-6), \quad x(0)=-2\text{.} \end{equation*}

The solution therefore consists of three terms:

\begin{equation*} x(t) = -2 e^t + 5 H(t-3) (e^{t-3}-1) - 5H(t-6)(e^{t-6}-1)\text{.} \end{equation*}

The first term is the homogeneous (or free) response, and the others switch on as the forcing changes its value. A piecewise expression of the solution is

\begin{equation*} x(t) = \begin{cases} -2e^t, \amp 0 \le t \lt 3, \\ -2e^t + 5(e^{t-3}-1), \amp 3 \le \lt 5, \\ -2e^t + 5(e^{t-3}-e^{t-6}), \amp 5 \le t \end{cases}\text{.} \end{equation*}
