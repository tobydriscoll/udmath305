Let's close with the diffusion equation, also with a "forcing term" (e.g., a source of heat),

\begin{equation*} \pp{u}{t} = \alpha^2 \ppp{u}{x} + A f(x,t), \quad 0 \le x \le L\text{.} \end{equation*}

We introduce

\begin{equation*} t = \tau T, \quad x = \xi X, \quad u = \nu U\text{,} \end{equation*}

so that

\begin{equation*} \frac{\nu}{\tau} \pp{U}{T} = \frac{\alpha^2\nu }{\xi^2} \ppp{U}{X} + A f(\xi X,\tau T), \quad 0 \lt \xi X \lt L \end{equation*}

Then

\begin{equation*} \pp{U}{T} = \frac{\alpha^2\tau }{\xi^2} \ppp{U}{X} + \frac{A\tau}{\nu} F(X,T), \quad 0 \lt \xi X \lt L \end{equation*}

We have two quantities to set equal to one, but three scales at our disposal. We'll use the space scale to get rid of the size of the domain: \(\xi=L\text{,}\) so that \(0\lt X \lt 1\text{.}\) Plus,

\begin{equation*} \frac{\alpha^2\tau }{\xi^2} = 1 \quad \Rightarrow \quad \tau = \frac{L^2}{\alpha^2}, \end{equation*}
\begin{equation*} \frac{A\tau}{\nu} = 1 \quad \Rightarrow \quad \nu = \frac{AL^2}{\alpha^2}\text{.} \end{equation*}

This makes everything boil down nicely to

\begin{equation*} \pp{U}{T} = \ppp{U}{X} + F(X,T), \quad 0 \lt \xi X \lt 1\text{.} \end{equation*}
