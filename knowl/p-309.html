Let's start with the simplest, scalar case, \(e^{it}\text{.}\) This should be the solution of \(x'=ix\text{,}\) \(x(0)=1\text{.}\) If we assume that \(x(t)\) is complex-valued, let us write it as

\begin{equation*} x(t) = u(t) + i v(t)\text{,} \end{equation*}

for real \(u,v\text{.}\) Then \(x'=ix\) becomes

\begin{equation*} u' + iv' = i(u+iv) = i^2v + iu = -v + iu\text{.} \end{equation*}

This leads to the coupled real equations

\begin{equation*} u' = -v, \quad v'=u\text{,} \end{equation*}

together with \(u(0)=1\text{,}\) \(v(0)=0\text{.}\) It won't take you long to verify that \(u(t)=\cos(t)\text{,}\) \(v(t)=\sin(t)\) is the solution. So this implies the identity
