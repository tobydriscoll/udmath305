All solutions to first-order linear problems share a basic structure. In fact there is a perfect (and rather deep) analogy to how one can specify a line using a direction parallel to the line, and one point on the line. For the line direction we could use a vector \(\vec{v}\text{.}\) Note that the set of all vectors \(\{C\vec{v}: C\in\mathbb{R}\}\) is a line through the origin. We can then translate this line by adding any constant vector \(\vec{p}\) representing a point on the line. Thus the set of all vectors representing points on the line is \(C\vec{v}+\vec{p}\text{.}\)
