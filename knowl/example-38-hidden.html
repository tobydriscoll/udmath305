To solve \(x'+x=-t e^t\text{,}\) first note that \(x_h(t)=c_1e^{-t}\text{.}\) For \(x_p\) we use the educated guess

\begin{equation*} x_p(t) = (B_1 t + B_0)e^t\text{.} \end{equation*}

Then

\begin{equation*} x_p'(t) = (B_1 t + B_0)e^t + (B_1)e^t \end{equation*}

Substituting that into the ODE yields

\begin{equation*} e^t(B_1t+B_0+B_1) + e^t(B_1 t + B_0) = -t e^t\text{.} \end{equation*}

On canceling the exponentials and matching like powers of \(t\text{,}\) we get the equations \(2B_1=-1\) and \(2B_0+B_1=0\text{.}\) Hence \(B_1=-1/2\) and \(B_0=1/4\text{.}\) So the general solution is

\begin{equation*} x(t) = c_1 e^{-t} + \frac{1-2t}{4}\, e^t\text{.} \end{equation*}
