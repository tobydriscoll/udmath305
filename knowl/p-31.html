The general solution is actually a family of solutions due to the arbitrary constant. We can get a unique solution if we specify one more thing about \(y\text{.}\) Typically this is in the form of an initial condition, or given value for \(y(0)\text{.}\) So if \(y=Cy_h+y_p\) is a general solution, we impose the initial condition to get \(y(0)=Cy_h(0)+y_p(0)\text{,}\) which can be solved for \(C\text{.}\)
