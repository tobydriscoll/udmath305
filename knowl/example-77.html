Example 4.7.1.

The critically damped oscillator \(x'' + 6x' + 9=0\) has a double eigenvalue at \(-3\text{.}\) To find the solution with \(x(0)=0\text{,}\) \(x'(0)=6\text{,}\) we use the transform to find

\begin{equation*} [s^2X(s) - 6] + 6[sX(s)] + 9[X(s)] = 0 \quad \Rightarrow \quad X(s) = \frac{6}{(s+3)^2}. \end{equation*}

Hence (4.7.1) applies with \(n=1\text{,}\) and

\begin{equation*} x(t) = 6t e^{-3t}\text{.} \end{equation*}
