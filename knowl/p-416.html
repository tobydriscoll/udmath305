Suppose \(x''+2x'+10x=6\delta(t-1)\text{.}\) A particular solution has

\begin{equation*} X_p(s) = \frac{6e^{-s}}{s^2+2s+10}\text{.} \end{equation*}

We can use the trick of completing the square to write this as

\begin{equation*} X_p(s) = e^{-s} \frac{6}{(s+1)^2+9} = e^{-s} F(s)\text{,} \end{equation*}

where we can now see that \(f(t) = 2e^{-t}\sin(3t)\text{.}\) By the shift theorem, \(x_p(t)=H(t-1)f(t-1)\text{.}\)
