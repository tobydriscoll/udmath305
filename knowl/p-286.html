We begin the journey with an eigenpair of \(\bA\text{,}\)

\begin{equation*} \bA \bv = \lambda \bv\text{.} \end{equation*}

If we posit that \(\bx(t)=g(t)\bv\text{,}\) then

\begin{equation*} g'\bv = \bA(g\bv) = g(\bA\bv) = g\lambda \bv\text{,} \end{equation*}

which can be satisfied if \(g'=\lambda g\text{,}\) or \(g(t)=ce^{\lambda t}\text{.}\) That is, each eigenpair gives us a homogeneous solution! (Yes, this the moment all the eigenstuff was preparing us for.)
