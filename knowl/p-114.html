Happily, there is a simple formula for the matrix inverse in the \(2\times 2\) case:

\begin{equation} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}^{-1} = \frac{1}{ad-bc} \begin{bmatrix} d \amp -b \\ -c \amp a \end{bmatrix}\text{.}\tag{1.8.1} \end{equation}

This formula breaks down if \(ad=bc\text{,}\) in which case \(\det(\bA)=0\) and the matrix is singular. You might remember that this is the same algebraic condition we derived for the linear system \(\bA\bx=\bb\) not having a unique solution.
