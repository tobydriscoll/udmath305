Applying integration by parts to the definition reveals the main reason Laplace transforms are useful in ODEs.

\begin{align*} \mathcal{L}[y'(t)] \amp = \int_0^\infty y'(t) e^{-st}\, dt \\ \amp = \left[ y(t)e^{-st}\right]_0^\infty - \int_0^\infty (-s) y(t) e^{-st}\, dt \\ \amp = -y(0) + s Y(s)\text{.} \end{align*}

Calculus (differentiation) is thereby turned into algebra (multiplication). So here is the plan of attack. Given an ODE \(y'=ay+q(t)\text{,}\) we transform everything in sight to get

\begin{equation*} sY(s)-y(0) = a Y(s) + F(s)\text{.} \end{equation*}

Now solve for the transform of the solution:

\begin{equation*} Y(s) = \frac{ F(s) + y(0)}{s-a}\text{.} \end{equation*}

Now "all" we have to do know is to compute the inverse transform, that is, find \(y(t)\) given \(Y(s)\text{.}\) This is less straightforward than computing the forward transform, unfortunately. But we can make it managable in much the same way you learned indefinite integration, by learning some canonical cases and then matching patterns.
