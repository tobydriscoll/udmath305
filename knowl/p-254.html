The solution to the linear ODE with an impulse forcing is called an impulse response. Thanks to the main impulse identity, this is easier to deal with than step forcing. Again we start with

\begin{equation*} x(t) = G(0,t)x_0 + \int_{0}^t G(s,t) f(s)\, ds\text{.} \end{equation*}

For \(f(t)=\delta(t-T)\text{,}\) there are just two cases to consider for the integral:

\begin{equation*} \int_{0}^t G(s,t)\delta(s-T) \, ds = \begin{cases} 0, \amp t \le T, \\ G(T,t) \amp t > T. \end{cases} \end{equation*}

Again using a step to express the result, we have

\begin{equation*} x(t) = G(0,t)x_0 + H(t-T) G(T,t)\text{.} \end{equation*}

For a constant-coefficient ODE this becomes

\begin{equation*} x(t) = e^{at} x_0 + H(t-T) e^{a(t-T)}\text{.} \end{equation*}

One way to read this formula is that at the impulse time, the state is instantaneously increased by one, with the usual evolution following.
