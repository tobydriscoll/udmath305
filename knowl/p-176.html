There is exactly one more possibility for our characteristic roots: they are complex numbers. This happens if \(B^2\lt 4AC\text{.}\) In this case the roots must be conjugates of one another: \(s_1=\overline{s}_2\text{.}\) We'll find it convenient to write them in cartesian form, as \(r\pm i\omega\text{.}\) Then our null solution is

\begin{equation*} y = c_1 e^{s_1t} + c_2 e^{s_2t} = e^{rt}\left( c_1 e^{i\omega t} + c_2 e^{-i \omega t} \right)\text{.} \end{equation*}

This is a perfectly fine way to write the solution, but not the only way. If we use Euler's formula and rearrange, we get

\begin{equation*} y = e^{rt}\left( C_1 \cos(\omega t) + C_2 \sin(\omega t) \right)\text{.} \end{equation*}

These are still exponential solutions, but in disguise.
