We look first at the spatial side,

\begin{equation*} X''+\lambda X = 0\text{.} \end{equation*}

All kinds of alarm bells should go off now, because this is our eigenvalue problem from the first section. We know that to continue the discussion, we need to bring in the boundary values. In the Dirichlet case we have

\begin{equation*} T(t)X(0) = T(t)X(\pi) = 0\text{,} \end{equation*}

which is guaranteed if \(X(0)=X(\pi)=0\text{.}\) So we have eigenvalues \(\lambda_n=n^2\text{,}\) \(n=1,2,\ldots\text{,}\) and the eigenfunctions \(X_n(x)=\sin(n\pi x)\) are the only nonzero solutions.
