This is neatly expressed using linear algebra:

\begin{equation*} \dd{}{t} \twovec{x_1}{x_2} = \frac{1}{200}\twomat{-14}{3}{2}{-8} \twovec{x_1}{x_2} + \twovec{12}{35} \text{.} \end{equation*}
