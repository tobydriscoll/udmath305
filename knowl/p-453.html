If the companion matrix has a double eigenvalue, then it is defective (unless \(b=c=0\text{,}\) which leaves the truly trivial problem \(x''=0\)). In terms of the original ODE, this occurs when \(b^2=4c\text{,}\) and then \(\lambda_1=\lambda_2=-b/2\text{.}\) Recalling the trick used in SubsectionÂ 3.6.2, we define

\begin{equation*} \bB = \bA + \frac{b}{2}\bI = \twomat{b/2}{1}{-c}{-b/2} = \twomat{b/2}{1}{-b^2/4}{-b/2}\text{.} \end{equation*}

You can easily check that \(\bB^2=\bzero\text{.}\) Hence the power series for the matrix exponential has just two terms:

\begin{equation*} e^{t\bA} = e^{-(bt/2)\bI} e^{t\bB} = e^{-bt/2}(\bI + t\bB) \text{.} \end{equation*}

The general solution is \(e^{t\bA}\bc\text{,}\) but again we need only the first row to get the solution for the original \(x=u_1\text{:}\)

\begin{equation*} x_h(t) = e^{-bt/2} \left[ c_1 + t\left(c_2 + c_1\frac{b}{2}\right) \right] = e^{-bt/2} \left( \tilde{c}_1 + c_2 t\right)\text{,} \end{equation*}

where \(\tilde{c}_1\) is just a different arbitrary constant. We can rename the constants and express the result as

\begin{equation} x_h(t) = e^{-bt/2} \left( c_1 + c_2 t\right) , \qquad \text{if } \lambda_1 = \lambda_2 \neq 0.\tag{4.2.4} \end{equation}
