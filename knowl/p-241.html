There is a method of expressing all this knowledge called the Laplace transform. Given (almost) any function \(f(t)\text{,}\) there is a doppelganger function \(F(s)\) defined by an integral,

\begin{equation} F(s) = \int_0^\infty f(t) e^{-st}\, dt\text{.}\tag{2.8.1} \end{equation}

In this business, time starts at \(t=0\text{,}\) and nothing before then matters. One thing to note right away is that the Laplace transform is a linear operator. It's often written using a script L, as in \(F(s)=\mathcal{L}[f(t)]\text{.}\) The usual convention is that a lowercase function of \(t\) transforms to its uppercase namesake as a function of \(s\text{.}\)
