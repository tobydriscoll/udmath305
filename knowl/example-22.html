Example 2.3.2

The solution of \(g''=\delta(t)\) is equivalent to \(g''=0\) with \(g(0)=0\text{,}\) \(g'(0)=1\text{.}\) We have \(g(t)=c_1+c_2t\) and then conclude that \(c_1=0\text{,}\) \(c_2=1\text{.}\) That is, \(g(t)=t\text{.}\)

Now suppose we want to solve \(g''=\sin(t)\text{.}\) We find

\begin{equation*} y_p(t)=\int_0^t (t-s) \sin(s) \,ds = \bigl[ (s-t)\cos(s) - \sin(s) \bigr]_0^t = -\sin(t)-t\text{.} \end{equation*}

This is not the same result we get by following the other methods of this section, which would suggest only \(-\sin(t)\text{.}\) The other \(-t\) coming from the integral formula is a part of the null solution, so both expressions are valid particular solutions.
