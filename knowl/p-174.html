There are plently of details to sift through, but the essence is quite simple. The general solution of this problem has two contributions, \(y=y_n+y_p\text{.}\) The first, \(y_n\text{,}\) is the null solution to the associated homogeneous equation \(ay''+by'+cy=0\text{,}\) and the second, \(y_p\text{,}\) is any particular solution of the original problem. To see how this works, plug in:

\begin{align*} ay''+by'+cy \amp = a(y_n+y_p)''+b(y_n+y_p)'+c(y_n+y_p) \\ \amp = (ay_n''+by_n'+cy_n) + (ay_p''+by_p'+cy_p) \\ \amp = 0 + f = f\text{.} \end{align*}

The null solution contains two arbitary constants that are used to soak up the initial conditions. We add that the particular solution is not unique, so the complete representation is not either. However, all such expressions are equivalent in the end, and the solution is always expressible in this form.
