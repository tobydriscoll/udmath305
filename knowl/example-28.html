Example 2.5.1

The ODE \(y'''+2y''-y'-2y=4 e^{2t}\) has characteristic polynomial \(P(s)=s^2+2s^2-s-2\text{.}\) The roots of \(P\) are \(-2,-1,1\text{.}\) So the general null solution is

\begin{equation*} y_n = c_1 e^{-2t} + c_2 e^{-t} + c_3 e^{2t}\text{.} \end{equation*}

The transfer function is evaluated at \(r=2\) to get \(Y(2)=1/P(2)=1/12\text{.}\) So the complete solution is

\begin{equation*} y = c_1 e^{-2t} + c_2 e^{-t} + c_3 e^{2t} + \frac{1}{3}e^{2t}\text{.} \end{equation*}

We would need three initial conditions in order to specify the integration constants uniquely.
