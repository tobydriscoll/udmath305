In the first chapter we pointed out that the linear function \(L\) defined by >\(L(\bx)=\bA\bx\) for a given matrix \(\bA\) has the same property. In the context of the first order linear ODE (2.2.1), the operator of interest to us is defined by

\begin{equation} \opA[x] = x' - a(t)x\text{,}\tag{2.2.2} \end{equation}

whose linearity you can easily validate for yourself. We can now express the ODE simply as \(\opA[x]=f\text{.}\)
