Above we drew some phase portraits for instances of the second-order problem \(Ay''+By'+Cy=0\text{.}\) Here is another way of looking at them. Let's define a time-varying vector with components \(u_1=y\) and \(u_2=y'\text{.}\) We can call \(\mathbf{u}(t)\) the state vector of the ODE. From the definitions we find

\begin{align*} \dd{u_1}{t} \amp = u_2 \\ \dd{u_2}{t} \amp = \frac{1}{A}(-Cu_1 -Bu_2) \text{,} \end{align*}

which is in the form \(\mathbf{u}'=\mathbf{f}(\mathbf{u})\text{.}\) All the phase portraits above are unchanged if we simply label the axes as \(u_1,u_2\text{.}\)
