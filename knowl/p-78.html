As for the details? The particular solution for \(Re^{i\omega t}\) is \(GR e^{i(\omega t - \alpha)}\text{,}\) as derived above. To recover the given forcing term we need only set \(\omega=1,R=2\) and take the imaginary part minus the real part:

\begin{align*} y \amp = \text{Im} \left[ 2 Ge^{i(t - \alpha)} \right] - \text{Re} \left[ 2 Ge^{i(t - \alpha)} \right]\\ \amp = \text{Im} \left[ \sqrt{2} e^{i(t - \pi/4)} \right] - \text{Re} \left[ \sqrt{2} e^{i(t - \pi/4)} \right] \\ \amp = \sqrt{2} \bigl[\sin(t - \pi/4) - \cos(t - \pi/4)\bigr] \text{.} \end{align*}

We could convert that into a single shifted cosine if required. In this case it happens to get a simpler answer, \(y=-2\cos(t)\text{.}\)
