Let's go back and revisit an old first-order linear friend, \(y'-ay=e^{ct}\text{.}\) Transforming both sides, we get

\begin{equation*} [sY(s)-y(0)] - aY(s) =\frac{1}{s-c}\text{,} \end{equation*}

and thus

\begin{equation*} Y(s) = \frac{y(0)}{s-a} + \frac{1}{(s-c)(s-a)}\text{.} \end{equation*}

The first part corresponds to \(y_n(t)=y(0)e^{at}\text{,}\) which is the null solution. To invert the transform of the second term, we need to express it using partial fractions. The text has a clever way mnemonic for this:

\begin{equation*} \frac{1}{(s-c)(s-a)} = \frac{1}{(a-c)(s-a)} + \frac{1}{(s-c)(c-a)}\text{.} \end{equation*}

Thus we get a sum of two exponentials,

\begin{equation*} y_v(t) = \frac{1}{c-a}\left( e^{ct}-e^{at} \right) \text{.} \end{equation*}

This is the "very particular" solution that has \(y_v(0)=0\text{.}\)
