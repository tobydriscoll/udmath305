Section 4.3 Undetermined coefficients

¶

We saw two ways to get particular solutions for first-order equations: variation of parameters and undetermined coefficients. Since a second-order problem has a first-order system equivalent, we can use variation of parameters for a system, as in Section 3.7, to find a particular solution. However, for many common second-order problems with constant coefficients, it is much easier to adapt the method of undetermined coefficients.

Specifically, the ODE \(x''+bx'+cx=f(t)\) can be solved with this method when \(f\) is a polynomial, exponential, sin, or cos function. The appropriate form for the particular \(x_p\) for each case is given in Table 2.5.1. This form is plugged into the ODE, and the resulting identity is used to deduce the undetermined coefficients.

Example 4.3.1.

Let's find a particular solution of \(x'' +4x'+4x=8t^2\text{.}\) The correct form of \(x_p\) is a quadratic polynomial, so

\begin{equation*} x_p = A + Bt + Ct^2\text{.} \end{equation*}

Plugging that into the ODE yields

\begin{align*} 2C + 4(B+2Ct) + 4(A+Bt+Ct^2) \amp = 8t^2\\ 4C t^2 + (8C+4B) + (2C+4B+4A) \amp = 8t^2\text{.} \end{align*}

This has to be an identity for all \(t\text{.}\) Matching like powers, we can read off \(C=2\text{,}\) then \(B=-4\text{,}\) and finally \(A=3\text{.}\) This provides us with \(x_p=2t^2+4t+3\text{.}\)

Example 4.3.2.

For \(x'' - 2x'-3x=10e^{4t}\text{,}\) the proper choice is

\begin{equation*} x_p = Ae^{4t}\text{.} \end{equation*}

Everything else is just basic manipulations.

\begin{align*} 16A e^{4t} - 2 (4A e^{4t}) - 3 (Ae^{4t}) \amp =10 e^{4t}\\ 16A -8A -3A = 10\text{.} \end{align*}

From this it's clear that \(A=2\text{.}\)

Example 4.3.3.

For \(x''+x'=\sin(2t)\text{,}\) we use

\begin{equation*} x_p = A\cos(2t) + B\sin(2t)\text{.} \end{equation*}

Inserting this into the ODE leads to

\begin{equation*} [-4A\cos(2t) - 4B\sin(2t)] + [-2A\sin(2t) + 2B\cos(2t)] = \sin(2t)\text{.} \end{equation*}

This is true for all time if and only if we match the coefficients of like trig functions:

\begin{equation*} -4A + 2B = 0, \qquad -4B-2A = 1 \text{.} \end{equation*}

The solution of these equations is \(A=-1/10\text{,}\) \(B=-1/5\text{.}\)

The examples above are the fundamental ones. There are rules for more intricate combinations of the same functions, but we won't go into them here.

We have to repeat the warning from the first time we saw this method: occasionally it fails, given our rules.

Example 4.3.4.

The equation \(x''+x=\cos(\omega t)\) suggests the particular solution \(x_p=A\cos(\omega t)+B\sin(\omega t)\text{.}\) Upon substitution,

\begin{align*} [-\omega^2 A\cos(\omega t) - \omega^2 B\sin(\omega t) ] + [ A\cos(\omega t) + B\sin(\omega t)] \amp =\cos(\omega t),\text{.} \end{align*}

which leads to the conclusion that \(B=0\) and, if \(\omega^2 \neq 1\text{,}\) \(A=1/(1-\omega^2)\text{.}\) However, if \(\omega = 1\text{,}\) the substitution would leave us with \(0=\cos(t)\text{,}\) which is impossible to satisfy for all \(t\text{.}\)

The failure of Example 4.3.4 at \(\omega = 1\) was due to the fact that the \(x_p\) we picked is actually a homogeneous solution. There are additional rules to cover this case, but here is an interesting workaround instead. Suppose we seek a particular solution for \(\omega=1\) as the limit of a particular solution as \(\omega\to 1\text{.}\) We won't use the exact \(x_p\) found in the example, but instead subtract a homogeneous solution from it to get

\begin{equation*} \frac{ \cos(\omega t) - \cos(t) } {1-\omega^2}, \end{equation*}

which is also a particular solution. Then L'Hopital applies in the limit to give us

\begin{equation} x_p(t) = \frac{ -t \sin(\omega t) }{ -2 \omega } = \frac{1}{2}t\sin(t). \label{so-eq-secular}\tag{4.3.1} \end{equation}

Now, to verify that this is a particular solution in the limiting case, we compute

\begin{align*} x_p \amp = \frac{1}{2}t\sin(t) \\ x_p' \amp = \frac{1}{2}[\sin(t)+t\cos(t)] \\ x_p'' \amp = \frac{1}{2}[\cos(t) + \cos(t) - t\sin(t)], \end{align*}

so that \(x_p''+x_p=\cos(t)\) as required.
