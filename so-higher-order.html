Section 2.5 Higher-order ODEs

Â¶

From 2.4 in the text.

First-order and second-order ODEs dominate the landscape when it comes to scientific applications. But once in a while we go to higher order. For linear problems with constant coefficients, this looks a lot like the second-order case.

Subsection 2.5.1 Solving linear high-order ODEs

Any solution of the linear ODE

\begin{equation*} a_N \frac{d^Ny}{dt^N} + \cdots + a_2 \frac{d^2y}{dt^2} + a_1 \frac{dy}{dt} + a_0y = f(t)\text{,} \end{equation*}

where the \(a_N,\ldots,a_0\) are all constants, can be expressed as \(y=y_n+y_p\text{,}\) the sum of a null part \(y_n\) that solves the problem when \(f(t)\equiv 0\) and a particular solution \(y_p\text{.}\)

An exponential \(e^{st}\) is a null solution if \(s\) is a root of the characteristic polynomial

\begin{equation*} P(s) = a_N s^N + \cdots + a_1 s + a_0\text{.} \end{equation*}

If the roots \(s_1,\ldots,s_N\) are all distinct, then every null solution has the form

\begin{equation*} y_n = c_1 e^{s_1t} + c_2 e^{s_2t} + \cdots + c_N e^{s_Nt}\text{.} \end{equation*}

(Repeated roots are handled by using powers of \(t\) as multipliers of repeated terms; we do not give the details.)

If \(f(t)=e^{rt}\text{,}\) then a particular solution is \(y_p=Ye^{rt}\) for the transfer function \(Y(r)=1/P(r)\text{.}\)

Example 2.5.1

The ODE \(y'''+2y''-y'-2y=4 e^{2t}\) has characteristic polynomial \(P(s)=s^2+2s^2-s-2\text{.}\) The roots of \(P\) are \(-2,-1,1\text{.}\) So the general null solution is

\begin{equation*} y_n = c_1 e^{-2t} + c_2 e^{-t} + c_3 e^{2t}\text{.} \end{equation*}

The transfer function is evaluated at \(r=2\) to get \(Y(2)=1/P(2)=1/12\text{.}\) So the complete solution is

\begin{equation*} y = c_1 e^{-2t} + c_2 e^{-t} + c_3 e^{2t} + \frac{1}{3}e^{2t}\text{.} \end{equation*}

We would need three initial conditions in order to specify the integration constants uniquely.

Factoring a polynomial of degree three or higher requires luck or a computer. Only a relative few special problems yield simple closed answers.

Example 2.5.2

The characteristic polynomial of \(y''''-y=0\) is \(s^4-1\text{.}\) The roots are the "fourth roots of unity," given by \(s_j=\exp(2\pi i j/4)\) for \(j=0,1,2,3\text{.}\) These simplify to \(\pm 1, \pm i\text{.}\) In real form the null solution is

\begin{equation*} y_n = c_1 e^{-t} + c_2 e^{t} + c_3 \cos(t) + c_4\sin(t)\text{.} \end{equation*}

Subsection 2.5.2 Linear operator notation

You're familiar with a function as a rule mapping a value (number) to another value. A rule for mapping functions to other functions is called an operator. The first nontrivial operator you ever encountered was differentiation, the map from \(u(x)\) to \(u'(x)\text{.}\) We sometimes write this operator as a capital D: \(u'=D[u]\text{.}\) (Many times we leave out the square brackets, too.)

One of the most salient and familiar facts about the differentiation operator is that it is linear. By this we mean that it obeys two rules:

  • \(D[ku] = kD[u]\) if \(k\) is any constant, and

  • \(D[u+v] = D[u]+D[v]\) if \(u,v\) are any functions.

We use \(D^k\) to refer to the operation of taking the \(k\)th derivative.

Given a linear ODE of order \(N\) as above, we can define

\begin{equation*} L[y] = a_N D^N y + \cdots + a_1 Dy + a_0 y\text{.} \end{equation*}

This makes it much easier to write the ODE, as \(L[y]=f\text{.}\) Furthermore, it's easy to check that \(L\) satisfies the two rules above and thus is a linear operator as well. The linearity also makes certain kinds of reasoning very simple, such as our most basic pattern for general solutions:

\begin{equation*} L[y_n+y_p] = L[y_n]+L[y_p] = 0+f=f\text{.} \end{equation*}

An interesting and sometimes useful consequence of the defitions is that if \(L\) has constant coefficients and the characteristic polynomial factors as \(P(s)=(s-s_1)\cdots(s-s_N)\text{,}\) then \(L=(D-s_1)\cdots (D-s_N)\text{.}\) Here the juxtaposition means operator compositions.

Example 2.5.3

The operator \(L[y]=y'''+2y''+y'+2y\) has characteristic roots \(-2,\pm i\text{.}\) Now note that for any function \(y\text{,}\)

\begin{align*} (D+2)(D-i)(D+i)y \amp = (D+2)(D-i)(y'+iy) \\ \amp = (D+2)\Bigl((y''+iy')-i(y'+iy)\Bigr) \\ \amp = (D+2)(y''+y) = y''' +y' +2y'' + 2y = L[y]\text{.} \end{align*}

We can write the factors in any order, and even things like \(L[y]=(D^2+1)(D-2)\text{.}\)
