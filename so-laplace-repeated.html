Subsection 4.7.1 Repeated poles and resonance

We know by now that repeated poles in \(X_p(s)\) arise in two ways: from repeated characteristic roots (i.e. poles of the transfer function), or perfect resonance with the forcing function. We can anticipate the effect on the transform, because we know the answer in the time domain is something like \(te^{rt}\text{.}\)

In fact if we have any \(f(t)\) with transform \(F(s)\text{,}\) then from the transform definition we compute

\begin{align*} F'(s) \amp = \frac{d}{ds} \int_0^\infty f(t) e^{-st}\, dt\\ \amp = -\int_0^\infty t f(t) e^{-st}\, dt\\ \amp = - \lx[ tf(t) ]\text{.} \end{align*}

Take a moment to appreciate this. Earlier we found that a derivative in \(t\) corresponds to multiplying by \(s\text{.}\) Up to a change in sign, a derivative in \(s\) corresponds to multiplying by \(t\text{!}\)

Example 4.7.1.

The undamped oscillator \(x''+9x=\sin(3t)\) has a perfect resonance. The transform suggests the particular solution

\begin{equation*} X_p(s) = \frac{3}{(s^2+9)^2}\text{.} \end{equation*}

This doesn't look like anything easy. However, note that

\begin{equation*} sX_p(s) = \frac{3s}{(s^2+9)^2} = -\frac{3}{2} \frac{d}{ds} \left[ \frac{1}{s^2+9} \right] \text{.} \end{equation*}

Therefore, \(sX_p(s)\) is the transform of \(\frac{1}{2} t \sin(3t)\text{.}\) So this must be \(x_p'(t)\text{.}\) An integration by parts then recovers

\begin{equation*} x_p(t) = \frac{1}{18} [ \sin(3t) - 3t\cos(3t) ]\text{.} \end{equation*}

If we keep differentiating in \(s\) we find

\begin{equation*} \frac{d^n}{ds^n} \lx[ f(t) ] = (-1)^n \lx[ t^n f(t) ]\text{.} \end{equation*}

Of interest is the special case of an exponential function \(e^{rt}\text{,}\) whose transform is \(1/(s-r)\text{:}\)

\begin{equation*} \lx \left[ t^n e^{rt} \right] = \frac{n!}{(s-r)^{n+1}} \text{.} \end{equation*}

The even specialer case \(r=0\) allows us to transform any power of \(t\text{,}\) and therefore any polynomial.

Subsection 4.7.2 Complex poles

Complex poles in the transform correspond to complex exponentials in the time domain. When they occur in conjugate pairs, the entire expression can be cast in real terms. We have

\begin{equation*} \frac{1}{s-(a+ib)} + \frac{1}{s-(a-ib)} = \frac{2(s-a)}{(s-a)^2+b^2} \text{.} \end{equation*}

Since we also know that

\begin{equation*} e^{(a+ib)t} + e^{(a-ib)t} = 2 e^{at} \cos( b t)\text{,} \end{equation*}

the conclusion is

\begin{equation*} \lx[ e^{at} \cos( b t) ] = \frac{s-a}{(s-a)^2+b^2}\text{.} \end{equation*}

In similar fashion we can show that \(e^{at} \sin( b t)\) transforms to \(b/[(s-a)^2+b^2]\text{.}\) In principle we can always find quadratic denominators like these for a partial fraction conversion. However, the details can get pretty intense for hand calculation.

Example 4.7.2.

Suppose \(x''+2x'+10x=6\delta(t-1)\text{.}\) A particular solution has

\begin{equation*} X_p(s) = \frac{6e^{-s}}{s^2+2s+10}\text{.} \end{equation*}

We can use the trick of completing the square to write this as

\begin{equation*} X_p(s) = e^{-s} \frac{6}{(s+1)^2+9} = e^{-s} F(s)\text{,} \end{equation*}

where we can now see that \(f(t) = 2e^{-t}\sin(3t)\text{.}\) By the shift theorem, \(x_p(t)=H(t-1)f(t-1)\text{.}\)

Now consider \(x''+2x'+10x=40\text{.}\) We get

\begin{equation*} X_p(s) = \frac{40}{s(s^2+2s+10)} = \frac{40}{s[(s+1)^2+9]}\text{.} \end{equation*}

We decompose into

\begin{equation*} X_p(s) = \frac{A}{s} + \frac{3B+C(s+1)}{(s+1)^2+9}\text{.} \end{equation*}

Once we find the constants, we know that the inverse of this is \(x_p(t) = A + Be^{-t}\sin(3t) + C e^{-t}\cos(3t)\text{.}\) To find the constants we clear denominators and get

\begin{equation*} 40 = A[(s+1)^2+9] + (3B+Cs+C) s\text{.} \end{equation*}

With \(s=0\) we conclude that \(A=4\text{.}\) From this we find that \(40 = Cs^2+(3B+C)s + 4s^2 +8s + 40\text{,}\) which tells us \(C=-4\) and \(B=-4/3\text{.}\) Hence

\begin{equation*} x_p(t) = 4 - \frac{4}{3} e^{-t} \sin(3t) - 4 e^{-t} \cos(3t)\text{.} \end{equation*}

For a harmonically forced case such as \(x''+2x'+10x=\cos(2t)\text{,}\) we have

\begin{equation*} X_p(s) = \frac{s}{(s^2+4)(s^2+2s+10)}\text{.} \end{equation*}

We could find a decomposition in the form

\begin{equation*} X_p(s) = \frac{2A+Bs}{s^2+4} + \frac{3C+E(s+1)}{(s+1)^2+9}\text{,} \end{equation*}

which is eminently invertible. The details are left to the proverbial reader.

The drawback to Laplace transforms as a one-stop solver is that either we include the initial conditions from the start, which forces us to do more partial fraction decompositions, or else the particular solution we find almost always includes some terms from the homogeneous solution. Mathematically, the poles of \(X_p(s)\) include those of the transfer function and the forcing function. Undetermined coefficients is a less sexy but simpler way to solve concrete problems in many cases.
