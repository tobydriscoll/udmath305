Subsection 2.9.1 Repeated poles and resonance

We know by now that repeated poles in \(Y(s)\) arise in two ways: from repeated characteristic roots (i.e. poles of the transfer function), or perfect resonance in the forcing function. We can anticipate the effect on the transform, because we know the answer in the time domain is something like \(te^{rt}\text{.}\)

In fact if we have any \(f(t)\) with transform \(F(s)\text{,}\) then from the transform definition we compute

\begin{align*} F'(s) \amp = \frac{d}{ds} \int_0^\infty f(t) e^{-st}\, dt\\ \amp = -\int_0^\infty t f(t) e^{-st}\, dt\\ \amp = - \mathcal{L}[ tf(t) ]\text{.} \end{align*}

Take a moment to appreciate this. Earlier we found that a derivative in \(t\) corresponds to multiplying by \(s\text{.}\) Up to a change in sign, a derivative in \(s\) corresponds to multiplying by \(t\text{!}\)

Example 2.9.1

The undamped oscillator \(y''+9y=\sin(3t)\) has a perfect resonance. The transform suggests the particular solution

\begin{equation*} Y(s) = \frac{3}{(s^2+9)^2}\text{.} \end{equation*}

This doesn't look like anything easy. However, note that

\begin{equation*} sY(s) = \frac{3s}{(s^2+9)^2} = -\frac{3}{2} \frac{d}{ds} \left[ \frac{1}{s^2+9} \right] \text{.} \end{equation*}

Therefore, \(sY(s)\) is the transform of \(\frac{1}{2} t \sin(3t)\text{.}\) So this must be \(y'(t)\text{.}\) An integration by parts then recovers

\begin{equation*} y(t) = \frac{1}{18} [ \sin(3t) - 3t\cos(3t) ]\text{.} \end{equation*}

If we keep differentiating in \(s\) we find

\begin{equation*} \frac{d^n}{ds^n} \mathcal{L}[ f(t) ] = (-1)^n \mathcal{L}[ t^n f(t) ]\text{.} \end{equation*}

Of interest is the special case of an exponential function \(e^{ct}\text{,}\) whose transform is \(1/(s-c)\text{:}\)

\begin{equation*} \mathcal{L} \left[ t^n e^{ct} \right] = \frac{n!}{(s-c)^{n+1}} \text{.} \end{equation*}

The even specialer case \(c=0\) allows us to transform any power of \(t\text{,}\) and therefore any polynomial.

Subsection 2.9.2 Complex poles

Complex poles in the transform correspond to complex exponentials in the time domain. When they occur in conjugate pairs, the entire expression can be cast in real terms. We have

\begin{equation*} \frac{1}{s-(a+ib)} + \frac{1}{s-(a-ib)} = \frac{2(s-a)}{(s-a)^2+b^2} \text{.} \end{equation*}

Since we also know that

\begin{equation*} e^{(a+ib)t} + e^{(a-ib)t} = 2 e^{at} \cos( b t)\text{,} \end{equation*}

the conclusion is

\begin{equation*} \mathcal{L}[ e^{at} \cos( b t) ] = \frac{s-a}{(s-a)^2+b^2}\text{.} \end{equation*}

In similar fashion we can show that \(e^{at} \sin( b t)\) transforms to \(b/[(s-a)^2+b^2]\text{.}\) In principle we can always find quadratic denominators like these for a partial fraction conversion. However, the details can get pretty intense for hand calculation.

Example 2.9.2

Suppose \(y''+2y'+10y=6\text{.}\) The very particular solution has \(Y(s)=6/(s^2+2s+10)\text{.}\) We can use the trick of completing the square to write this as \(Y(s)=6/[(s-1)^2+9]\text{,}\) and thus \(y(t) = 2e^{-t}\sin(3t)\text{.}\)

Things are much more complicated for a harmonically forced case such as \(y''+2y'+10y=\cos(2t)\text{.}\) Now

\begin{equation*} Y(s) = \frac{s}{(s^2+4)(s^2+2s+10)}\text{.} \end{equation*}

It is true that we have a decomposition in the form

\begin{equation*} Y(s) = \frac{A+Bs}{s^2+4} + \frac{C+Es}{(s-1)^2+9}\text{,} \end{equation*}

which is eminently invertible. But finding the four coefficients is a pretty heavy lift to do by hand.

The problem with Laplace transforms as a one-stop solver is that the particular solution we find is always the "very particular" one that includes some terms from the null solution. Equivalently, the poles of \(Y(s)\) include those of the transfer function and the forcing function. It's extra algebra to find the coefficients of the null/transient parts, but that step can't be skipped. Undetermined coefficients is a less sexy but simpler way to solve concrete problems in most cases.
