Section 3.6 Matrix exponential

¶

One solution of \(x'=ax\) for constant \(a\) is \(x=e^{at}\text{.}\) We know the Taylor series

\begin{equation*} e^{at} = 1 + at + \frac{1}{2!}(at)^2 + \frac{1}{3!} (at)^3 + \cdots\text{.} \end{equation*}

This series is something we could generalize to a square matrix \(\bA\text{,}\) for which integer powers are possible:

\begin{equation} e^{t\bA} = \bI + t\bA + \frac{1}{2!}t^2 \bA^2 + \frac{1}{3!} t^3 \bA^3 + \cdots\text{.}\label{fs-eq-matrixexp}\tag{3.6.1} \end{equation}

Let's not worry too much about whether this converges (it does). What are its properties?

Theorem 3.6.1. Matrix exponential.

Let \(\bA,\bB\) be \(n\times n\) matrices. Then

  1. \(e^{t\bA}=\bI\) if \(t=0\text{,}\)
  2. \(\dd{}{t}e^{t\bA} = \bA e^{t\bA} = e^{t\bA}\bA\text{,}\)
  3. \([e^{t\bA}]^{-1} = e^{-t\bA}\text{,}\) and
  4. If \(\bA\bB=\bB\bA\text{,}\) then \(e^{\bA+\bB} = e^{t\bA}e^{t\bB} = e^{t\bB}e^{t\bA}\text{.}\)

These conclusions follow from the series definition (3.6.1). They establish some important facts.

Fact 3.6.2.
  1. For any constant vector \(\bc\text{,}\) \(\bx(t)=e^{t\bA}\bc\) is a solution of \(\bx'=\bA\bx\text{.}\)
  2. \(\bX(t)=e^{t\bA}\) is the unique fundamental matrix for \(\bx'=\bA\bx\) with \(\bX(0)=\bI\text{.}\)

Summing an infinite series of matrices may not seem like a great prospect. But we have two ways to keep it fairly tame. Which to use depends on an important property of the matrix.

In section Subsection 1.11.2 we saw the definition of a defective eigenvalue (Definition 1.11.14). A matrix with one or more defective eigenvalues is also called .

Fact 3.6.3.

A defective \(n\times n\) matrix has fewer than \(n\) independent eigenvectors--that is, the sum of the eigenspace dimensions is less than \(n\text{.}\)

This is especially easy to spot when \(n=2\text{:}\)

Fact 3.6.4.

A \(2\times 2\) matrix with repeated eigenvalue is either a multiple of the identity or defective.

Subsection 3.6.1 Nondefective matrix

¶

For a nondefective \(\bA\text{,}\) Theorem 3.3.1 guarantees that

\begin{equation*} \bX(t) = \begin{bmatrix} e^{t\lambda_1}\bv_1 \amp e^{t\lambda_2}\bv_2 \amp \cdots \amp e^{t\lambda_n}\bv_n \end{bmatrix} \end{equation*}

is a fundamental matrix for the ODE \(\bx'=\bA\bx\text{,}\) where the \((\lambda_j,\bv_j)\) are eigenpairs. If we now define

\begin{equation*} \bY(t) = \bX(t) \bX(0)^{-1}\text{,} \end{equation*}

then

\begin{equation*} \bY'(t) = \bX'(t) \bX(0)^{-1} = \bA \bX(t) \bX(0)^{-1} = \bA \bY(t)\text{,} \end{equation*}

so \(\bY(t)\) is a fundamental matrix as well. But also \(\bY(0)=\bX(0)\bX(0)^{-1}=\bI\text{,}\) so in fact \(\bY(t)\) is the matrix exponential.

When this argument is fully unraveled, it can be summarized as follows. Given a complete set of eigenvectors, i.e.,

\begin{equation*} \bA \bv_1 = \lambda_1 \bv_1, \; \bA \bv_2 = \lambda_2 \bv_2,\; \dots, \; \bA \bv_n = \lambda_n \bv_n\text{,} \end{equation*}

define

\begin{equation*} \bV = \begin{bmatrix} \bv_1 \amp \bv_2 \amp \cdots \amp \bv_n \end{bmatrix}. \end{equation*}

(In the notation above, this is \(\bX(0)\text{.}\)) Then

\begin{equation} e^{t\bA} = \bV \begin{bmatrix} e^{t\lambda_1} \amp \amp \amp \\ \amp e^{t\lambda_2} \amp \amp \\ \amp \amp \ddots \amp \\ \amp \amp \amp e^{t\lambda_n} \end{bmatrix} \bV^{-1}\text{.}\label{fs-matrixexp}\tag{3.6.2} \end{equation}

This is one of those cases in which you're probably better off working from the formula than going each time through the process we used to get it.

Example 3.6.5.

Let's find the exponential of

\begin{equation*} \bA = \twomat{1}{1}{4}{1}\text{,} \end{equation*}

given \(\lambda_1=3\text{,}\) \(\bv_1 = \twovec{1}{2}\text{,}\) and \(\lambda_2=-1\text{,}\) \(\bv_2=\twovec{1}{-2}\text{.}\) We identify

\begin{equation*} \bV = \twomat{1}{1}{2}{-2} \end{equation*}

and compute

\begin{align*} e^{t\bA} \amp = \twomat{1}{1}{2}{-2} \twomat{e^{3t}}{0}{0}{e^{-t}} \twomat{1}{1}{2}{-2}^{-1} \\ \amp = -\frac{1}{4} \twomat{1}{1}{2}{-2} \twomat{e^{3t}}{0}{0}{e^{-t}} \twomat{-2}{-1}{-2}{1} \\ \amp = -\frac{1}{4} \twomat{e^{3t}}{e^{-t}}{2e^{3t}}{-2e^{-t}} \twomat{-2}{-1}{-2}{1} \\ \amp = \frac{1}{4} \twomat{2e^{3t}+2e^{-t}}{e^{3t}-e^{-t}}{4e^{3t}-4e^{-t}}{2e^{3t}+2e^{-t}} \text{.} \end{align*}

Not exactly a walk in the park, but doable.

Example 3.6.6.

Suppose

\begin{equation*} \bA = \twomat{1}{-1}{5}{-3}\text{,} \end{equation*}

and we are given or work out that

\begin{equation*} \lambda_1 = -1+i, \; \lambda_2 = -1-i, \qquad \bV = \twomat{1}{1}{2-i}{2+i}\text{.} \end{equation*}

Then

\begin{equation*} \bV^{-1} = \frac{1}{2+i-2+i} \twomat{2+i}{-1}{-2+i}{1} = \frac{i}{2} \twomat{-2-i}{1}{2-i}{-1}\text{,} \end{equation*}

and then

\begin{align*} e^{t \bA} \amp = e^{-t}\frac{i}{2}\twomat{1}{1}{2-i}{2+i} \twomat{e^{it}}{0}{0}{e^{-it}} \twomat{-2-i}{1}{2-i}{-1} \\ \amp = e^{-t}\frac{i}{2}\twomat{e^{it}}{e^{-it}}{(2-i)e^{it}}{(2+i)e^{-it}} \twomat{-2-i}{1}{2-i}{-1} \\ \amp = e^{-t}\frac{i}{2} \twomat{-(2+i)e^{it}+(2-i) e^{-it}}{e^{it}-e^{-it}}{-(2-i)(2+i)e^{it}+(2+i)(2-i) e^{-it}}{(2-i)e^{it}-(2+i) e^{-it}} \text{.} \end{align*}

Each entry is written in the form of a complex number minus its conjugate, which gives \(2i\) times the imaginary part. This helps a lot.

\begin{equation*} e^{t \bA} = -e^{-t} \twomat{-\cos(t)-2\sin(t)}{\sin(t)}{-5\sin(t)}{-\cos(t)+2\sin(t)}\text{.} \end{equation*}

Because the matrix exponential is a fundamental matrix, the general solution to \(\bx'=\bA\bx\) can always be written in the form \(e^{t\bA}\bc\text{.}\) This isn't the easiest way to get a general solution, because the \(\bX(t)\) above requires less work. But this form is handy with an IVP, because

\begin{equation*} \bx_0 = \bx(0) = e^{0\bA}\bc = \bc\text{.} \end{equation*}

That is, \(e^{t\bA}\bx_0\) solves the IVP.

Example 3.6.7.

Using the matrix from the previous example, we show how the numerical solution of \(\bx'=\bA\bx\) (which doesn't know what kind of ODE it solves) is the same as the one computed via \(e^{t\bA}\bx_0\text{.}\)

Subsection 3.6.2 Defective matrix

¶

In the 2-by-2 case, a defective matrix has a double eigenvalue but is not diagonal. In this case we have no clear way to construct the fundamental \(\bX(t)\) that gave us the matrix exponential above. Fortunately, we have another reasonable path for the \(2\times 2\) situation.

Example 3.6.9.

The matrix

\begin{equation*} \bA = \twomat{-1}{1}{0}{-1} \end{equation*}

has a double eigenvalue of \(\lambda=-1\) and is defective. Now consider

\begin{equation*} \bB = \bA - \lambda \bI = \twomat{0}{1}{0}{0}\text{.} \end{equation*}

This matrix has the property that \(\bB^2=\bzero\text{.}\) That means the exponential series for it is just

\begin{equation*} e^{t\bB} = \bI + t\bB = \twomat{1}{t}{0}{1}\text{.} \end{equation*}

Finally, we can use Theorem 3.6.1 to calculate

\begin{equation*} e^{t\bA} = e^{t\lambda\bI}e^{t(\bA-\lambda\bI)} = e^{-t}\twomat{1}{t}{0}{1}\text{.} \end{equation*}

Matters are less simple for larger matrices, but we won't be going there.
